严毅不知道自己想的对不对，毕竟他的总裁在商界无往而不利，强大到翻云覆雨，但是，自从遇到太太，总裁总是会在不经意间流露出一点小心翼翼，在太太工作需要自由的空间时，总裁会让自己保持安静，太太说了这是不可以的事，总裁会收敛自己的寒芒和锋锐让自己看起来听话一些，总裁对待太太始终很努力，带着几分谦卑的讨好变成她所喜欢的样子，靠她近一点。
<br/>
<br/>        这时陆寒霆收回了手机，他抬眸看向窗外的霓虹灯，嗓音低沉磁性道，“她很乖的，如果回来自然会回来的，我打电话过去反而像查岗，我不想逼她太紧，听说这样的男人很讨人厌的。”
<br/>
<br/>        严毅将咖啡送了过去，“总裁，你怎么会这么想，夫妻之间查岗很正常，吃醋也正常，还喜欢胡思乱想，今天微博爆出来的照片…我看到太太拽了二少一下，如果总裁想问都可以问。”
<br/>
<br/>        哦，是吗？
<br/>
<br/>        陆寒霆知道自己现在很想打电话让她回来，最好是立刻马上，他还想让她好好解释一下她为什么去拽陆子羡，她明知道他很介意。
<br/>
<br/>        可是，他并没有那样做。
<br/>
<br/>        他像一个没事人一样安静的坐在这里，结束一场会议，明明耳畔里回荡的都是陆茵茵让他放手的话，让他成全那对其实在两年前就在帝都城遇见的才子才女，明明胸腔里已经燃烧起了一把烈火，醋意的，介意的，怀疑的，阴郁暴躁的…
<br/>
<br/>        但他什么都没有表现出来，他压着心里所有的风起云涌，努力的让自己看起来…像一个正常人。
<br/>
<br/>        陆寒霆将薄唇勾出了一道浅浅的弧线，最后低声道，“她一直让我相信她，我正在努力，所以我相信她没有任何的问题。”
<br/>
<br/>        ……
<br/>
<br/>        财务会议就在总裁办公室里召开的，陆寒霆抽完了两根香烟然后坐在了主位上，财务总监将报表递上来，陆寒霆垂眸快速的翻看了两眼，然后他蜷起手指在桌面上“咚咚”的敲了两下，凌厉有力，“这就是你做出来的东西？”
<br/>
<br/>        财务总监一脸的憋屈，总裁大人就给了他十分钟，他能整理出这些不容易了啊。
<br/>
<br/>        陆寒霆抬手，一份报表直接抛到了空中，纸张飞舞，“重新去做，直到你能做出让我满意的东西！”
<br/>
<br/>        话音落下，整个总裁办公室陷入了压抑的诡异中，所有人都低下头，不敢去看主位那男人阴云密布的样子。
<br/>
<br/>        就在这个时候，“咔嚓”一声，办公室里面的休息间门突然被打开了，一只纤白的小手伸了过来，扒着墙壁，然后里面有一张巴掌大的小脸悄悄的露了出来，脸藏在门框后，看不见，但那双黑漉而澄亮的眸子确实是往会议桌这里探来了。
<br/>
<br/>        所有人的目光齐刷刷的落了过去，天哪，这是谁啊，竟然出现在了总裁的私人休息室里？
<br/>
<br/>        听到动静的陆寒霆扭头，一下子就撞上了夏夕绾那灵气逼人又带着几分试探柔怯的水眸。
