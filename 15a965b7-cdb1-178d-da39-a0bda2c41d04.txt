　　第169章
<br/>　　“大当家，咱们准备这么久了，什么时候走？”高飞虎。
<br/>　　“快该走了，我心里有算计。”
<br/>　　“那大当家你要算计到什么时候，万一戎族杀过来，咱们可都走不了。”
<br/>　　“怎么可能这么快，你以为戎族个个都是飞毛腿呀。”
<br/>　　“大当家……”
<br/>　　程大雷突然回身，堵住了高飞虎的话。
<br/>　　“其实我一直在思考一件事。”
<br/>　　“什么？”
<br/>　　“你说戎族会杀过来嘛？”程大雷认真问。
<br/>　　“怎么不会……难道大当家觉得不会？”
<br/>　　“其实我这几天一直在思考这件事。”程大雷道：“你说戎族已经抢了一座黑石城，他还杀过来干什么。万一他不杀过来，我们丢下这么大一份家业，颠颠跑了，岂不是很尴尬。”
<br/>　　关于戎族会不会杀过来，山寨内部一直都有讨论。当然，戎族不过来更好，如果他杀过来，大家只有死路一条。所以，没人愿意用脑袋去赌命。
<br/>　　“大当家的意思是？”
<br/>　　“我想去黑石城看看。”程大雷道。
<br/>　　“大当家，你疯了不成！”
<br/>　　“疯什么疯，我都想好了，咱们选十几个能打的兄弟，一人两匹快马，夜里去，夜里回，也不和他们动手。”
<br/>　　“大当家，你这是去送死！”
<br/>　　“住口！”程大雷突然板起脸：“我还是不是山寨的当家，我的话还作不作数！”
<br/>　　“这个……”
<br/>　　“什么这个那个，我是大当家，我的话你就要听。我主意已定，今夜就出发。”
<br/>　　到黄昏，程大雷已经点齐人手：秦蛮、赵子龙、徐神机、关鱼、张肥、高飞虎、高飞豹、小白狼，熊大熊二等山寨头目，包括程大雷在内，一共三十六人，实力最低都是优秀等级的山寨。
<br/>　　留着黄三元、刘悲在山寨主持大局，一行三十六人，每人两匹马，磨快兵器，带足弓箭，连夜出发。
<br/>　　路上丝毫不爱惜马力，等到黑石城外围时，马已经跑得气喘吁吁，再无半点力气。
<br/>　　此时已经是深夜，众人躲在城外的山林中，周围安静得没有任何声音，清晰得只剩下众人的心跳声。
<br/>　　就算占了黑石城，戎族也并没有在城内扎营，而是把大帐按在城外，黑夜里看去，乌压压一片，隐隐有篝火的光亮。
<br/>　　“喂，听说了没，咱们要出发了。”
<br/>　　“是该去下个地方了，这里都快待腻了。”
<br/>　　“话说，这帝国女人生得就是水灵啊，白得像羔羊。”
<br/>　　“哈，你敢说这话，不怕家里的母老虎把你吃了。你抢的那个女人不错，不如送给我吧。”
<br/>　　“好，拿两头牛我就给你换。”
<br/>　　一队巡逻的戎族从面前经过，高飞虎躲在黑暗的树林里吓得呼吸都屏住了，等他们走远了才长长的松口气。
<br/>　　“听见没，听见没，他们要杀过来了，大当家，咱们快回山寨，准备逃吧！”
<br/>　　正说着，突然发现程大雷已经不见了。再回过神来，只看他已骑马拎斧冲了过去。
<br/>　　“劈脑袋！”
<br/>　　“剔牙齿！”
<br/>　　“掏耳朵！”
<br/>　　秦蛮和赵子龙见到这一幕，也是吓了一跳，抽弓搭箭射杀攻击程大雷的敌人。见程大雷已经解围，忙骑马冲杀出去。
<br/>　　“这是干什么，这是干什么！”高飞虎大叫：“不是来刺探情报么！”
<br/>　　关张二将，熊大熊二，小白狼等人此刻也没有别的办法，只好随大家一起冲杀出去。
<br/>　　瞬间马啸人叫，双方杀成一片。戎族虽强，但程大雷这次带来的人也并非弱手，更何况还有秦蛮和赵子龙这两员猛将。
<br/>　　短时间内，一个十几人组成的戎族小队，已经被诛杀。
<br/>　　这里的厮杀已经惊动大帐中的人，有人骑着马狂冲出来。
<br/>　　“敌袭！”
<br/>　　“敌袭！”
<br/>　　“什么人，什么人！”
<br/>　　“吾乃落叶城城主，青牛山陷阵营游骑将军，幽州王王印执掌者！”程大雷骑在马上大叫：“戎族的渣渣们，受死吧！”
<br/>　　高飞虎欲哭无泪：“这是在搞什么，不是说今天就是刺探么，怎么还报上名了！”
<br/>　　只见此刻戎族大营内，已经有上千人冲杀出来。
<br/>　　“跑呀！”
<br/>　　程大雷大喊一声，拨转马头，向后狂窜。
<br/>　　“杀呀杀！”
<br/>　　“射箭，射箭！”
<br/>　　身后追兵如狼似虎，程大雷跑得也极快，高飞虎边用刀抽马屁股，边在马上大叫。
<br/>　　“姓程的，你在搞什么！你究竟在搞什么！”
<br/>　　程大雷在马上放声大笑：“逃命去喽，逃得慢，就要吃戎族的箭！”
<br/>　　“大当家，他们的马快，我们快被追上了！”秦蛮回身射箭，纵马而奔。
<br/>　　“不怕，走大道，大道上我有准备！”
<br/>　　程大雷已经命令廖甲廖乙在官道上设下路障，山石，大木，还有挖得深深浅浅的河沟。
<br/>　　程大雷近日随着廖甲廖乙做这些事，对于道路已经摸熟，知道那里有路障，那里没有。由他带路，队伍的速度虽然略微减慢，但并无太大妨碍。
<br/>　　而当戎族的追兵进入到此地的时候，骑兵的威力立刻被限制住，纵然戎族的骑术好，但也有追兵人仰马翻。
<br/>　　“逃！”
<br/>　　“逃！”
<br/>　　“逃！”
<br/>　　高飞虎口中大叫，心中松口气，这下终于可以逃走了。
<br/>　　“大当家，咱们赶紧回山寨，能逃的逃吧！”
<br/>　　令高飞虎悲催的是，戎族一旦追不上时，程大雷反而停下来，冲着追兵大喊。
<br/>　　“北蛮部的狗贼，程大爷在此，速速来我斧前受死啊！”
<br/>　　一路逃回山寨，众人骑马奔进山寨，山寨大门在他们身后关上。
<br/>　　“你要搞什么！”高飞虎大叫。
<br/>　　山寨的众人已经被惊醒，此刻聚在校场上，只听说戎族杀过来了，顿时人心惶惶，吵嚷声一片。
<br/>　　“戎族杀来了！”
<br/>　　“什么！那我们还怎么逃，逃到半路就被他们杀死了！”
<br/>　　“我说什么来着，我说什么来着，我说咱们快点走吧！”
<br/>　　点亮火把，山谷内灯火通明，程大雷站在点将台上，众人看着他的眼神有埋怨，有指责，有愤怒。
<br/>　　“你根本没想过逃走，磨磨蹭蹭拖延时间，又故意把戎族引过来，就是要大家和你一起送死！”高飞虎气急败坏的大喊，此时他已经看穿程大雷的用意。
<br/>
