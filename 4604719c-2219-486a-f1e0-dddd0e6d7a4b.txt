　　罗昊把战术中队的人分成三个小组，从三个方向接近佟千雪被关押的村子实施营救。
<br/>　　碧绿茂密的丛林中，穿着丛林迷彩作战服，脸上涂着浓厚的伪装迷彩披着伪装网，伪装网上还别着一些植被枝叶增加伪装效果的罗昊，手中握着一把MC迷彩涂装的紧凑型HK416自动步枪小心的前行着。
<br/>　　树干和树干间蜘蛛结的蜘蛛网上因为大雨的冲刷，破了几个洞，蜘蛛丝上还挂着晶莹的水珠，一只五彩斑斓的大蜘蛛正在忙碌的爬动着，修补着蜘蛛网，然后安静的等待着猎物的上门；全身碧绿，跟周围植被几乎融为一体的昆虫蹦上一片叶子，头上两根触须不断抖动着，两只往外凸起的眼珠子来回转动着，观察着周围的情况，两条强壮的后腿在叶片上用力一蹬，蹿了出去，叶片晃动着，叶片上的水珠也是被晃得滚落了下来。
<br/>　　罗昊在丛林中前行的时候，不时注意着自己脚下，观察泥土是不是有被翻动重新伪装过的痕迹。
<br/>　　布克尔特自己在“绿色贝雷帽”中服役过，而且他手下也有不少从特种部队退役的老兵，把他们当做南美洲当地那些散漫的武装毒贩来处理，那就大错特错了。跟布克尔特和他手下的武装势力作战，就跟一支特种部队交手没有太多的区别，罗昊他们懂得，他们也懂。
<br/>　　所以，如果在丛林中发现一颗经过高度伪装的诡雷也不需要有什么意外，这是很正常的情况。
<br/>　　突然，走在队伍前面的罗昊竖起左手，示意队伍停止前进。
<br/>　　罗昊用手指摸了摸留在地上的脚印，捡起地上一小截被碰擦折断的植被断枝，仔细观察后，把断枝递给自己身后的周柏，说道：“作战靴，两个人。脚印和断枝都是新鲜的，人应该刚经过不久，提高警惕，这附近有巡逻兵。”
<br/>　　就在此时，不远处有人说话的声音传来，还有人在丛林中走过时，身体跟植被碰擦发出的“沙沙”声。
<br/>　　注意到有人正朝着自己的方向靠近，罗昊眼神为之一凝，低声吩咐道：“隐蔽，有人过来了！”
<br/>　　战术B组的几个人迅速隐蔽起来，司徒空从小习武，即便身上背了几十公斤的装备，也丝毫不影响他的行动，抬头看了眼自己身边树干笔直的大树，“蹭蹭”两下就爬了上去，动作简直比猴子还要快。
<br/>　　居高望远，隐藏在树枝上的司徒空看清楚了远处的情况，往自己这边过来的有三个人，M4A1自动步枪，从他们走路的姿态上看，他们应该不是职业军人出身，但是受过专业的军事训练。
<br/>　　两个人抬着一个木箱子，看的出，木箱子的分量很重，两个带着木箱子的武装人员的手绷得笔直。
<br/>　　三个武装分子往前走了一段距离，观察后停了下来，两个武装分子把抬着的箱子放在地上打开，从司徒空的角度看去，可以清楚的看到装在那个木箱子里的是什么，赫然是一箱子的反步兵地雷！
<br/>　　三个武装分子用铲子在地上挖坑进行埋雷，因为下了雨，地面泥土湿软的关系，三个人挖坑的速度很快。
<br/>　　“狂龙，他们在埋雷，要怎么办？”司徒空通过耳麦向罗昊做出询问。
<br/>　　“解决掉他们，不能在这里滞留太长的时间，停留的时间越长，事情越容易发生变化。”
<br/>　　罗昊眼中闪过如极地寒风般刺骨的杀意，按住耳麦说道：“龙猫，那两个埋雷的交给你，那个正在警戒的交给我。”
<br/>　　“这可有点难度，不过我喜欢挑战高难度！”
<br/>　　司徒空嘿嘿一笑，把枪甩到身后，摘下挂在背后的反曲弓，从箭袋中抽出两支弓箭搭在弓弦上，把弦拉满，瞄准了那两个蹲在地上正在埋雷的武装分子，右手弓弦一松，两支箭镞中含有剧毒的弓箭呼啸着朝蹲在地上埋雷的两个武装分子飞去。
<br/>　　噗嗤！
<br/>　　噗嗤！
<br/>　　两支弓箭分别扎在了两名武装分子的后背和肩膀上，尖锐的刺痛，令得两个武装分子大惊，可是还没等到两个人做出反应，从箭镞中释放出来的剧毒已经发挥了作用，使得他们的心脏骤停，瞬间毙命。
<br/>　　剩下一个负责警戒的武装分子看到自己的两个同伴身上插着弓箭，瞬间倒地身亡，惊慌过后就想要鸣枪示警，可是还没等他把枪举起来，一颗钨芯穿甲弹呼啸而至，从他的右前额钻入，他的右半侧脑袋炸裂开来，在他身后的树干上留下一个深陷的弹孔，混杂着骨头碎渣的血浆黏糊糊的喷溅在树干上，将这碧绿的自然美景破坏的一干二净。
<br/>　　“仿前苏联的单兵地雷，绊发引信。”
<br/>　　罗昊拿起一颗铸铁弹体，带有破片凹槽，形状跟老式木柄手榴弹有些相似的反步兵地雷，对周柏和林晓阳吩咐道：“把这里布置一下，这附近应该不止他们一组人，如果发现他们失联，肯定会试着找过来。”
<br/>　　周柏和林晓阳自然知道怎们做，拿出反步兵地雷四个捆成一组插在树干下，每组反步兵地雷上分别拉出两根绊线分别系在两个中箭身亡的武装分子的腰带和他们身上的反曲弓弓箭上，不管是搬动尸体，还是拔出他们身上弓箭都会引爆地雷。
<br/>　　“残忍是你们残忍，这一爆炸是要把人炸成肉渣啊。”
<br/>　　司徒空看着藏在两个武装分子尸体背后的两组反步兵地雷，砸吧了一下嘴。司徒空没有去收回两个武装分子尸体上的弓箭，虽然把弓箭回收还能再使用，但是箭镞中的剧毒已经没有了，杀伤力发生了改变，而且箭镞的重量也发生了改变，会影响到弓箭的精准度。
<br/>　　处理掉那三个武装分子后，战术B组的人继续前进，他们距离关押佟千雪的村子应该不远了。
<br/>　　“我现在就担心无常的状态。”周柏略显担忧的说道：“如果他不能保持冷静的话，营救行动会很麻烦。”
<br/>　　“该说的都跟他说了，如果他处理不好的话，回去以后，我会考虑让他离开战术中队！”罗昊平静的说道。
