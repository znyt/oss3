　　　　“叶少阳。”雪魔站在，上下望着叶少阳，“你就是叶少阳。”
<br/>　　叶少阳跟他没什么好说的，脑海中飞快地思索，制定着逃走的计划。
<br/>　　“今天本该杀你，但时间不多，却是算你走运！”
<br/>　　雪魔说完，长剑横扫，霎时间寒风呼啸，无数雪花从剑锋带起的气息中凭空生出，打在剑阵上，立刻砍开了一个缺口，欺身而上。
<br/>　　叶少阳也想试试他的实力，干脆也挺剑上前，用剑诀斗了起来。
<br/>　　之前有广宗天师帮着疗伤，自己也吐纳了一个周天，叶少阳法力恢复得差不多了，将体内被炼化的戾气也一并用上，与雪魔缠斗起来，短时间内顶得住，不过很快也察觉出来，雪魔的实力比自己要高出半个到一个等级。
<br/>　　不愧是太阴山来的。
<br/>　　“你一个人间法师，有这等修为，实属难得，然而……还是差了一点。”
<br/>　　雪魔不再缠斗，而是后退了一步，长剑凌空横扫，不断生成冰雪风暴，一波一波地朝叶少阳打过来。
<br/>　　叶少阳虽然擅长用剑诀，但他毕竟是人间法师，人间法师受限于实质的肉身，不可能修炼出某种魂器或者邪术，可以一招鲜吃遍天，眼前雪魔所用的冰雪风暴，李浩然的剑，黎山老母的幻灵花海，都是这种方式，看上去有点像是玄幻剧里那些夸张的破碎山河的手段，其实都是法术，视觉效果也是灵力所产生的幻象。
<br/>　　人间法师，再对付鬼妖邪灵的时候，最强大的手段，是布阵结印，寻求各种法术和法器的配合，这是一套十分复杂的门道，在这方面，叶少阳敢在人间称第二，基本上没人敢称第一。
<br/>　　如果可以完全发挥自己的特长，叶少阳绝对敢跟雪魔一战，就算不能打赢，坚持个一段时间也绝对没问题。
<br/>　　但现在只是用剑诀剑招，叶少阳也只能靠着七星龙泉剑自身的强大灵力，勉强跟雪魔周旋，很快就有点支撑不住。
<br/>　　“七星长鸣！”
<br/>　　叶少阳左手捏决，将七星龙泉剑掷了出去，在剑诀的驱使下，七星龙泉剑平时蓄积的灵力在一瞬间爆发出来，一时间化生出万道剑气，将雪魔团团围住。
<br/>　　叶少阳急忙转身，从背包里摸出一件物事，对着身后不远处的芮冷玉抖了出去。
<br/>　　猛然间打开，是一副山水画卷，看上去普普通通。
<br/>　　什么东西？
<br/>　　天池罗刹皱起了眉头。他本来是想跟雪魔一起动手的，但是想了一下，雪魔这家伙十分傲气，如果是一对一战斗，绝不许任何人插手，再看到叶少阳在他手下狼狈的样子，知道胜负已定，就更没有动手的必要了，于是朝芮冷玉走过去，想要带她走。
<br/>　　叶少阳就是这个时候展开的山河社稷图。
<br/>　　山河社稷图，能吸收一切生灵，是一个绝好的去处。
<br/>　　之前事发突然，叶少阳带着芮冷玉逃走的时候，完全没想到这个，不然直接带芮冷玉进去，然后让道风或者小九把山河社稷图一收，带走就好了。可惜错过了最好的机会。就算在雪魔等人对峙的时候，叶少阳知道想要逃走怕是没希望了，于是想到了这个，让芮冷玉先进去躲着再说。
<br/>　　“小玉，你看图上有什么！”叶少阳喊了一声，山河社稷图对着芮冷玉扇过去。
<br/>　　就算她不主动进去，只要盯着打开的山河社稷图看上一下，神识就会被吸进去。如果接触到上面的图画，就算是肉身也会被吸进去的。
<br/>　　就在山河社稷图快要触碰到芮冷玉的一瞬间，天池罗刹赶到了，虽然她不知道这张图有什么问题，但叶少阳在最关键时刻把它祭出来，一定有着某种用意，干脆提前行动，冲芮冷玉飞了过去，想要先抓住她再说。
<br/>　　结果现在的芮冷玉，对她也没什么好看，纵身后退，避开了她。
<br/>　　因为天池夜叉取代了芮冷玉的位置，山河社稷图差一点击中她，本来以她的修为，如果觉察的早，是没有什么问题的，结果她转头看了一眼图中山河，皱眉道：“什么鬼东西！”
<br/>　　一只手朝山河社稷图拍过去，以为是什么法器，想要出手将其毁掉。
<br/>　　结果……手掌拍打在图上的一瞬间，人影一闪，整个人被吸了进去，瞬间消失。
<br/>　　叶少阳彻底懵了。
<br/>　　他万万没想到，会是这样一个结果。
<br/>　　“哗啦！”
<br/>　　雪魔手中的长剑，总算劈开了剑光组成的结界，打开了一个缺口，本来是想继续进攻叶少阳的，定睛一看，天池罗刹不见了，朝左右看去，也都没有。他并不知道天池罗刹被山河社稷图收进去这件事，当场懵逼。
<br/>　　“雪主！天池教母被那张图收起来了！”远处跟着他们一起来的那些鬼兵尸兵，却是看得真真切切，立刻叫嚷起来。
<br/>　　“什么！”雪魔大惊，怔怔地看着叶少阳。法器收灵这种事，是很常见的。叶少阳的阴阳镜，四宝的金玟钵盂，都有这种能力，原则上可以将任何生灵收进去，只要加上一个单向的封印，就算实力再强，只要收进去就处不来。
<br/>　　但是那些法器对太强大的对手，基本上没用，就算是作法成功，所产生的吸力，在正常情况下也很难将一个强者吸进去。
<br/>　　退一步说，就算叶少阳手中有这么强的法力，然而就这么顷刻之间就将天池罗刹这样的强者吸进去，而且一点挣扎都没有……这简直就是天方夜谭。
<br/>　　雪魔还是头一回见到这样的事情。
<br/>　　叶少阳灵机一动，转头看了一眼芮冷玉，见她还是一脸茫然地站着，似乎在回忆什么事情，但是又想不起来的样子。
<br/>　　见她站着不动，叶少阳暂时放心，把山河社稷图不急不慢地收起来，对着雪魔微微一笑。
<br/>　　雪魔一怔，道：“天池怎么样了？”
<br/>　　“放心，这里面很舒服，一时半会也死不了。不过没有咒语，她是出不来的。”
<br/>　　雪魔一听，心中更是震惊，盯着叶少阳手中的画卷说道：“这究竟是什么法器？”
