　　叶少阳摇摇头，“你不是不知道我的性格，为了怕她将来超过我，现在就把人困住，这不是我的作风啊，就算真的有你说的这一天，那就真刀真枪的干一场，就算是天选之子，我又怕什么，你见我怕过谁啊？”
<br/>　　叶少阳自信没有别的优势，就只有一点：不怂。
<br/>　　不管对手是谁，不管实力差距多大，真逼到杠子上，都敢上去拼命。
<br/>　　林三生目光软下来，吐槽道：“是是是，你最牛逼，我们都多虑了。”
<br/>　　“没有，我知道你是为我着想，而且每到关键时候，都少不了你帮我顶着啊，没有你我不知道死了几回了。”叶少阳上去拍着他的肩膀，肉麻兮兮地拍着马屁。
<br/>　　林三生白了他一眼，道：“那你这个什么都不怕的，又有什么打算啊？”
<br/>　　“这个……”叶少阳刚才还豪气云天，现在一下子懵了，是啊，说了那么多大话，下一步到底怎么办，自己还真没想到，主要是这些信息都堆在一起，自己还没能完全消化，更不要提什么计划了。
<br/>　　“就知道你小子。我来给你提几个意见吧，第一，你最近不要再去阴司，免得被酆都大帝关起来，虽然他不敢明着动你，但你总不能送上门去，第二——”
<br/>　　“等下，为什么酆都大帝不敢动我？”
<br/>　　“我说了，你现在背后势力太大，牵一发而动全身，除非能秒杀你或抓住你，否则他不会动你，大帝再强，也不可能到人间来抓你，总之你别去阴司就行，在外面，你就当什么也不知道，回头我找机会，把阴阳司剩下的人全弄到空界来。”
<br/>　　叶少阳表示赞同。
<br/>　　“第二，你，冷玉，还有这个陈悦，你们三个都浮出水面了，救世四子之中，还剩下一个，我们要全力寻找这人，就算你不想杀他，我们也要提前做好准备，至少多了解一些他的情况，没问题吧？”
<br/>　　“没问题，不过……人间真的还有这样的天才，完全没人知道吗？”叶少阳皱眉问道。
<br/>　　林三生歪头看他：“你在知道陈悦的身份以前，你能猜到她是？法术界又有谁认识她。”
<br/>　　叶少阳一想也是，问道：“还有吗？”
<br/>　　“暂时没了，反正你什么都知道了，也迈过了这道坎，将来天劫之乱，你还是要牵头的，还有，你想要证明自己，只有一个办法：灭掉太阴山，到那时候，什么天选之子、转世鬼童，统统没了意义，大帝和鬼王的‘棋局’也就结束了。”
<br/>　　“这个……”叶少阳笑哭，“这算是终极任务么？”
<br/>　　“你要是做不到，就去老实辅佐天选之子。”道风冷冷地插了一句。
<br/>　　叶少阳转头看他，道：“你都敢，我又什么不敢的，反正都跳出棋盘了，搏一把呗，该死卵朝天。”
<br/>　　四人讨论了一会，林三生要叶少阳先为眼前的战斗做好准备，叶少阳答应下来，问他们芮冷玉的情况，得知为了不让她担心，这件事情，道风和林三生都没通知她，她还在洪荒世界里教书。
<br/>　　叶少阳这才放心。
<br/>　　小九又安慰了叶少阳一番，四人正聊着，瓜瓜回来了。
<br/>　　“她走了？”叶少阳问。
<br/>　　瓜瓜点点头。
<br/>　　“她……算了，问你也不会告诉我她去哪了，少阳，你们走吧，等我准备好了找你。”
<br/>　　四人道别，道风一言不发，进入了林三生带来的山河社稷图中，林三生离去。
<br/>　　小九送叶少阳和瓜瓜下山，一路上牵着叶少阳的手，安慰着他。到了山下，打开虚空裂缝，依依不舍地把他送进去。
<br/>　　又回到了道观的后山。
<br/>　　叶少阳想到自己离开这一个小时所发生的事，这短短时间，所发生的事情，却真的是天翻地覆，自己的命运，还有陈悦的命运，必然都会因为这个晚上而发生重大的改变。
<br/>　　如今重回旧地，陈悦还不见了。叶少阳的心情很复杂。
<br/>　　山风吹得人浑身发冷，叶少阳裹紧了身上的大衣，招呼瓜瓜下山。
<br/>　　“老大……”
<br/>　　“嗯？”
<br/>　　“你……为什么不问我，我娘去了哪里？”
<br/>　　“没必要知道啊，最好不要知道，你自己知道就行了。”
<br/>　　瓜瓜咬了咬嘴唇，道：“她走的时候，倒是想让我跟她一起走来着……但我拒绝了，哎，有些事真的没办法的。”
<br/>　　叶少阳站住，俯身看着他，问道：“你想去是吧？”
<br/>　　瓜瓜毫不犹豫地点点头，“但是谁让我先遇到的是你呢。我是要一直留在你身边的。”突然跳到他肩膀上，蹭着他的脖子。
<br/>　　“你是我在这世上最亲的人，没有之一，也没人可以取代……”
<br/>　　“肉麻。”叶少阳在他脸上捏了一下，心中却是很感动。
<br/>　　“瓜瓜，你知道吗，其实什么天选之子，根本就不重要，最重要的，是有你们一直陪着我。”
<br/>　　洪荒世界，道观后山。
<br/>　　两个人叠在一起，女上男下，身体蠕动着。
<br/>　　男子不断呻吟着……
<br/>　　女子从他身边站起来，抹了抹嘴上的鲜血，对着山风，理了理头发，长叹一声。突然觉察到身后有动静，回头看了一眼，顿时怔住了。
<br/>　　“你……”
<br/>　　“一周死一个，你需求还挺旺盛。”道风望着芮冷玉，淡淡说道。
<br/>　　芮冷玉很快缓过神来，冷冷道：“你既然看到了，我也没什么好说的，幸好，你并没有恢复实力——”
<br/>　　“力”字出口，她人已经冲到道风身边，双手结印，五指中间各自喷射出一股鲜血，在脚下汇聚成流，宛如五条巨蛇，朝道风扑过去。
<br/>　　道风站着不动，一层黑气在周身自动形成，挡住了血蛇，双方僵持了几秒钟，血蛇突然崩溃，融化成一股血浪，堪破了结界，排山倒海地将道风裹在了中间。
<br/>　　芮冷玉嘴角，现出一丝冰冷的微笑，等着血海褪去，笑容却僵在了脸上——道风不见了。
<br/>　　意识到危险的她，猛然回头，果然道风就在身后，对她伸出了一根手指，芮冷玉想要反击，却发现浑身上下被一股无形的气息包裹住了，丝毫不能动弹。
