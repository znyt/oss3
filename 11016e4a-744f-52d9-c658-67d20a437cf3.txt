<br/>    陈明再次渡劫，这个消息瞬间传遍了大陆，其他的可以作假，但是天劫的气息，绝对不可能作假。
<br/>    
<br/>        谁能用天劫作假？
<br/>    
<br/>        消息只是刚刚传出，就已经震动了整个大陆，即便是古仙门和魔宗，这两个大陆之上最强大的势力，也是将目光投向了陈明，希望可以得到更多关于陈明渡劫的消息，只是关注点有所不同罢了。
<br/>    
<br/>        古仙门是知道陈明的境界的，陈明实际上还是悟道境，他们估摸着陈明是要真正突破到超脱。
<br/>    
<br/>        群星仙门老君道:“陈明又在渡劫，到底是什么情况？”
<br/>    
<br/>        雪山老君道:“可能他这一次是准备真正突破到超脱吧，他修行的功法，应该是一种特殊的功法，就好像他门下的卓青瑶一般，修行的是九九玄功，是极致闻道境，闻道境可战诸王，陈明修行的，可能也是一种怪异的功法！”
<br/>    
<br/>        铁衣老君道:“可是陈明为何如此张扬地准备突破到超脱境，这不符合常理。”
<br/>    
<br/>        群星仙门的老君道:“这其中可能也有陈明自己的打算吧，紫霞不是给了我们一柄灵剑吗，等会儿将陈明叫过来，问问陈明不就得了！”
<br/>    
<br/>        众多老君点头道:“也好！”
<br/>    
<br/>        群星仙门老君道:“现在燕山在平乱之中可谓是势如破竹，这是一个好消息，不过众多附属鬼仙的大君却是迟迟没有露面，还真是诡异啊！”
<br/>    
<br/>        在魔宗看来，陈明渡劫，这个消息可就不一样了，首先，他们得到的关于陈明的消息，是陈明已经是一位大君，而且是一个隐藏极深，从来没有入世的隐世宗门，而这个隐世宗门的强大，就在于他们拥有着数万炼丹师。
<br/>    
<br/>        而最为诡异的，在于燕山的数万炼丹师，他们竟然是一个都没有见到，所以在十万大山之中的燕山，在世人的眼中，只是一个燕山入世的前哨站而已，并不是燕山本部真正所在地，燕山，应该是如同玉虚宫一般，拥有自己的秘境，数万炼丹师都是居住在秘境之中。
<br/>    
<br/>        所以对于燕山的实力，魔宗一直是不确定的，甚至是觉得陈明不过是负责宗门和外界交流的负责人，并不是真正的燕山之主。
<br/>    
<br/>        陈明大君的实力，可谓是没有任何疑问，而一位大君渡劫，只能是成仙了！
<br/>    
<br/>        难道说，这一片大陆，又要出现真正的仙人了吗？
<br/>    
<br/>        一时之间众说纷纭，修行者之中，除了最接近仙人的存在，其他人怎么可能知道仙路关闭的消息，若是这个消息放出来，还有多少人愿意去修行，他们修行，可都是为了成仙！
<br/>    
<br/>        即便是知道这个秘密的无疆，还是四大古仙门的老君，都是对此守口如瓶。
<br/>    
<br/>        陈明站在天劫之下，很是满意城中修行者的反应，而天空之中，也是落下一缕星光，陈明站在星光之中，开始仔细参悟这一缕星光，只是一缕星光而已，还不足以让陈明受伤，陈明便站在星光下仔细参悟。
<br/>    
<br/>        还好不用像上一次一样，直接所有的天劫一起来了，估摸着是因为自己已经成功突破到悟道境，所以没有那么多的天劫一起降临了。
<br/>    
<br/>        城头上的修行者看向陈明，只觉得在见证一个奇迹，“莫非，这是仙劫！”
<br/>    
<br/>        “不像啊，这个阵势，也太小了，根本没有仙劫的威力！”
<br/>    
<br/>        “说过你见过仙劫一样！”
<br/>    
<br/>        “整个大陆上，谁见过仙劫？”
<br/>    
<br/>        “也对啊，说不定仙劫就是这样的！”
<br/>    
<br/>        陈明眼看头顶之上的劫云快要消散，而他也已经完全参悟了这一缕星光之中蕴含的阵法，仙风道骨地看了城中的修行者一眼，“此处，无法渡劫！”
<br/>    
<br/>        道完这句话，陈明一个闪身，已经是带着头顶的劫云，消失在了原地。
<br/>    
<br/>        来到一处荒山之中，陈明展开感知，确定周围没有任何人存在，陈明拿出绘刻着传送阵的灵剑，传送通道已经打开，“不知道古仙门找自己什么事？”
<br/>    
<br/>        将灵剑插在地面上，陈明已经是来到了四位老君的面前，四位老君看到陈明，都是看出了陈明的实力又有精进，问道:“陈明道友，你刚刚是在渡超脱劫？”
<br/>    
<br/>        陈明摇头，然后道:“不是，悟道劫，我修行的功法有些特殊，可能还有几千场悟道劫要渡。”
<br/>    
<br/>        四位老君看向陈明，都是疑惑不解，不过对方好歹是自己人，而且现在仙魔大战在即，也管不了那么多了，随陈明去吧。
<br/>    
<br/>        群星老君道:“陈明道友，你此举，到底意欲何为？”
<br/>    
<br/>        陈明道:“根据我那份名单之上，在我们腹地之中，还有着众多的大君没有浮出水面，我猜测，他们可能会对青瑶动手，毕竟青瑶不死，这腹地之中，注定会被她一举收复！诸位老君也是知道，我能够在我的剑之间传送，所以我想青瑶吸引出那些隐藏的大君，然后传送过去斩杀！”
<br/>    
<br/>        “而对外，我们完全可以将我塑造成成仙失败的存在，我也知道，仙路关闭，所以不可能成仙，青瑶手中的剑，其他人并不知道我有传送过去的能力，我们完全可以说是我留存在其中的一道术法，一道虚影，并不算大君直接介入战场，我们并没有违反规定。”
<br/>    
<br/>        “这样一来，便有两个好处，一来可以将这些隐藏的大君全部斩杀，二来，可以让魔宗忌惮我的实力，不敢对燕山轻举妄动，保全燕山，毕竟燕山的位置有些尴尬。”
<br/>    
<br/>        四位老君听完，顿时拍手称妙，“妙啊，妙啊，谁能知道，你可以在这些剑之间传送，燕山的安危，我们也一直担忧不已，这样一来，我们就可以放开手脚，不必太顾及燕山了。”
<br/>    
<br/>        群星老君当场拍板道:“就这么定了，我们会帮你处理其他的事的。”
<br/>    
<br/>        陈明见四位老君同意，直接传送回到荒山了。
<br/>    
<br/>        雪山老君道；“陈明此举，倒是帮了我们不少，至少可以让魔宗投鼠忌器，我们就想想，怎么帮陈明圆过去吧！”
<br/>    
<br/>        群星老君笑了笑，道:“要不我们假装准备打开仙路，如何？”
<br/>    
<br/>        四位老君相视一笑，“如此甚好！”
