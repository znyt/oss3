　　看到气氛变得有些凝固，金木研讪讪的挠了一下头发，向前走了两步，插在了几人中间。
<br/>　　“到底发生什么啦，这位客人有什么事情吗？”脸上带着灿烂的笑容金木研的声音听起来颇为开朗，莫名给人带来一丝好感。
<br/>　　瞥了一眼这个傻头傻脑的家伙，刘哔耸了耸肩，没有说话。
<br/>　　“啰嗦，这个家伙，可没有那么简单。”董香皱着眉头，正欲说些什么，柜台旁的木门却打开了。
<br/>　　木门处探出来一个小小的脑袋，是一个幼小的萝莉，似乎很怕见人，大半个身体都缩在了门后。
<br/>　　“董香姐姐，店长请这位先生进去。”看着前方挤作一团的几人，小萝莉咬了咬嘴唇，小心翼翼的说道。
<br/>　　像是棉花一般的声音，带着些许胆怯，软绵绵的很是好听。
<br/>　　闻言，董香眼中闪过一抹疑惑，不过向来服从店长命令的她，也没有多加阻拦，顺从的让开了身体。
<br/>　　轻笑一声，刘哔站起身来，整理了一下衣领，向内屋走去。
<br/>　　因为这几天实在受不了脖子被束缚的感觉，他早就将领带解开，丢到了垃圾桶里，此时刘哔胸前解开了两颗纽扣，裸露出了些许皮肤。
<br/>　　像是患有重疾一般，他的皮肤有些苍白，看不到丝毫血色。
<br/>　　方才出来传话的那个娇小萝莉，走在刘哔前方为他带路，诱人的血肉气息从前方传来，钻进了他的鼻腔，让刘哔忍不住舔了舔嘴唇。
<br/>　　笛口雏实，同时拥有甲赫和鳞赫的特殊喰种，继承了父母最优秀的基因，潜力巨大，想必，将她吞噬的话，实力会提升很多吧。
<br/>　　但是，雏实现在的状态，和金木相比差不了多少，都还没有完全觉醒能力，现在吞噬的话，并不是最好的时机。
<br/>　　咽了一口唾液，刘哔紧紧跟在雏实后面，两只眼睛如狼一般，盯着前方娇小的身影。
<br/>　　内屋的空间很大，曲曲绕绕的拐了两个弯，两人才到达目的地，雏实停下脚步，伸手敲了敲眼前的屋门。
<br/>　　“请进。”屋子里传来一个苍老的声音，雏实打开屋门，对刘哔做了一个请的手势。
<br/>　　温柔的冲雏实笑了一声，刘哔目光毫无波澜，抬脚走了进去。
<br/>　　屋子里的装饰十分简单，似乎是专门用来待客的地方，房间中央放着一套沙发和日式茶几。
<br/>　　沙发上，一个苍老的男子眯着双眼，正专心致志的泡着咖啡，姿态优雅，带着一副这个年纪老人特有的平和。
<br/>　　又有谁会想到，这个看似朴实的老人，曾经却是一名杀人无数的喰种，是整个CCG做梦都想找到的SSS级喰种‘枭’。
<br/>　　这个时间，独眼‘枭’还未出现，CCG并不知道‘枭’其实有两只，而刘哔面前的这个，正是独眼‘枭’的父亲芳村功善，被誉为‘不杀的枭’。
<br/>　　因为长时间没有吞噬活人血肉，他的实力和全盛时期已经无法相比了，但即使是这样，也不是等闲人敢去招惹的存在。
<br/>　　据说，在此之前CCG有过一次针对他的行动，那次行动中CCG出动了所有的特等搜查官，依旧拿他不下，最后还是有马贵将出手，方才逼退了他。
<br/>　　这还是芳村功善留手的原因，如果他全力出手，那时还不知道要死掉多少人。
<br/>　　“呐，店长把我叫过来，是有什么事情吗？”听到身后的关门声，刘哔心头微动，却没有坐下，和芳村功善拉开了一些距离。
<br/>　　虽然原著中这个家伙已经很久没杀过人了，但谁知道今天会不会破例，一名SSS级别的喰种，他没有任何胜利的希望。
<br/>　　甚至，想要逃走都很困难。
<br/>　　“我见过你，虽然那个时候你带着面具，但我还是能嗅出你的气息。”芳村功善往桌上的两只杯子里注满了咖啡，对刘哔做了一个‘请坐’的手势：“你的味道不一样，像是人类，又不全是。”
<br/>　　耸了耸肩，刘哔稍稍犹豫了一下，随后似乎是想到了什么，坐在了沙发上。
<br/>　　将咖啡轻轻向茶几中间推了一段距离，刘哔翘起了二郎腿，右腿的脚尖，笔直指向芳村功善的位置。
<br/>　　“呵呵，不喜欢喝咖啡？”稍稍愣了一下，店长和蔼的一笑，丝毫没有介意他的无理，反倒是起身重新拿出一只杯子，倒上了一杯白水。
<br/>　　这个男人，在不杀人的时候，总是如此儒雅。
<br/>　　接过店长递过来的白水，刘哔道了声谢，随后他也把腿放了下来。
<br/>　　他之前的动作，只是试探，既然没试探出什么，也就没必要继续保持下去了。
<br/>　　这老家伙的城府，比他想象中的要深许多，在他的计划中，刚才那种态度，怎么都会让这家伙给出一些反应来的。
<br/>　　但偏偏，店长没给他任何回馈，让他很难从中发现什么蛛丝马迹。
<br/>　　“不知道先生，应该怎么称呼？”喝了一口咖啡，店长脸上带着温和的笑意。
<br/>　　刘哔皱了皱眉，不知道他到底有什么目的。
<br/>　　“我啊，姓刘，CCG的那些家伙，都叫我修罗，这个消息，芳村先生应该是知道的。”转动着茶杯，刘哔似笑非笑的回答道。
<br/>　　在他刚进来的时候，店长点出过，见过自己的面具，也就是已经知道了自己的身份。
<br/>　　礼尚往来，他也应该按规矩回答一些什么，比如点出自己也知道店长的身份。
<br/>　　听到刘哔对他的称呼，店长稍稍有些惊讶，但也没多说什么，脸上依旧是笑吟吟的。
<br/>　　“你杀了太多人，也杀了太多喰种，整个20区，现在因为你都乱套了。”喝了一口咖啡，店长的语气平静，让人不知道他的想法。
<br/>　　撇了撇嘴，刘哔毫不在意的回答道：“哦，关我什么事。”
<br/>　　“我没别的意思，只是希望，刘先生换一个地方折腾，毕竟，我们还要在这个区域生存，也不希望引来太多麻烦。”店长脸上依旧是一幅笑吟吟的模样。
<br/>　　但是他的话语，却让刘哔脸色稍稍阴沉了下来。
<br/>　　“如果不呢？”看着店长眯起的双眼，刘哔语气有些强硬。
<br/>　　脸上的笑容有些僵硬，店长缓缓放下了手中的咖啡杯，一股庞大的气势，缓缓从他身上散发出来，像是一头正在苏醒的野兽一般。
