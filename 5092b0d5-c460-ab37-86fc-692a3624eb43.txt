　　金光险地的夜晚，几乎处处充斥着激烈打斗。
<br/>　　这一个辽阔无边的妖兽森林，在金光笼罩之下，氤氲美丽。
<br/>　　黑夜，被金光所代替；黑暗，被光芒所冲破。
<br/>　　但也正是如此，这里的一场场战斗，映衬得无比清晰；这里的一声声嘶吼，乃至痛苦哀鸣，亦异常清澈。
<br/>　　……
<br/>　　这般‘热闹’，一直持续到翌日天蒙蒙亮。
<br/>　　金光，渐渐散去。
<br/>　　热闹，忽然消失。
<br/>　　无法借助金光力量的妖兽，纷纷远遁，没入了森林深处。
<br/>　　而对于一支支人类武者队伍而言，没了金光力量的妖兽，并不值得去击杀。
<br/>　　他们特地深夜进入金光险地，只为了那些蕴含金光力量的妖兽内丹，只为了金光丹。
<br/>　　单纯普通的妖兽内丹，并不值得他们如此涉险。
<br/>　　几乎一瞬间，‘热闹’的金光险地，霎时陷入了寂静。
<br/>　　一支支人类武者队伍，渐渐退出了金光险地。
<br/>　　即便不是夜晚，金光险地仍旧是个庞大无比，危机重重的妖兽森林。
<br/>　　自然没有队伍愿意长久待在此。
<br/>　　唯一的例外的，便也只有险地接近深处的某一道飘逸身影。
<br/>　　身影，亦停下了战斗，不过并未出这片险地，而是随便寻了棵参天巨树，一跃而上。
<br/>　　粗壮的树干之上，身影盘膝而坐，背靠树身，缓缓打坐起来。
<br/>　　身影，自然是萧逸。
<br/>　　整整一晚上，他都处于战斗状态。
<br/>　　一路前行深入，所经战斗，不下数十。
<br/>　　每遇之妖兽，必是圣皇境以上妖兽。
<br/>　　这里的范围，已经是接近险地深处了，自然只只妖兽都强横无比。
<br/>　　而他的战斗，亦无一例外并不轻松。
<br/>　　算起来，遇到最弱的一只妖兽，要数那头金光幽狼了。
<br/>　　圣皇境三重妖兽，吸食金光后，达到圣皇境七重巅峰实力。
<br/>　　这算是萧逸昨夜数十场战斗里，最轻松的战斗了。
<br/>　　之后的战斗，虽不轻松，但对萧逸而言，却大有裨益。
<br/>　　每一次的堪堪胜之，每一次的险胜，皆是一次对自身战斗水准的提升。
<br/>　　每一次的躲避、交锋，皆是对自身身法、反应、速度的一次考验。
<br/>　　每一次的仗剑挡之，最后击杀，亦皆是对元力爆发的掌控。
<br/>　　总而言之，对萧逸来说，昨晚的金光险地之行，果真不虚此行。
<br/>　　而现今，他还未彻底进入金光险地深处。
<br/>　　整个金光险地，确实大得惊人。
<br/>　　即便他全速前行，怕也还要大半天的时间。
<br/>　　而当真正到达深处，里头妖兽，也绝非他昨晚遇到的妖兽那么简单。
<br/>　　当然，这对萧逸而言，又是一次磨砺。
<br/>　　不过，萧逸并未急着继续前行。
<br/>　　如今夜色已过，已是清晨。
<br/>　　他打算今晚再继续深入前行。
<br/>　　虽然他来金光险地的目的，并不如其他人类武者队伍那般，仅仅是为了金光丹。
<br/>　　他来此，更多的是为了历练，金光丹只是顺带的。
<br/>　　但即便如此，对他而言，还是夜晚的金光险地，更具挑战性。
<br/>　　吞食金光后的妖兽，实力更强。
<br/>　　他前进深入时，也才更有意思。
<br/>　　现今清晨，他大可打坐修炼，且对昨夜数十激战进行感悟。
<br/>　　另外，武者打坐修炼时，其实也算是很不错的休息。
<br/>　　对于萧逸而言，睡觉，早已是很多年的事。
<br/>　　他的瞬间时间，几乎都已被打坐所代替。
<br/>　　如无意外，他将在这片金光险地历练一月左右。
<br/>　　单单昨日初来金光险地，昨夜初战，便收获颇丰。
<br/>　　萧逸甚至猜测着，一月之后，结束金光险地的历练时，他必将大有收获。
<br/>　　即便有些事他急着去做，即便他的时间确实很赶；但，一切还是需得足够实力的前提下。
<br/>　　他能做的，只有极限提升自身实力。
<br/>　　武者的修炼，可以说充满了枯燥；毕竟，在那条遥遥无尽的武道之路上，武者一直都在漫漫求索。
<br/>　　日复一日，年复一年，始终在这条仿佛永无止境的道路上摸索着，前行着。
<br/>　　但同时，武者的修炼，也可以充满着精彩。
<br/>　　每一次的修为提升，每一次的境界突破，都是一次对武者来说不可多得的喜悦。
<br/>　　每一次感悟，每一次偶有收获，都是仿佛敞开了一道新的武道大门。
<br/>　　每一次历练，每一次激战，都是武者开阔眼界，领略这个精彩世界的一次机会。
<br/>　　对萧逸而言，他的修炼，从无枯寂。
<br/>　　他的前行，有着方向；那个方向，并非武道的指引，而是两颗在他生命中曾经绽放过绚烂光彩，却转瞬即逝的流星。
<br/>　　两颗流星，仿佛是上天的赠予，却一闪而过，落到了未知之地。
<br/>　　这片未知之地，便是指引他前行的目的。
<br/>　　那片未知之地，未必是他武道之路的终点。
<br/>　　却必然是他眼中更甚于武道之路遥远前方的某地。
<br/>　　他的历练，总是充斥着危机。
<br/>　　但，或许，也只有那些危机，那些九死一生，方能让他冷漠的内心，再燃热血；方能让他急切的心绪，暂时平缓。
<br/>　　再燃，平缓，两个截然相反的形容，在他身上，却无比契合着。
<br/>　　或许，在很多人眼中，他是个屡创奇迹，以极恐怖速度成长着的妖孽。
<br/>　　或许，在很多人看来，这是个值得惊叹、赞叹，乃至惊骇的妖孽。
<br/>　　但谁又知晓，这般恐怖成长速度，不过是他勉强压抑心头不安与躁动以及急切的唯一办法。
<br/>　　咔咔…
<br/>　　萧逸的拳头，忽然握得劈啪作响。
<br/>　　在他的这般思索中，晨光，渐渐散去。
<br/>　　在他那‘安静’的身躯下，时间似乎过得很快。
<br/>　　下午的旭日，不多时便化作了余辉。
<br/>　　余辉的挥洒，在他脸庞上停留了一小会儿。
<br/>　　之后，夜色再次降临。
<br/>　　那张夕阳浅留的微暖面庞，忽然化作了冷漠，以及一丝挥之不去的坚毅。
<br/>　　“入夜了，该动身了。”萧逸自语一声，结束了打坐。
<br/>　　一缕缕金光，再次在这金光险地中飘荡而起。
<br/>　　不多时，金光，笼罩了整片险地。
<br/>　　一只只平静了一日的妖兽，忽然躁动起来。
<br/>　　一股股强猛的妖兽气息，陡然充斥整个险地。
<br/>　　平静了一日的金光险地，陡然暴走。
<br/>　　树干上，那道傲然的身影，缓缓站起。
<br/>　　身影一闪，向那险地深处，疾速前行。
<br/>　　......
