　　五、六十里路，快马加鞭之下一个多时辰就可以赶到。
<br/>　　所以李承乾和称心两人赶到玉山山脚下的那个茶摊的时候，天色距离午时竟然还差着将近一个时辰。
<br/>　　因为时间已经是九月初，并不怎么热，加上现在时间还有些早，所以茶摊上的人并不多，只是其中一个头戴斗笠的家伙，格外的引人注意。
<br/>　　李承乾与称心对视一眼，心照不宣的同时将目标定在了那个斗笠人身上。
<br/>　　“太子殿下果然仁义，竟然比胡某料想的早到了半个时辰。”李承乾与称心刚刚坐到那个斗笠人对面，斗笠人就有了反应，同时也轻轻的抬起头来。
<br/>　　“胡老八？”李承乾瞬间就从脑海中将这个人的资料翻了出来。
<br/>　　“太子殿下好记性，竟然会记得胡某。”斗笠人咧嘴笑笑，带着一丝嘲讽说道。
<br/>　　“这事儿是李元昌让你做的？”李承乾既然能认出胡老八，自然知道他是汉王李元昌的贴身护卫，所以自然而然的就想到是不是李元昌搞出来的事件。
<br/>　　“我说不是，殿下信么？”胡老八旋转着手中的茶碗反问道。
<br/>　　“我妹妹在哪里？”李承乾决定先不考虑李元昌的事情，直接问起长乐的情况。
<br/>　　不管这一次是谁想搞事，等最终结果出来了，幕后主谋也就出来了，现在问多少显得有些多余。
<br/>　　“公主殿下很安全，殿下不用担心。”胡老八嘴角的嘲讽依旧，看看李承乾，又看看称心：“不过殿下果真是多情的种子啊，选个随行之人，竟然会把这个兔爷带在身边。”
<br/>　　李承乾咬了咬牙，给旁边的称心打了一个稍安勿躁的眼色，冷声说道：“废话少说，本宫要见长乐！”
<br/>　　“福祸本无门，惟人自召取，这话果然不错。”胡老八假模假式的叹了口气，然后起身说道：“既然殿下那么急着去见妹妹，那就跟我来吧。”
<br/>　　“你不说去什么地方，我们不会去的。”称心一把按住想要起身的李承乾，十分女性化的声音响起。
<br/>　　“你一个兔爷，有什么权利代替你的主子说话，去不去还是问问太子殿下吧。”胡老八有些不屑的嘿嘿一笑，对着称心挑了挑眉毛：“待会儿让你尝尝不一样的滋味如何？”
<br/>　　“无耻！”称心几乎快要被气炸了肺，如果不是因为一直没有见到长乐，他恨不得抽出刀子直接弄死这个胡老八。
<br/>　　“别说了，跟他走。”半晌没有说话的李承乾深深了吸了口气，反手压住称心。
<br/>　　“这才对嘛，公主是那么容易见的么？”胡老八并不在乎称心的怒视，撇撇嘴走向茶摊外面栓着的几匹马。
<br/>　　李承乾两人自然随后跟上，一行三人就这样从茶摊离开，顺着官路又走了一段之后，拐上了一条山坳中的小路。
<br/>　　“一会儿发现情况不对，你立刻就走，明白么？”眼看着路越走越荒，李承乾小声的叮嘱称心。
<br/>　　“不行，我……”
<br/>　　“这是命令，总不能我们两个都陷在这里，到时候连个报信的都没有。”李承乾沉声打断称心。
<br/>　　路上的马蹄印并不少，这证明目的地一并不会只有胡老八一个人，这样的情况下，称心根本不可能一个人护住他和长乐两个，所以让他离开，使胡老八投鼠忌器才是最好的选择。
<br/>　　“太子殿下还是不要白废力气了。”骑马走在前面的胡老八听到了李承乾的安排，回头说道：“本来让你带一个随从是怕你胆子小不敢来，想让你带个高手护身，没想到你胆子竟然这么大，竟然只带了一个兔爷出来。”
<br/>　　李承乾给称心打了一个眼色，示意他一会儿照办，然后冷笑一声说道：“少给本宫来这套，本宫不信你敢冒着抄灭九族的罪名来杀本宫。”
<br/>　　“九族？老子天生天养的，自己都不知道爹妈是谁，如果皇帝陛下帮老子屠了他们，老子还真要去谢恩呢！”胡老八嗤了一声，语带怨毒的说道。
<br/>　　话说到这份儿上，已经没什么可说的了，李承乾干脆闭嘴不言，与称心两人就这么跟着胡老八一路前行。
<br/>　　直到大概半个时辰之后，才在一处峭壁之下停了下来，路也在此时到了尽头。
<br/>　　“人呢？”李承乾一边向四周打量，一边问着。
<br/>　　实际上他已经作好准备，如果这是一个引他上当的骗局，立刻就掏枪作掉这个该死的胡老八。
<br/>　　“别急嘛，走了这么长时间，总要让老子喘口气！”胡老八拿出一只竹哨，嘴里不干不净的说着，同时吹响了哨子。
<br/>　　很快，就在小路的尽头一块石头后面转出三个人来，个个长的孔武有力，一看就是练家子。
<br/>　　“赖子，带我们的太子爷去看看公主。”胡老八一双贼眼死死盯着称心，口中随意的对三个汉子中的一个秃头说道。
<br/>　　“好喽！”秃头答应一声，对着李承乾勾勾手指：“过来吧，太子殿下。”
<br/>　　“走。”李承乾朝称心偏了偏脑袋，当先走了过去。
<br/>　　三个汉子，如果出其不意的话，李承乾估计自己可以捅死或捅伤一个，到时候称心再挡住一个，另一个完全可以用火铳搞死。
<br/>　　到时候只有胡老八一个，估计怎么弄出能搞定了。
<br/>　　结果跟着三人转过大石之后，李承乾愕然发现，石头后面竟然别有洞天。
<br/>　　那里的山壁上竟有一个天然的石洞，洞并不深，一眼就可以看到头，在山洞里，长乐双手被绑在身后，头发有些凌乱，正坐在一块石头上啜泣着。
<br/>　　而在长乐的身后，则是另一个汉子，拿着出鞘的横刀百无聊赖的磨着。
<br/>　　“大妹！”看到长乐的一瞬间，李承乾伸手在称心的胳膊上不着痕迹的拍，示意他见机行事。
<br/>　　“哥？！”低头啜泣的长乐听到李承乾的声音陡然抬头，以为是自己听错了，疑惑的向洞外看去，接着她就看到李承乾推开身边的一个汉子，冲了进来。
<br/>　　“哥！”像是看到救星一般，长乐猛的从石头上站起来。
<br/>　　“坐下！”雪亮的横刀唰的一下从长乐的前面掠过。
<br/>　　那个磨刀的汉子不知什么时候已经站了起来，拿着横刀挡在了李承乾与长乐中间。
