　　高空凛冽罡风中，萧逸御空飞行着。
<br/>　　呼啸而来的罡风，并未能伤他分毫。
<br/>　　天元境后期以上的武者，能在极寒之地的高空随意飞行。
<br/>　　并非天元境后期有着什么特殊手段。
<br/>　　而仅仅是因为天元境后期，元力外溢到小世界表面，让小世界变得更为强大了。
<br/>　　御空飞行，本就是武者达到洞玄境后，拥有小世界，才获得的手段。
<br/>　　天元境后期，小世界变强了，御空飞行的能力，便也强了。
<br/>　　顺带一提。
<br/>　　萧逸之前离开极寒裂缝，追击前往寒冰城。
<br/>　　一开始，本是打算用紫炎火翼，增加飞行速度。
<br/>　　紫炎，可将高空中吹袭而来的极寒罡风焚毁殆尽，让得自身飞行速度大增。
<br/>　　可后来，萧逸发现，正常飞行，借助冰雪剑主的操控风雪之道，反而速度更快。
<br/>　　一是焚风而行，一是借风而行，孰快孰慢，一目了然。
<br/>　　故萧逸后来前往寒冰池，一直使用正常的飞行，以风雪之道辅之。
<br/>　　现在也是如此。
<br/>　　当然了，这只适用于极寒之地这边的高空罡风。
<br/>　　若是去了别的地方，又反倒是紫炎火翼的爆发速度更快。
<br/>　　言归正传。
<br/>　　萧逸一路御空飞行，目的地，乃是…极寒裂缝。
<br/>　　不错，他要回极寒裂缝。
<br/>　　现在冰心圣果已夺回，得替易老治疗了。
<br/>　　冰心圣果，只此一枚，出不得差池。
<br/>　　萧逸必须寻一个百分百安全的地方。
<br/>　　极寒之地，处处是危机；加之冰武王国方面，也不知是否会追击而来。
<br/>　　唯一安全的地方，只剩极寒裂缝。
<br/>　　自从猜测出冰承天等人要炼制冰极丹，萧逸就知道，事情不简单。
<br/>　　冰极丹，可并非寻常丹药。
<br/>　　位列八品，且哪怕在八品中，都是比较难炼的丹药。
<br/>　　整个炎武王国，怕是只有苏震玄这个八品炼药师有本事炼制，还不一定成功。
<br/>　　而冰武王国，能炼制的，只有一人，正是冰武国主。
<br/>　　说起这冰武国主，乃是个惊才绝艳之辈。
<br/>　　自执掌王位以来，雄韬伟略，尽显王者风范。
<br/>　　而更惊人的是，他本身亦是个了不得的武者。
<br/>　　武道、炼药一道，尽皆声名赫赫。
<br/>　　萧逸轻呼一口气，没再想此事。
<br/>　　要炼制冰极丹，必须要有冰心圣果。
<br/>　　但冰心圣果，是现今救易老的唯一办法，谁也休想觊觎！
<br/>　　……
<br/>　　两天后，萧逸一路疾速飞行，来到了极寒裂缝。
<br/>　　身影一跃而下。
<br/>　　仍旧是那黑暗、冰冷、罡风袭面的感觉。
<br/>　　萧逸轻车熟路般，回到了那山洞里头。
<br/>　　手中乾坤戒光芒一闪。
<br/>　　寒晶冰棺，凭空而现。
<br/>　　里头，沉睡着一位老人。
<br/>　　哪怕沉睡着，仍旧难掩他神色中的桀骜，以及不羁。
<br/>　　萧逸笑笑，同时取出了冰心圣果。
<br/>　　眼中，尽是期待之色。
<br/>　　张开易老的嘴巴，萧逸将冰心圣果缓缓喂入。
<br/>　　待得整颗冰心圣果进入易老腹中。
<br/>　　一道澎湃得惊人的生机力量，陡然间从易老身上爆发。
<br/>　　萧逸脸色一喜，连忙放出感知，并伸出一只手，搭在易老胸膛。
<br/>　　易老如今处于无意识的状态。
<br/>　　萧逸自然要引导这股庞大的生机力量，以治疗易老的身躯。
<br/>　　在萧逸精准而完美的控制下，整颗冰心圣果的力量，没有被浪费半分。
<br/>　　悉数被易老的身体所吸收。
<br/>　　那一道道游走于四肢百骸的精纯生机力量，让萧逸脸上的笑容，愈发浓郁。
<br/>　　易老冰冷的身体，渐渐出现了些许温暖。
<br/>　　原本已经感知不到的武者气息，此刻，渐渐涌现。
<br/>　　易老的生机，正以惊人的速度，恢复着。
<br/>　　“不愧是冰心圣果，位列九品，誉为圣果，果然厉害。”萧逸赞叹一声。
<br/>　　然而，不过几秒，萧逸脸上的笑容，忽然戛然而止。
<br/>　　“嗯？”萧逸皱了皱眉头。
<br/>　　易老身体中，不断恢复的生机，竟忽然在缓缓消散。
<br/>　　“给我回来。”萧逸冷喝一声。
<br/>　　在他的引导下，生机力量的消散速度，慢了下来。
<br/>　　但，又是数秒过后。
<br/>　　生机力量，消散的速度，再度变快。
<br/>　　易老的情况，竟陡然剧变。
<br/>　　“怎么回事？”萧逸大惊失色。
<br/>　　双手齐出，强行留下这些消散的生机力量。
<br/>　　生机力量的消散，再次变缓。
<br/>　　但，一分钟后，他的这些引导，渐渐变得无力了起来。
<br/>　　生机力量的消散，一直在持续着。
<br/>　　且，速度越来越快。
<br/>　　每一分生机速度的消散，易老身上的气息，便减弱一分。
<br/>　　好不容才恢复的气息，渐渐开始变得微弱。
<br/>　　“给我留下来。”萧逸咬着牙，神色有些狰狞。
<br/>　　但，任凭他如何努力，手段尽出。
<br/>　　仍旧无法阻拦生机力量的消散。
<br/>　　时间，渐渐过去。
<br/>　　易老的气息，再次趋向于微不可闻。
<br/>　　生机力量，几欲散尽。
<br/>　　萧逸的心头，在滴血，却无可奈何。
<br/>　　这种无力感，让他感到愤怒，感到悲伤，感到着急。
<br/>　　乃至抓狂。
<br/>　　半晌，易老的生机，再次散尽，恢复了以往的冰冷。
<br/>　　“为什么…为什么…”萧逸喃喃自语着，失神着。
<br/>　　咔咔。
<br/>　　无力的身躯，跌坐在地面。压着了地面无数的白骨，发出咔咔声响。
<br/>　　人力有时穷，生死不可违。
<br/>　　裂天剑姬只说过，冰心圣果，或许能救易老。
<br/>　　并未百分百保证。
<br/>　　萧逸苍白着脸，比地面上这些白骨，更白。
<br/>　　半晌，他摇了摇头，站了起来。
<br/>　　他不信。
<br/>　　更不愿相信。
<br/>　　一双有力的手臂，再次搭在了易老的胸膛上，认真地感知着。
<br/>　　半晌，他明白了过来。
<br/>　　冰心圣果的生机力量，之所以消散。
<br/>　　是因为它们不属于易老。
<br/>　　或者更简单来说，是易老无法留住这些生机力量。
<br/>　　当初，在炎武卫一拳洞穿易老身躯时。
<br/>　　萧逸接过易老的身躯，那时，他的判断是，易老已经死了。
<br/>　　其实，萧逸当初并没有判断错误。
<br/>　　从炼药师或者正常的角度来说，易老确实死了。
<br/>　　之所以裂天剑姬前辈说易老没死，是因为易老身上还有一丝生机。
<br/>　　只是，这丝生机，如同大海中的一滴水滴，渺小至极。
<br/>　　而正常来说，这一丝生机，对于正常武者来说，是不可能存在的。
<br/>　　哪怕裂天剑姬那等手段，也无法为其封存。
<br/>　　真正保下了这丝生机的，是易老本身小世界中的大自在剑道的武道力量。
<br/>　　这些武道力量，让这丝生机能够短暂再寄存一段时间。
<br/>　　才让裂天剑姬前辈有了时间，为其封存。
<br/>　　可，这又如何？
<br/>　　易老无法苏醒，又有何意义？
<br/>　　“弟子，让你失望了。”萧逸无力地跌落在地，喃喃说着。
<br/>　　......
<br/>　　第二更。
