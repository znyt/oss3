　　小马又挑了一个看上去身强体壮的汉子，附在他身上，让人赶紧上酒菜。
<br/>　　于是一些见都没见过的菜，被一道道地从后堂送过来。小马大快朵颐，招呼叶少阳和吴嘉伟一起吃。
<br/>　　“这些食物，都是从哪里来的？”叶少阳吃着一只烧乳鸽，嚼得满嘴喷香，突然就想到了这个问题。
<br/>　　“养的呗，总不能是变出来的。”
<br/>　　“这个空间才造出来不久，你要说这些动物是从人间抓来我也认了，这些粮食是从哪里来的？麦子稻子什么的，都得小半年才能长成吧，那时候这个世界还没有开拓出来吧？”
<br/>　　这问题小马答不上来，叶少阳怀疑，这些食物都是从人间运来的，虽然野外也有庄稼地和各种菜地，但多半是第一茬，这个世界的生灵虽然不多，但每天也要吃用消耗，再大的储备量都是不够的。
<br/>　　叶少阳据此猜测，这个世界跟人间多半有连接点，从那里不断将粮食之类的物资运送进来。不过这个连接点一定掌握额在圣灵会的手中。
<br/>　　那些大臣并没有放弃营救皇帝，想了很多点子，除了下毒，还利用上菜的时候行刺小马，都没用，也只能站在宫殿下面干瞪眼。
<br/>　　“少阳，而今我们已经控制了皇帝，现在最重要是利用他的影响力，号召军队服从我们，将来如果找到圣灵会的老巢，至少人数上占优势。”碧清在一旁出谋划策，她虽然不喜欢动脑子，但最近看了太多的古装剧，这方面的主意倒是很多。
<br/>　　叶少阳思索了一下，走到皇帝面前，一只手搂住他的肩膀，皇帝吓的一哆嗦，可怜兮兮地看着他。
<br/>　　“别怕，我不打你。跟你商量个事啊。我们呢，也不想做皇帝，你如果能配合我们，等事情结束之后，你继续做你的皇帝怎么样？”
<br/>　　皇帝怔怔地看着他，“真的？”
<br/>　　“骗你干啥，我们要是想当皇帝，刚就可以把你杀了，留你做什么。”
<br/>　　皇帝立刻点头，表示愿意合作。
<br/>　　叶少阳觉得好笑，果然还是假皇帝，这要是真的嘉靖皇帝，估计就算落到这步田地，也会保持天子威仪。
<br/>　　“起开！”
<br/>　　叶少阳上去把小马从龙椅上踹下来，把皇帝扶上去坐着。
<br/>　　“这这这……合适吗？”皇帝吓的两腿哆嗦。
<br/>　　“这就是你的座位啊，怕什么。”
<br/>　　皇帝这才坐了，叶少阳又让宫女上茶，给皇帝压压惊。
<br/>　　皇帝喝了茶，稍微缓过来一点，看着站在下面的那些文武百官，觉得之前真的太丢人了，想要安慰一下，于是干咳两声，硬生生挤出来几句安慰的话，那些官员脸色倒也好看了些，有个文官试图跟叶少阳谈判，叶少阳不理他，在碧清的建议下，让皇帝下令，把几个掌军权的将军都叫来。
<br/>　　皇帝不敢不从，于是又太监出下去传令，有四个将军就在朝堂上，不知所措地走上前来。
<br/>　　叶少阳带着他们穿过侧门，正好边上是一个小房间，里头挤着一堆宫女太监，叶少阳把他们赶出来，看里头都是茶壶痰盂什么的，应该是个茶水间，于是把几个将军叫进去，一个个检查，发现他们都中了抽魂术，于是让他们站着不要动，逐个为他们化解了符印。
<br/>　　然后趁他们不注意，在他们脑后一人给了一巴掌，略微用了点真气，当然几个人都软软地倒在地上抽搐起来，很快，就听见他们身上的盔甲被撑得咯吱咯吱响，几个人相继都现出了真身：一个是蛇妖，一个是鬼魂，另一个是人类，还有一个居然是鱿鱼精。
<br/>　　不等他们四个发狂，叶少阳把准备好的灵符一人在他们脑门上贴了一张，先将他们定身，一个个躺在地上，瞪大眼睛看着叶少阳。
<br/>　　叶少阳又作法驱散他们身上的戾气，等他们恢复意思，大致把情况跟他们讲了一遍，四个人听的眼睛发直。
<br/>　　“好了，现在你们也知道怎么回事了，想不想回地球——呸，想不想回我们自己的世界？想的话就眨眼。”
<br/>　　四个邪物一起眨眼，眨个不停。
<br/>　　“喏，你们在这里的身份，都是将军，手上都是有兵权的，你们要配合我的行动，帮我寻找法术公会的老窝，一举给他们端了，只有这样才能回去，谁都不许掉链子，愿意合作的再眨眨眼睛。”
<br/>　　四个邪物又拼命眨眼。
<br/>　　叶少阳这才满意地帮他们揭掉脸上的灵符。
<br/>　　“掌教师伯！”
<br/>　　其中那个人类立刻激动地叫起来。
<br/>　　“呃？”叶少阳一惊，那人立刻跪下，激动地说道：“给掌教师伯磕头了，师伯，我可算是见到你了。”
<br/>　　“你是……”
<br/>　　“我叫清波，是茅山弟子啊，我师父就是您师弟啊，师伯，呜呜，见到你就好了……”
<br/>　　“呃，你是小苏的徒弟？”
<br/>　　“是了，我拜师半年，之前您回山的时候，还见过一面，我还给您倒了杯茶，您肯定不记得我了，师伯啊，徒儿总算见到你了……”
<br/>　　清波激动地痛哭起来。
<br/>　　这里居然还有自己茅山的弟子！
<br/>　　叶少阳也很兴奋，仔细打量他，这小伙十八九岁，天庭饱满，命格修长，双眼也格外有神，倒是个修道之材，定了定神道：“你先别哭，你告诉我怎么回事，你怎么会在这？”
<br/>　　“启禀师伯，是家师派我来的。”
<br/>　　清波简单说了情况，原来这承德的异动，几大门派都发现了，私下里也串联了一下，各自也都派了弟子过来调查，清波虽然拜师时间不长，人也不够聪明，但为人还算老实，办事靠谱，而且苏钦章看出他有仙缘，于是派他过来走一趟，本想着查出什么线索，再跟叶少阳汇报的，结果跟几个法师一起下古墓，再也没能出来。
<br/>　　跟其余被卷进来的人一样，刚到这个世界，他们立刻就被修改了记忆，什么都记不住了……
<br/>　　叶少阳拍了拍他的肩膀，道：“没啥，你以后反正跟我混吧。”
<br/>　　清波十分高兴，说了一些赴汤蹈火之类的话。
