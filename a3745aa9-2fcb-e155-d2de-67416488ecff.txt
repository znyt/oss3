　　在摧毁了那一道锋芒之后，千丈剑光，更是向着那无道斩杀而去。
<br/>　　“什么？”
<br/>　　无道一惊，没想到他打出的锋芒，居然这样被楚风眠所化解，连连向后退去。

<br/>　　这一枪之力打出一刻，天空之上都是传来阵阵爆裂之声。

<br/>　　鹰鸣响彻而起，这一枪的锋芒，如同一只雄鹰，冲天而起，向着楚风眠咆哮而来。
<br/>　　“少宗主的枪法，更加精湛了！”
<br/>　　“就凭这小子，也想要跟少宗主交手？简直是做梦！”
<br/>　　“哼，什么圣龙宗的天才，在少宗主的面前，算得了什么！”
<br/>　　“看来这一次少宗主的动怒了，这小子今日，必死无疑！”
<br/>　　在场圣院宗圣子，都是纷纷议论道，许多人都是冷笑连连的看着楚风眠，等待看楚风眠是如何死在无道枪锋之下的。

<br/>　　砰！
<br/>　　两股力量在空中交锋的刹那之间，便是显现出了结果，那无道打出的枪锋，在这力量碰撞之间，轰然破碎。
<br/>　　反震的力量，让无道都差点拿不稳手中的长枪，连连退后数步，在他的眼神之中露出了不可思议的光芒。
<br/>　　“这？楚兄的实力，怎么又变强了这么多？”
<br/>　　燕皇刚刚趁机疗伤完，苏醒过来，便是看着眼前这一幕，不由的目瞪口呆道。
<br/>　　他虽然一直知道楚风眠实力的强横，可没有想到，在这短短的时间之内，楚风眠实力的提升居然又这么大。
<br/>　　身为圣院宗少宗主的无道，跟楚风眠交手，居然就被楚风眠这样轻易的打退了出去？
<br/>　　无道的实力，足以称得上是圣龙秘境九劫古帝之中，最为强横的一位了，不然这圣院宗少宗主的位置，也轮不到他。
<br/>　　之前青梦跟燕皇联手，都难以反抗这无道的力量，这种强横的实力，无道在九劫古帝之中，已经是近乎堪称无敌的存在了。
<br/>　　可就是这样的无道，居然是被楚风眠这样轻易的逼退？
<br/>　　甚至看起来，楚风眠就连剑术都没有动用。
<br/>　　仅仅只是随手一剑，就蕴含着如此强横的力量，逼退了无道。
<br/>　　青梦的目光，也是看向楚风眠，她的目光之中，也是无比复杂，楚风眠现在的实力，就算是她，都无法看得出来了。
<br/>　　“只有这种实力吗？这种实力，也敢打洛汐的主意？”
<br/>　　楚风眠看着无道，眼神之中的不屑之意，毫不掩饰。
<br/>　　“小杂种！”
<br/>　　看到楚风眠这不屑的眼神，无道的心中一股无边怒火爆发而出，一种耻辱的感觉占据了他的心头，让他简直要发狂。
<br/>　　他可是无道，圣院宗的少宗主，圣院宗历史以来最为耀眼的天才，什么时候受到过如此轻视。
<br/>　　无道看向楚风眠的眼神之中，已经是将他心中受到的屈辱，化为了对楚风眠的杀意，眼中的杀意，恨不得现在就要将楚风眠撕碎。

<br/>　　无道脸色狰狞到了极点疯狂的咆哮道。
<br/>　　“你以为这就是我的全力？本来我还打算好好玩弄玩弄你，既然你自己找死，那我先成全了你！”

<br/>　　无道一声爆喝，只见他的身上，一股青雾缓缓凝聚而出，围绕在他的身边，瞬息之间，无道身上的气息，俨然是跟之前，截然不同。
<br/>　　这种力量，已经是半步天人境的层次！

<br/>　　看着无道身上力量的变化，楚风眠不由的目光打量了起来。
<br/>　　这无道的境界，应该是九劫古帝不错，但是他的身上，却是爆发出了一种独特的秘术，将他的实力，硬生生提升到了半步天人境的层次。

<br/>　　无道，在听到楚风眠这不值一提四个字的时候，嘴角都是浮现出了几分戏虐的笑容，充满杀意的双眼，变的更加的冰冷。
