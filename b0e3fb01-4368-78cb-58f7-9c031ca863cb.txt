        “如果，真是这样的话。我肯定会跟你一样，得一辈子关在这里了。”童筱颖泄气道。
<br/>
<br/>        就算傅夜沉想来救她，一定会有一大把人跳出来，纷纷阻拦他吧！
<br/>
<br/>        其实，她也不指望傅夜沉会来救她，原因很简单，因为连华生一而再再而三地警告过她，傅夜沉的命，比她童筱颖的命更重要。
<br/>
<br/>        这里戒备这么森严，外面的人，进来就是死路一条，救她的希望太渺茫。
<br/>
<br/>        “不过，你放心，你是婴宁的亲妹妹，阿十不会不管你。”季雅珊苦笑，转瞬间又给了童筱颖一丝希望。
<br/>
<br/>        三年前，十爷抓着童筱颖去医院做血缘关系鉴定，想要鉴定童筱颖和婴宁的之间是否有血缘关系。
<br/>
<br/>        那个时候，季雅珊也在医院里，她买通了鉴定科的医生，不管鉴定结果如何，她都让那个出报告的医生写明被鉴定的两人是姐妹关系。
<br/>
<br/>        当时，她只是出于对童筱颖的保护，没想到这个“护身符”，至今还能用。
<br/>
<br/>        至于，童筱颖和婴宁到底是不是亲姐妹，其实她季雅珊也不知道，因为事后，她没有再联系那个医生了。
<br/>
<br/>        “你对这里的事情，知道多少？”童筱颖下意识地想要从季雅珊嘴里套话。
<br/>
<br/>        季雅珊抿了抿唇，微微摇了摇头，接着说道：“关于这里的事情，你不知道比知道要好。”
<br/>
<br/>        “那你可以不可借手机给我用一下？”童筱颖又问道。
<br/>
<br/>        季雅珊苦笑：“我已经有三年没用过手机了。”顿了顿后，她又接着说道，“你放心，今晚阿十会回来。等他回来了，我就告诉他，你被八爷抓来了。他一定会让八爷放你回去的。”
<br/>
<br/>        “谢谢你。”童筱颖一时间不知道该说什么好，这是季雅珊第二次救她了。
<br/>
<br/>        “我们是好朋友，对不对？”季雅珊忽然很真挚地问道。
<br/>
<br/>        童筱颖微微一笑，点了点头。
<br/>
<br/>        “那就不用谢我啊！我救你，是应该的！”季雅珊嘴角微扬，轻轻地拍了拍童筱颖的肩膀，接着说道，“我会把你的一日三餐都送上来，那张小门过去是厕所，你乖乖待在阁楼里，不要到处乱跑。古堡里不安全，若是撞见了不该撞见的人或事，会被灭口的。你懂了吗？”
<br/>
<br/>        童筱颖尴尬地笑了笑，再次乖巧地点了点头。
<br/>
<br/>        这一整天，童筱颖真就乖乖待在阁楼里没出去。
<br/>
<br/>        不知道为什么，她对季雅珊有种说不出的信任感，她相信她的话，认为她不会伤害她。
<br/>
<br/>        吃过晚饭后，童筱颖无聊的趴在阁楼的窗台前，看着日落西山，夜幕降临。
<br/>
<br/>        不知过了多久，望着满夜繁星，她有点儿思念大哥了。
<br/>
<br/>        她突然失踪不见了，不知道大哥会不会很担心？
<br/>
<br/>        他一定很着急吧！
<br/>
<br/>        可是，她无法给他报平安。
<br/>
<br/>        此刻，童筱颖觉得自己一点睡意也没有，但是不得不强迫自己爬上那张单人床，然后盖上被褥，闭上眼睛。
<br/>
<br/>        说不定，一觉醒来后，明早她又回到了傅家大庄园。
<br/>
<br/>        童筱颖翻来覆去，仍旧有些睡不着。
<br/>
<br/>        其实，她不是因为认床而睡不着，而是因为在这里让她无法安然入睡。
<br/>
<br/>        突然，一阵清脆的钢琴声传来。
<br/>
<br/>        童筱颖倏然地坐起身子，莫名其妙地竖起耳朵来仔细地聆听。
<br/>
<br/>        这首曲子充满悲伤与凄凉……
<br/>
<br/>        她听得正入神时。
<br/>
<br/>        钢琴的声音戛然而止，童筱颖才从沉醉中惊醒。
<br/>
<br/>        “是谁在弹钢琴？”她自言自语着，又倒回床上，睁着双眼，仍旧无法入睡。
<br/>
<br/>        刚才的钢琴旋律依然在脑海里回荡不去……
<br/>
<br/>        童筱颖翻来覆去，最后将枕头抱在怀里侧躺着，心里怀念着躺在大哥傅夜沉怀中的安全感。
<br/>
<br/>        大哥……
<br/>
<br/>        快来救我……
<br/>
<br/>        我想你了……
<br/>
<br/>        她睡得迷迷糊糊之际，忽然有一双手臂环上了她的腰，而她的臀部正紧贴着一双结实的大腿。
<br/>
<br/>        童筱颖“啊”地一声，猛地从惊慌中彻底苏醒过来，完全不知道自己是什么时候睡着的，更不知道此刻身旁这人是谁。
<br/>
<br/>        她刚想从床上下去逃跑，却又被这人一把抱住腰肢，给压回了床上。
<br/>
<br/>        童筱颖浑身冷不丁地打着寒颤，心悸得额头渗出细密的冷汗。
<br/>
<br/>        对方的手将她抱得很紧，死死地将她箍在怀里让她没有逃脱的机会。
<br/>
<br/>        此时，两个人的脸离得很近，呼吸也混在一起。
<br/>
<br/>        屋内一片漆黑，她看不清楚对方是谁。
<br/>
<br/>        童筱颖拼命地推，却推不开，反而被对方轻易扣住双手，压向了头顶。
<br/>
<br/>        下一刻，对方突然把脸埋入了她的颈窝上，吻起了她的耳朵和脸颊。
<br/>
<br/>        “轰”地一声，童筱颖整个人懵了。
