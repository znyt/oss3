　　屏障之内，显然别有洞天日月。
<br/>　　就这般从外而内远远看去，俨然是一片巨大的森林。
<br/>　　其内，各种参天之树，密密麻麻，山川湖泊，穿插其中。
<br/>　　忽然，一眼尖的弟子惊呼一声，“那是…天材地宝？好…好多。”
<br/>　　众人细细看去，森林边缘，一株株天材地宝，漫山遍野，生机气息浓郁惊人。
<br/>　　“安静。”游执事双手虚压，示意各院弟子安静。
<br/>　　全场安静下来。
<br/>　　游执事沉声道，“天藏秘境内，自成一方天地。”
<br/>　　“其内苍木延绵，辽阔范围不亚于一个地域。”
<br/>　　“里头有妖兽，亦有各种天材地宝。”
<br/>　　“斩杀妖兽无分，摘取收获天材地宝有分，品阶越高，分数越多。”
<br/>　　“第二轮，团队积分赛，最后根据各支队伍分数高低给出排名。”
<br/>　　这时，一个弟子脱口而出，问道，“那些天材地宝？”
<br/>　　游执事看了这个弟子一眼，道，“采摘下来后，便归你们自己所有。”
<br/>　　周遭天骄闻言，眼睛霎时一亮。
<br/>　　游执事继续沉声道，“第二轮比拼，共计10日。”
<br/>　　“稍后，你们会通过秘境阵法，随机出现在阵法内的任何地方。”
<br/>　　“秘境之内，自有危险。”
<br/>　　“若尔等不敌，可捏碎自己随时携带的学院令牌。”
<br/>　　“令牌上，已有阵法，与秘境阵法相连，可立即离开秘境，回到此山巅。”
<br/>　　事实上，第二轮的详细规则，各学院的长老早便解释过。
<br/>　　游执事现在只是大概阐述一遍。
<br/>　　至于捏碎令牌的规则，那是各院弟子的护身手段。
<br/>　　若遇不敌之危，可立刻捏碎令牌，回到此山巅。
<br/>　　但同时，这个弟子在捏碎令牌的那一刻起，便等同弃权，身上无论有无积分，都算作0分。
<br/>　　“各院队伍，出列。”游执事低喝了一声。
<br/>　　各院队伍，纷纷出列，站到秘境屏障之外。
<br/>　　“有劳诸位长老了。”游执事看向一旁各位长老。
<br/>　　几位学宫长老点了点头。
<br/>　　他们来此的目的，除了开启天藏秘境外，也是协助天藏秘境的阵法运行。
<br/>　　嗖…嗖…嗖…
<br/>　　一阵阵光芒闪烁。
<br/>　　不过片刻，各院弟子悉数消失在原地。
<br/>　　数位学宫长老大手一挥，面前亦瞬间消失。
<br/>　　各院队伍，已悉数进入天藏秘境。
<br/>　　“诸位学院长老。”游执事看向各学院的长老，拱了拱手。
<br/>　　“天藏秘境屏障已关闭，待得十日后，第二轮比拼结束，屏障会再次开启。”
<br/>　　“这段时间，便劳烦诸位在此等候了。”
<br/>　　……
<br/>　　天藏秘境内，这座辽阔无比，不亚于一个地域的巨大森林，不知已存在了多少岁月。
<br/>　　参天之树，数之不尽。
<br/>　　此时，随着一道道陌生气息出现于森林的一个个角落，古老秘境的静谧，瞬间被打破。
<br/>　　一位位天骄，刚刚站稳身体，已面露警惕之色。
<br/>　　虽秘境之内极大，三千天骄分布其中如若江河水滴。
<br/>　　但能进入第二轮的天骄，哪个不是实力过人。
<br/>　　所谓的辽阔范围，对他们而言不过是以‘时辰’而论的短暂距离。
<br/>　　第二轮比拼，名唤积分赛，自然是以积分为目标。
<br/>　　所谓积分，遍布秘境内的天材地宝算积分，品阶越高，积分越多。
<br/>　　参加比拼的弟子，亦算积分，不过无论弟子强弱，积分一样。
<br/>　　简单来说，各院弟子间，需要靠自己的本事去寻天材地宝。
<br/>　　秘境内，本就有妖兽。
<br/>　　而同时，各院弟子还需提防别的对手，否则很可能自己寻到的天材地宝，只是为他人做嫁衣。
<br/>　　当然了，自己本身也是积分之一。
<br/>　　寻、防，这对天骄自身的能力有不弱的要求。
<br/>　　另外一点，根据规则，第二轮积分赛，各院队伍只需积蓄足够的积分，即可进阶。
<br/>　　所谓的淘汰规则，乃是指那些直到十日后比拼结束，仍旧是0分，或者分数未积蓄够的学院队伍。
<br/>　　故，各院队伍，完全可以在获得足够的积分后便躲起来，以避免战斗，避免积分被躲，等候十日后秘境屏障再次开启时即可。
<br/>　　寻、防、躲，这无一不考验着各个天骄间的实力，甚至是整体水准。
<br/>　　十天时间，不短不长。
<br/>　　虽各天骄间暂未接触，但，这不过是真正激烈交锋前的短暂安宁。
<br/>　　一旦爆发，这将会成为整个百院之争中最激烈的战斗，且是混战。
<br/>　　……
<br/>　　此时，森林某处角落，一道百无聊赖的身影，无聊地四处逛着。
<br/>　　逛了不多时，身影干脆愤愤坐下。
<br/>　　坐了半晌，又忽然似是坐不安稳似地站起。
<br/>　　站了一会儿，身影又闲逛了一阵，最后干脆在靠近的一株参天大树上倒头便睡。
<br/>　　刚睡不多久，一阵鼾声传出。
<br/>　　恰在此时，参天大树之后，一道庞然身躯缠绕而起。
<br/>　　一双冰冷的巨大眸子，直视那树干上酣睡的身影。
<br/>　　嘶嘶…
<br/>　　泛红的信子，吐了吐。
<br/>　　很显然，这是一头巨大的蟒类妖兽。
<br/>　　吼…下一秒，一声嘶吼。
<br/>　　巨大的蟒蛇之口，疾速朝酣睡身影吞去。
<br/>　　本在大睡的身影，猛地双眼睁开，“老子睡个觉也惹你了？”
<br/>　　身影一把跳起，极快的速度下竟是远远超过了这等蟒类妖兽。
<br/>　　“霸王吞风蟒？”身影微微惊讶。
<br/>　　霸王吞风蟒，圣王境巅峰妖兽，体型庞大，凶狠至极。
<br/>　　“哼。”身影冷哼一声，面对袭来的巨大蛇口，丝毫不惧，反倒双手一抓。
<br/>　　有力的双手，一把抓住了蟒蛇头颅。
<br/>　　下一秒…轰…轰…轰…
<br/>　　巨大的霸王吞风蟒，在身影手中不断扔甩，巨大的身躯，直将周遭树木与地面撞了个粉碎。
<br/>　　数分钟后，身影才气喘吁吁地一掌轰下。
<br/>　　掌落，一道湮灭罡风一闪而过，霸王吞风蟒瞬间分尸而死。
<br/>　　恰在霸王吞风蟒身死之际，周遭树木被毁的不远处，一缕青色光芒陡然而起。
<br/>　　“嗯？”身影疑惑下，丝毫未顾是否会有危险，闪身而去。
<br/>　　待得看清那青色光芒，身影脸色一喜，“千风皇果，好家伙，难怪会遇到霸王吞风蟒这等妖兽。”
<br/>　　千风皇果，风属性珍稀天材地宝，品阶为皇品。
<br/>　　即便放眼这天藏秘境，这也是不可多得的珍稀天材地宝。
<br/>　　其积分之高，绝对可入这天藏秘境内各种天材地宝中前三之数。
<br/>　　“好东西。”身影脸色微喜，就要采摘而下。
<br/>　　恰在此时，不远处，一道魁梧身影快步而来。
<br/>　　“千风皇果？”魁梧身影看着前方散发着青色光芒的朱果，面露喜色。
<br/>　　但看到那道即将采摘朱果的身影时，则脸色一冷，“黑云学教的杂碎？还真是冤家路窄。”
<br/>　　身影皱了皱眉，转过身看向那魁梧身影，“石惊涛？”
<br/>　　不错，从不远处走来的魁梧身影，正是五痴学宫，力痴弟子，石惊涛。
<br/>　　而那即将采摘朱果的身影，正是青麟。
<br/>　　青麟瞥了石惊涛一眼，淡漠道，“老子现在心情不好，我劝你别来惹我，滚。”
