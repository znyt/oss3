　　第743章
<br/>　　真如果程大雷拎上斧头与陈梦交手，也未必会胜得这么快。就算二人在马上交手，程大雷也赢不了这么轻易。
<br/>　　但程大雷轻身功夫了得，匹夫剑又犀利迅速，陈梦只靠两柄斧头，完全挡不住程大雷的攻击。
<br/>　　程大雷一剑递出，指住陈梦的咽喉，距离不过三指，剑尖轻轻震荡。
<br/>　　陈梦宛若僵住了相仿，手里空举着两柄斧头，一动也不敢动。
<br/>　　“来来来，你不是有与本当家交手么，现在你认输不认？”
<br/>　　陈梦脸上青一阵红一阵，脸颊肌肉僵硬得宛若岩石。他不得不承认，自己败在了程大雷的剑下。他输得很憋屈，输得很不服，但输了便是输了。
<br/>　　步法不行，经验不足，招式简单，程大雷冷冷的笑着，对方这『绝世』也就仗着力气大而已。如今他已被程大雷看穿，同样境界，程大雷有一百种方法弄死对方。
<br/>　　在陈梦一愣神的光景，程大雷忽然收剑，先前进步，单手拎起陈梦的肩膀，往后重重一靠，直接将陈梦掀过头顶，重重砸在地上。
<br/>　　他挣扎着想要站起，程大雷的匹夫剑已抵住他的胸膛，给了他一个挑衅的眼神。
<br/>　　“来来来，看本当家敢不敢穿了你，将你做成烤羊肉串。”
<br/>　　陈梦一张脸已经变成猪肝色，他很想翻身站起，再和程大雷大战三百回合。可程大雷的剑就抵着他的胸口，只要他敢动一动，程大雷就敢穿了他。
<br/>　　毕竟，再硬的铁汉心脏也是软的。
<br/>　　马孟起看着这一幕，忽然笑笑：“我就知道你小子不行，来来来，轮到我了，让我与你分个高低。”
<br/>　　程大雷的心情已经很不开心，他抬起头，冷冷瞥了对方一眼。
<br/>　　“你也配。”
<br/>　　马孟起不得不收起脸上的笑容，从程大雷眼里他感受到了蔑视。抱着混出头的念头来到长安，阴差阳错成为李乐天的近臣，他以为没人再敢轻视他，日后便是出名出名不停的出名，直到整个天下再无人不知道自己的名字。
<br/>　　却原来，还是有人会蔑视自己。
<br/>　　来自这个至强至恶之贼。
<br/>　　马孟起没再说话，他本身就是个不擅长说话的人，他更习惯用自己的武器说话。
<br/>　　脚尖一踢枪尾，长枪甩直，一招直枪向程大雷扎来。
<br/>　　只有赢家才有资格滔滔不绝，失败的人没资格开口。比如地上的陈梦。
<br/>　　到现在马孟起还不忘稍稍鄙视一下陈梦。
<br/>　　面对这一枪，程大雷退，一退再退，马孟起进，一进再进。
<br/>　　然后，程大雷握住了枪杆。
<br/>　　引他将力气使老，正是马孟起旧力已竭，新力未生之际，程大雷轻飘飘握住了枪杆，几乎没费什么力气。
<br/>　　程大雷要与他比力气。
<br/>　　马孟起眼睛一眯，单手攥住枪柄，右手狠狠一旋。一股震荡的力量向程大雷袭来，程大雷手臂一震，险些将铁枪脱手。
<br/>　　但今日程大雷已经发了狠心，今日拼着胳膊受伤，也要给马孟起一点颜色看看。
<br/>　　关键是如果自己放开铁枪，想要胜过马孟起便不容易了。
<br/>　　他咬紧牙关，一股力气从地上长起，腰马合一，程大雷单手攥住枪杆，口中猛喝了一声。
<br/>　　“开。”
<br/>　　直接将长枪夺了过来，程大雷将铁枪恶狠狠的扎在地上。与马孟起比力气，结果是对方没有他力气他，马孟起双手握枪，都没有敌过程大雷。此刻他双手已经磨出伤口，血淋淋一片。
<br/>　　如果真实武艺较量，双方都能拿出全部实力，程大雷想要胜过陈梦和马孟起中的任何一人都极其不容易。
<br/>　　但高手之争，就是在方寸之间。如果给陈梦和马孟起一些成长的时间，程大雷怕就没办法胜过任何一人。但现在他们没有太多战斗经验，程大雷欺负得就是他们少年未长成。
<br/>　　程大雷将剑收回鞘中，估计二人也没脸再和自己动手。假若他们仍旧不知好歹，那就勿怪程大雷会动杀心。
<br/>　　徐神机与刘发财在一旁观战，二人都是得意便猖狂的小人性格，此刻见程大雷轻而易举便击败二人，忍不住欢呼雀跃。
<br/>　　“天下之大，藏龙卧虎，少年郎，你们还是太年轻了，回去再练几年，再来与大当家挑战吧。”徐神机。
<br/>　　“我看再练几年也是不够的，怕是再练个十几年，或者是几十年。嘿，别老到骑不上马，提不动兵器，那你们就没有机会了。”刘发财。
<br/>　　“嘿，若谁学个三招两招，便想和大当家过手，那大当家这一辈子就不用做别的了。”徐神机呵呵笑着：“人呐，最重要的还是要知道自己几斤几两。”
<br/>　　少年心性，一是骄，一是傲，你可以给他一顿胖揍，但却听不得一句难听话。
<br/>　　陈梦含羞带愧的骑上马，本想着临走之前撂下几句狠话。可是技不如人，实在没脸说什么，骑上马踏上了回程的路。
<br/>　　马超同样跨上马，冲手下喝了一声：“我们走。”
<br/>　　临走之前恶狠狠看了程大雷一眼：“我的枪先留在你这里，等到了长安再向你取。”
<br/>　　程大雷摸了摸鼻子，无语的看着两人在视线中消失。
<br/>　　他将马孟起的兵器捡起来，搁在马车上，挥挥手：“我们走。”
<br/>　　几人继续上路，说也奇怪，接下来的路程极其顺利。程大雷一路过州过县，很明显有人将他认了出来，但再也没有一人出面阻拦。
<br/>　　程大雷却是不知道，马超带了一千人出京，目的是护送程大雷。只不过马超心性着急，舍开大部队，自己率先赶往汝南城。
<br/>　　结果是闹了个灰头土脸回来，与大部队汇合。沿途颇有不少人想向程大雷挑战，有些是为了扬名，有些是为了过往的仇恨。
<br/>　　赶上马超心情不好，将这些人一股脑全部收拾了。连我都打不过，你们有什么资格与他动手。
<br/>　　他如推土机一般，给程大雷开了一条路，捎带手的还收拾了路上一拨山贼。
<br/>　　如此状况下，程大雷接下来的路途再没有发生意外。他在八月十五之前，总算是赶到了长安城。
