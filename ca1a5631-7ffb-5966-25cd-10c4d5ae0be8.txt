　　山下，观战的法师们面面相觑。
<br/>　　崔晓明既然是“晓”字辈的弟子，自然也是星月奴的嫡传，培养了十几年，结果今日一战命丧在此，连魂魄也没留下……
<br/>　　“这个女子，是什么人啊？”
<br/>　　叶少阳听见身边充满了对于碧清的讨论。本来一个蓝衣人已经够厉害的了，居然还有一个这么厉害的帮手。
<br/>　　再看山上，蓝衣人也加入了战斗，与碧清一起对付剩下的五人。
<br/>　　“哈哈哈，自我了断，这是个好主意，你们几个死的时候，也可以学习，不过我不保证你们能有这个时间……”
<br/>　　碧清手持炼尸缸，对众人展开了狂攻，叶少阳在山下，看到她如此癫狂的表现，再一次想到了苏沫，这两个人，真的是太像了……
<br/>　　蓝衣人加上碧清，两人一起进攻，局面立刻变化。梧桐等人很快就只有自保的份，不断朝着山下倒退。
<br/>　　“叶少阳在什么地方？”碧清对上梧桐，立刻问道，不过因为是两人之间的对话，声音没有传到山下，叶少阳并没有听见。
<br/>　　梧桐一怔，怎么她到这个时候还在找叶少阳？
<br/>　　没等开口，碧清已经将炼尸缸抛出来，罩在她头顶上方，一股黑气从中溢出。
<br/>　　梧桐想到之前崔晓明的经历，不寒而栗，急忙作法，祭出了五宝金莲，熊熊金莲业火，抵抗着炼尸缸的黑气……
<br/>　　叶少阳站在山下，紧张到了极点，一只手隔着裤袋按着七星龙泉剑的剑柄，刚要冲上山去，突然之间，视线之中多了一个黑乎乎的东西，转头一看，是一面旗帜模样的东西，从山下一直飞了上来，仔细一看，顿时一惊，是之前鬼姨祭出来的那副星月奴的画像。
<br/>　　鬼姨也看见这一幕，当场跪在地上，大声说道：“祖师仙灵了！”
<br/>　　张晓寒也跟着单膝跪地。身后那些小门派的弟子紧随其后，陆续跪下。
<br/>　　画卷在空中展开，悬空半空，一动也不动，也没有落下来。组成人像的金粉和朱砂缓缓脱落，在空中凝聚成了一个人，跟画上的人一模一样，只不过只有形状，却是没有容貌，叶少阳用力愿望，感觉就像是从手机上看一个放得太大的图像，看到的只是星星点点的像素。
<br/>　　“这是星月奴？”叶少阳问身边的人。
<br/>　　“只是她的一缕神念。”云春生说道。
<br/>　　“身材可以啊。”叶少阳摸着下巴，望着天空中那个巨大的身影，称赞道。既然星月奴的神念来了，他对梧桐也算是放心了……尽管只是星月奴的一缕神念，但她必然有办法救她的弟子们，否则的话她绝不会显灵，不然当着这么多门人和依附法术公会的人的面，实在是威信扫地。
<br/>　　“你竟敢亵渎神祇！放肆！”鬼婆听见叶少阳自言自语的嘀咕，出声呵斥道。虽然这个时代，身材这个词不流行，但也能想到是什么意思。
<br/>　　“什么啊，我这是夸奖。”叶少阳嘿嘿一笑，他虽然不认识星月奴，但是对于她，却没有身边那些法师的那种膜拜的心理。
<br/>　　神祇，自己见过的还真不少。别的不说，单就黎山老母、摇光仙子这些阐教金仙，哪一个不是神祇？
<br/>　　“你们下去！”
<br/>　　空中那一抹星月奴的神念，发出威严的命令。
<br/>　　梧桐等人听见之后，不敢恋战，一个个都退了下去。
<br/>　　蓝衣人也命令桃花山弟子们不要追赶，不过其实这命令是多余的——桃花山弟子已经被杀得都没有几个人了，基本上只剩下山河道人这个光杆司令，活着的弟子，不超过三个。
<br/>　　蓝衣人迎着星月奴走了过去。
<br/>　　碧清也立刻跟上，冲星月奴说道：“星月奴，见到我师兄，还不下跪行礼！”
<br/>　　“荒唐！”星月奴哼了一声，人影突然散开，组成了一个“”字，朝蓝衣人兜头砸下来。
<br/>　　蓝衣人作法结印，双手高举，将“”字托住。
<br/>　　碧清召来莲花，花瓣散开之后，每一朵莲花都化成了一个字，却是九字真言：临兵斗者皆阵列前行，围着星月奴所化的“”字，环绕起来。
<br/>　　这一战，才是今天的重头戏。
<br/>　　山下，所有人都睁大了眼睛，打算好好看个热闹，然而，在峰顶的四面八方，地上突然冒出了一朵朵的金色莲花，升到半空中之后，却又化作金光，形成了一朵彤云般的所在，将三人团团围在中间，什么也看不见了。
<br/>　　“地涌金莲。这可是大乘佛法中最高深的一种……”云春生吃惊地喃喃自语。
<br/>　　叶少阳深表赞同，心中很是震撼，要说星月奴能施展出“地涌金莲”这一招，他丝毫不会感到吃惊，但是……现在出现的，只是星月奴的一缕神念啊，那么，她的本尊，究竟强到了什么样的地步？
<br/>　　彤云之中，悄无声息，但是大伙相信，那三个人一定是在彤云之中进行着最为猛烈的战斗，然而事实上……战斗早就进入了僵持的阶段。
<br/>　　“星月奴，你的手伸得太长了。”李浩然双手托举着字，淡淡说道。
<br/>　　“青牛祖师，我不知道你为何会出现在这，你本该证道混元、跳出三界，人间之事，又与你何干？”
<br/>　　“你既知我是谁，便该知晓，我从人间来，天下道佛二门，皆为我主之后人，我如何能看着你入侵人间？”
<br/>　　“你只有一个人。”
<br/>　　“两个人。”碧清说道。
<br/>　　“呵，芙窕仙子……你当我不知道你是谁？”
<br/>　　碧清道：“那又如何？”
<br/>　　“青牛，我称你一声祖师，而今这人间，已经不再是你的人间，但你既然想管这档子事，我也可告诉你，我所做一切，都为抵抗天劫。”
<br/>　　李浩然冷笑：“应劫？现在离天劫还远着。”
<br/>　　“你怎知道？”
<br/>　　“我往来于轮回之中，能知过去未来，末法时代还在百年之后，你这借口找得可不怎么样。”
<br/>　　星月奴没有去质疑他的话，说道：“即便如此，也当提前运筹，以免到时多方行动，人间一盘散沙，无力抵抗。”
<br/>(本章完)
