　　这是人间法师与阴司之间的一个传统，在黄泉大道上，鬼差要对进入阴司的生魂验明正身，如果是脖子上挂着千金坠的，说明是被法师检验过的，一般都是身上背负着什么错综复杂的干系，需要快速去天子殿报道。
<br/>　　叶少阳千金坠挂在赵四身上，并不是因为他有什地方特殊，只是自己希望他能够快速通关，自己好混进天子殿去。
<br/>　　叶少阳让赵四先站着别动，自己盘膝坐下，作法之后，鬼魂出窍，钻进千金坠中，念咒激活了千金坠的法力。
<br/>　　霎时间，在千金坠的灵力牵引之下，赵四的身影迅速坠落进了虚空裂缝之中……
<br/>　　这就是叶少阳想到的进入阴司的办法！
<br/>　　既然一切与阴司有关的法术都不能用，走阴也不行，叶少阳昨晚想了半个晚上，总算想到了这个略有点奇葩的法子！
<br/>　　既然自己没资格去阴司，那就找一个有资格的生魂，想办法隐藏在他身上，跟他一起去阴司。
<br/>　　叶少阳自己觉得，这个办法一定是可行的。所以他才会到处寻找将死之人。
<br/>　　只要到了阴司，他不怕自己见不到崔府君。
<br/>　　在虚空中穿梭了一段时间，赵四总算是到了鬼域。他伸手摸了摸自己脖子上的千金坠，凭着本能地朝阴司走过去。
<br/>　　过了鬼门关，进了黄泉大道上，人影重重，就像是赶集一样，所有人都在朝着一个方向走。这些都是人间刚死的生魂，前来人间报道。
<br/>　　有一部分是自己来到，也有一部分是被鬼差从人间拘来的，哭哭啼啼的，一路被驱赶着往前走。赵四就夹在人群中，郁郁前行，一路走到酆都城下，城门下方，两队鬼差分列左右，身后还站着一排银甲鬼武士，一个个目光警觉地从过路的生魂身上扫过，生怕有什么妖邪混进酆都城内。
<br/>　　轮到赵四的时候，一个鬼差在他脖子上发现了千金坠，扫了他一眼，也没有再多作检查，直接放行。
<br/>　　等到进城的鬼魂足够多了，银甲鬼武士们立刻喝令后面的生魂站住，由一队鬼差，押送着进城的这一批前往天子殿销帐。
<br/>　　“大法师，进了酆都城了。”赵四嘀咕了一声。叶少阳躲在千金坠里，自然不敢开口，他的目的尽量能混到哪是哪，能直接见到崔府君更好。
<br/>　　“你说什么，跟谁说话？”鬼差听见赵四暗自嘀咕，警觉地问了一句。
<br/>　　“没，没有谁。”赵四也不敢乱说。
<br/>　　好在那鬼差也没有再问下去，只是瞪了他一眼，继续驱赶他们，一路来到天子殿。
<br/>　　到了天子殿，比之前的检查就严格多了，需要排队，然后报名字，经过两个功曹的检验才可以进去。
<br/>　　轮到赵四的时候，两个功曹看到他脖子上的千金坠，也没有多问，吩咐道：“进门之后往右走，到立魂厅则可。”
<br/>　　赵四刚要迈步进去，一个功曹突然意识到什么，拦住他说道：“等会，你身上怎么会有生魂的味道。”
<br/>　　叶少阳在千金坠中，听见这句话，叫苦不迭，自己而今法力有限，虽然藏身在千金坠中，仍然没有办法完全隐匿身上的气息，之前在那些普通鬼差面前还好蒙混过关，在天子殿功曹这里就不好使了。
<br/>　　两个功曹上前，仔细在他身上嗅了嗅，对视一眼，一个道：“没错，他身上还有另一个生魂！”
<br/>　　一个功曹立刻出手，一把抓住赵四的头发，提到自己面前，喝问他怎么回事。
<br/>　　赵四吓的浑身哆嗦，无法开口。那功曹上下检查起来，找不到他身上另一个生魂藏在什么地方，心下正在疑惑，另一个功曹猛然想到什么，一伸手抓住了他脖子上的千金坠，扯了下来，紧紧握在手里，感知了一下，嘿嘿一笑，口中念咒，手中放出蓝色的灵光，不断挤压着千金坠上的五帝钱。
<br/>　　叶少阳立刻感觉到了一股窒息般的痛苦，没奈何，只好抽身离开了千金坠，落在地上，刚要起身，两个功曹已经抽身上前，一人一边，捏住了他两边肩膀。
<br/>　　两股罡气，一瞬间窜进了体内，叶少阳顿时感到痛苦不堪，运气抵抗。但他现在只有一成法力，根本不是两个天子殿功曹的对手，体内罡气很快就被冲散了，疼的龇牙咧嘴。
<br/>　　这两个功曹见他法力低微，自己也放松下来，一个松开他，冷笑道：“我还以来有什么厉害人物敢擅闯天子殿，原来就这两下子，你胆子可真够大的，说吧，你是从哪里来的，莫不是太阴山派来的探子？”
<br/>　　“差爷息怒，息怒！”自己被人扣着，叶少阳也不敢来硬的，只好先说好话，拱手行礼，说道：“差爷误会了，我是人间来的，我是茅山第……茅山天师叶少阳。”
<br/>　　他本来想说自己是第三十九代传人，猛然想到了在人间被那个龙虎山弟子视为骗子的经历，天子殿功曹虽然不怎么去人间，但是对于茅山这样的大派的情况不可能一无所知，自己要是那么说，立刻又会被当成骗子。
<br/>　　“天师，哈哈哈，你是天师？”那功曹笑起来，“就你这实力，我看连方士都达不到，居然敢自称天师，还说是茅山弟子，小子，你以为茅山天师这么不值钱？”
<br/>　　叶少阳看了他一眼，心中很是不算，心想天师算什么，我几天前还是灵仙呢。不过这话现在说出来也没人信。叶少阳只好又拱手行礼说道：“我真的是茅山天师，我有天师牌在身上。”
<br/>　　伸手往衣服里摸出天师牌，递了过去。那功曹狐疑地接过去，上下检查了一下，以法力检验了一下，天师牌在他手中立刻幻化出了一股蓝色的柔光，功曹定睛看去，天师牌晶莹剔透，上面的枫叶形状呈现出金色，看上去十分显眼。
<br/>　　“是真的……”功曹当场就懵了。天师牌是天师的身份象征，这个是绝对做不了假的，但是以叶少阳的法力，绝不可能是天师。
<br/>　　那功曹怔了半晌，说了一句让叶少阳喷血的话：“这天师牌你是偷的还是夺来的？”
