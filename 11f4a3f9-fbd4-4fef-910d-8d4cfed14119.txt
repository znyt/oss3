        大哥已经气愤到了极点……
<br/>
<br/>        不！
<br/>
<br/>        他是对她失望到极点了吧！
<br/>
<br/>        童筱颖怀抱双臂，匀称而修长的双腿紧紧地并拢，赤脚呆愣在房门外。
<br/>
<br/>        她聋拉着脑袋，乌黑的长发及腰，衬得她脸色格外惨白。
<br/>
<br/>        “大哥，你开门，好不好？可不可以相信我？大哥……大哥……你开门呐！”她舔了舔有些干涩的唇，抬起手来握住门上的手柄，试图推门进去，但是房门已经被傅夜沉从里面反锁了，她压根就开不了。
<br/>
<br/>        此时此刻，童筱颖只觉自己的身体开始渐渐发冷，明明这还未入秋，她便觉得四肢连同心脏一起都感受到了一种刺骨的寒意。
<br/>
<br/>        她要怎么做，才能让大哥相信她？
<br/>
<br/>        心里迷茫到无助……
<br/>
<br/>        就在她欲要再次敲门的时候，门突然间打开了，她一拳捶在了一堵肉墙上。
<br/>
<br/>        童筱颖欣喜地抬起头来，连忙拉起傅夜沉的手，当她想要解释的时候，傅夜沉却看都不看她一眼，甩开她的手，只身离开房间往楼梯口那边走。
<br/>
<br/>        “大哥，你听我说！”童筱颖疾步追了上去，再次挽住傅夜沉的手。
<br/>
<br/>        而下一秒，他剑眉微蹙，仍旧不愿意多看她一眼，用力地甩手推开了她，并加快了脚下的步伐径直下了楼，很快就没了人影。
<br/>
<br/>        童筱颖被这道力度推得未站稳脚，整个人往旁边倒去，岂料左耳直接撞到了长廊上放瓷器的置物架的尖角，置物架上的瓷器也随之跌落，砸到了她的额头上。
<br/>
<br/>        顿时，耳朵里产生了一阵耳鸣，这尖锐的响声，让其它的声音全部消失了，在她耳朵里响得令她头晕目眩，甚至忘了额头上传来的阵痛感。
<br/>
<br/>        过了许久，她才从地上爬起来，捡起地毯上的瓷器，将它重新放回了置物架上。
<br/>
<br/>        童筱颖有些狼狈地回到自己房间里，拿出药箱给自己处理伤口。
<br/>
<br/>        额头上只是鼓了个包，但左耳里却在流血，她拿着棉花团替自己擦了许久才将血止住，然而，里面却依旧痛得让她很难受。
<br/>
<br/>        一整晚，耳朵里都在鸣叫，刺激得她在床上翻来覆去，难受得快要虚脱。
<br/>
<br/>        童筱颖不知道自己最后是怎么睡着的，只是在睡醒后，刚从床上坐起来，刘婶便叩门进了房，碎碎念叨：“大少奶奶，您已经起床了啊！那我敲门，您怎么没听见呢？”
<br/>
<br/>        刘婶碎碎念着，一边递给她一叠A4纸的文件袋，一边唉声叹气地接着说道：“你和大少爷不是好好的吗？怎么突然就闹离婚了呢？”
<br/>
<br/>        “刘婶，你说大点，我听大不清你在说什么。”童筱颖刚想去揉左耳，却发现自己一碰，就会痛。
<br/>
<br/>        刘婶叹了口气，直接将手里的文件袋往童筱颖手中一塞，摇头道：“这是大少爷让律师送过来的，您自己慢慢看吧！”
<br/>
<br/>        童筱颖这才拿起文件袋，将里面的资料拿出来看了看，上面白纸黑字写了很多内容，醒目的是“离婚协议书”这五个字，末尾有傅夜沉的亲笔签名。
<br/>
<br/>        看到这里，她整个懵了。
<br/>
<br/>        甚至，忘记了心痛是什么滋味，忘记了自己其实还有眼泪，忘记了所有本该有的不该有的任何念想。
<br/>
<br/>        刘婶唉声叹气地转身离去。
<br/>
<br/>        童筱颖手里拿着离婚协议书，在床边呆如木鸡地愣了许久，脸色惨白得比这离婚协议书的纸还要白。
<br/>
<br/>        她忽然间从床边起身，就像一具行尸走肉一般走到了一旁的梳妆台前，然后从抽屉里拿出一支笔，在末尾女方的那一栏，一笔一划，慢吞吞地签上了自己的名字。
<br/>
<br/>        一式三份的离婚协议书签完后，她用力扯着自己的中指，直到中指的指骨脱臼，她才将自己中指上的那枚求婚戒指给摘了下来。
<br/>
<br/>        文件袋里有一张银行卡，她没要，甚至把戒指也一并放进了文件袋里。
<br/>
<br/>        随后，她开始整理自己的行李箱。
<br/>
<br/>        只要是他买给她的东西，她一样也没要，离开私宅前，她强颜欢笑着将文件袋转交给了刘婶，和刘婶拥抱了一下，便独自离开了。
<br/>
<br/>        她没有叫出租车，而是独自一个人步行离开了私宅。
<br/>
<br/>        童筱颖浑浑噩噩地走了许久，感觉自己仿佛一具行尸走肉一般，心里是空空的，眼睛也蒙了，终于，她看到了有车辆经过的大马路。
<br/>
<br/>        当她正要往对面去的时候，一辆私家车从她的左边驶来，在马上要撞到她的一瞬间，稳稳地刹住了。
<br/>
<br/>        司机落下车窗，忍不住对她破口大骂：“你他妈的是聋子啊！想死去跳楼！别来这祸害我！”
