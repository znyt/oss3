　　叶少阳朝靠墙那排尸体走过去，来到第一具尸体面前，从背包里拉出墨斗，上面挂上一枚铜钱，栓在了尸体的脖子上。
<br/>　　青冥道长等人看见他这么做，有些不解，但也没敢多问。
<br/>　　把红线往尸体脖子上套上之后，叶少阳开始画符，先用紫色灵符画了一张符，包裹住一刻铸母大钱，接着又画了六张黄符，分别贴在几个尸体的身上，再给每一具尸体脖子上都套上栓着五帝钱的红绳，唯独那具特殊的女尸，红绳多绕了几道，然后画了三张定尸符，分别贴在她额头和两边肩膀上，封住三大鬼穴，免得尸体被外来的邪物顶上，钻进她尸体。
<br/>　　做完这一切之后，叶少阳转头对陈三说道：“我这就起尸了，你在前面摇铃开道吧。”
<br/>　　陈三怔了一下，懵懂地点点头，意思我看你这样怎么起尸。
<br/>　　青冥道长和张道长互相看去，也是一脸的困惑。叶少阳的作法，与赶尸匠用的手法完全不同，就算是道门法术，他们也没见过什么法术能同时将六具尸体一起驱赶起来。
<br/>　　叶少阳拿出自己的阴阳镜，用朱砂笔在什么画了个符文。然后找青冥道长要来一碗水，口中默念咒语：“天清地明，七子连心，御极万物，达者通明！闻我咒者，上达紫薇大帝，传我法力，急急如律令！”
<br/>　　一抖灵符，凌空燃烧起来。叶少阳将灵符丢在水中，灵符不灭，漂在水面燃烧，不多时一会，竟然将一碗凉水烧的沸腾起来，不断蒸发。
<br/>　　张道长等人还是第一次见到这种离奇的法术，忍不住凑到跟前去看，这才发现凉水沸腾，不是因为灵符燃烧，而是中间的铸母大钱通体赤红，在碗底不断旋转，将水蒸发。
<br/>　　等到一碗符水完全被蒸发，铸母大钱也被符水染成了黑色。叶少阳把铸母大钱小心地拿起来，对着钱眼轻轻吹了一下，大喝一声：“起！”
<br/>　　一道金光，从铸母大钱上闪过，分作六束，分别落在了贴在六具尸体脸上的灵符中，金光一闪而逝。接着神奇的事情发生了：七具原本直挺挺靠在墙上的尸体，突然一起站了起来，然后转身，两只手臂刷地一下弹了起来，伸得笔直，朝着房门的方向跳了过去。
<br/>　　“啊！”翠云吓的捂着嘴叫了起来。
<br/>　　这一叫，让青冥道长回过神来，眼看最前面的尸体快要出门，急忙大喊一声：“撤槛！”
<br/>　　站在门外的两个道士也醒悟过来，急忙一人一头，抓着门槛的两边，将门槛卸掉。
<br/>　　义庄的门槛都是非常高的，这样万一尸变，尸体要跳出去的时候，有可能会绊倒在门槛上，一旦有尸体要驱赶的时候，就需要将门槛卸掉，尸体才能跳出去。
<br/>　　门槛卸掉之后，七具尸体陆续都跳了出去，陈三这才反应过来自己的工作已经开始了，追上前去，在前面带头，摇起了铜铃。
<br/>　　叶少阳对青冥道长抱拳，笑了笑，然后拉起一脸懵逼的翠云，朝门外走去。
<br/>　　青冥道长和张道长立刻追出来，送他们下山，路上两人十分诧异地问起叶少阳赶尸的手段，叶少阳跟他们解释，自己是用驱尸符，之前烧掉的那张是母符，贴在七具尸体脸上的是子符，叶少阳烧符念咒，只需要一张母符，就可以通过七张子符，对七具尸体实现控制。
<br/>　　这是茅山内门法术的一种，叶少阳施展起来并不费事，只是他现在法力有限，怕自己施展起来太吃力，所以才用紫符包裹铸母大钱，化出符水，在燃烧和蒸发水汽的过程中，将灵符的灵力吸收进铸母大钱中，这样就能借助铸母大钱的灵力，达到施法的效果。
<br/>　　这是一种法器和符咒合一的手段，光是听上去就足够复杂，张道长和青冥道长法力有限，在对法术的理解方面，跟叶少阳相比更像是个儿童，因此也只有佩服的份了。
<br/>　　现在，驱尸符已经发挥了效用，叶少阳只要念咒，对尸体下达他们力所能及的命令，他们都会执行，铸母大钱的灵力来驱使几具普通的尸体，完全没有问题。就算是那只人形煞，因为被叶少阳用红线五帝钱和定三尸符牢牢封印住肉身，在不破除封印的情况下，跟普通尸体也没什么两样。
<br/>　　这是叶少阳想了半天之后，从自己那些法术中找到了相对容易的驱赶尸体的方式，只要铸母大钱在手，他现在可以命令这些尸体做任何事，就算是攻击人都可以，当然了，越是复杂的动作需要消耗的法力越多，就算是借助铸母大钱的灵力，以他现在的实力来说，想要操控几具尸体做太复杂的事也是做不到的，也就是最简单的赶路了。
<br/>　　陈三在前面摇铃开道，铃是特殊制成，也算是法器，声音对尸体能产生一定程度的刺激，让他们能够跟随铃声的节奏前进。
<br/>　　叶少阳虽然用的不是一般赶尸匠的手段，但是赶的毕竟是尸体，在这方面是相通的。
<br/>　　在陈三的带领下，七具尸体一蹦一跳地顺着小路下山，叶少阳和翠云跟在后面，翠云起初很是害怕，一直缩在叶少阳身后，后来逐渐也麻木了，见这些尸体只是赶路，别的也不会怎么样，于是胆子也慢慢大起来，跟在叶少阳身边问东问西。
<br/>　　叶少阳抬头看着长野星空，天空中的那一轮明月，不禁暗暗叹了口气，摇头苦笑。自己那些身在一百年后的小伙伴们，一定想不到自己竟然在一百年前的山野中，在自己都不知道的地方，从事着赶尸这种古老的职业。
<br/>　　用了半个晚上的时间，三人七尸，下到了山坳地步，陈三告诉叶少阳，山坳里有个小镇，镇子边上有他们义庄建的木屋，可以休息一天，算是赶尸的第一站落脚点。同时镇上有些做小买卖的，可以补充一下在路上的吃用消耗。
<br/>　　在陈三带领下，三人赶着尸体来到了镇上，这时候将近黎明，镇上一点灯火都没有，更没人在外头，就算有，听见赶尸的铃声也会躲得远远的。
