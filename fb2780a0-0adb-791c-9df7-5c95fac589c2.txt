　　第130章
<br/>　　初九，牛日冲羊，凶神在北，忌：远行，会客，安床；宜：诸事不宜。
<br/>　　雪地中，缓缓行来两人，一老一少，老者约有五十出头，外貌更老，少者十三四岁年纪。两人都穿道袍，后绣八卦，云纹挂边。少者手里托一根白幡，上纹四个黑字：看破人间。
<br/>　　老者掐指寻根，缓缓吐出一串话，半晌嘴里嘟囔了一句：“诸事不宜呐。”
<br/>　　“师父，那咱们还要去风雨亭么？”少年唤了一声。
<br/>　　“运的事，天的事，命的事，人的事。天道不仁，脚下的路还是要自己走的。况且……”
<br/>　　“况且，咱们已两天没吃东西了。”少年快走两步，跟在师父身边。
<br/>　　老者抚须，微笑：“然。”
<br/>　　“师父，幽州王招安绿林道，为何偏偏将地点选在风雨亭？”少年问。
<br/>　　“这次招安，不知惊动多少草莽，大家也要提防幽州王有诈借机把大家一网打尽。”
<br/>　　“所以选在风雨亭？”
<br/>　　“风雨亭在幽州城外五十里，依河而建，临一高台，一台一亭都是为祭天求雨建的，取风调雨顺，五谷丰登之意。当然，也没什么用处啦，不过风雨亭四周围都是开阔的田野，周围有没有伏兵一览无遗。选在风雨亭，也是杨龙停表达诚意的方式。”
<br/>　　“那杨龙停会有诈么？”
<br/>　　“有诈估计不会有诈，好意估计也无多少，我们就去看看……”
<br/>　　“混两天饱饭？”
<br/>　　“然。”老者笑笑，忽然板起面孔：“了儿，行走江湖什么最重要？”
<br/>　　“走得快。”
<br/>　　“三十六计那一条不能忘？”
<br/>　　“走为上。”
<br/>　　“聪明。”老者迈开脚步：“走着！”
<br/>　　白雪皑皑，两道人影愈行愈远，雪地之上只留下两道脚印，渐有风吹来，脚印很快被雪埋住。
<br/>　　……
<br/>　　今天本没有太阳，偏偏到得正午，乌云散去，太阳在天空中探出个脑袋。这样的天气，太阳照在人身上也谈不上暖和。
<br/>　　风雨亭外摆了两排太师椅，左右排开，约有百十张。这自然是给幽州绿林道上有名有姓的人准备的，亭中摆着三张椅子，自然是给绿林道最叫得响的人物。
<br/>　　外有官兵持戈把守，约有百十位，都是八尺身材，虎背熊腰。不能说杨龙停帐下都是这样的精兵，但今天也是亮肌肉的时候，杨龙停潜台词是告诉大家：我也是很凶的。
<br/>　　到得中午，幽州城的头面人物没出现，江湖上叫得响的也没有现身，毕竟，重要角色总是要压轴出场。
<br/>　　不过，此刻风雨亭下来得人已经不少，众贼从四面八方赶来，绿林豪杰相聚，自然没那么温文尔雅，风雨亭下已经是乱哄哄一片。
<br/>　　“张四爷，辛苦辛苦，最近发什么财？”
<br/>　　“发什么财，也就一个月前抢了一笔，饭都快吃不上了。”
<br/>　　“是呐，今年年景不好，日子不好过呐。”
<br/>　　“坐，坐，四爷这一道来可是辛苦。”
<br/>　　二人在椅子上坐了，叙着最近的买卖。张四聊得开心，忽觉得周围的空气冷下来。他抬起头往前瞧，发现面前杵着一张脸。尖嘴嘬腮，头上没有几顶毛，像是从坟里爬出的食尸鬼。
<br/>　　“你也配坐这里？”
<br/>　　张四心底一寒，立刻认出这人是谁。狗兄鼠弟中的老大周狗，幽州有名的盗墓贼，外人相传二兄弟睡在坟墓中，妻鬼母尸。关于这二人有很多谣言，稍微可信的一条是：
<br/>　　这二兄弟是一对孪生子，他们的母亲因故早亡，有一盗墓贼去扒新尸上的衣服，听到棺材里传出哭声，撬开棺材盖后发现棺材里刚出生的二个娃娃。二人长大后自然也成了盗墓贼，又穷又丑找不到媳妇，二兄弟商量：反正咱是鬼生的，干脆就找个鬼做老婆吧。于是有哪家姑娘早亡，往往尸体没埋进土里三天，坟头就会被扒开。
<br/>　　于是有了狗兄鼠弟，妻鬼母尸的说法，这还是稍可信的说法，其他就更加匪夷所思听上去像鬼故事一样，至于真相究竟是不是如此，却也没人去和周狗周鼠两兄弟确认。
<br/>　　“滚！”周鼠凑过来，嘴里吐出一个字，吐沫星子直接喷在张四脸上。
<br/>　　张四突然意识到，风雨亭前虽然摆着很多椅子，但座次顺序好像有某种讲究。
<br/>　　风雨亭下摆着三张椅子，目前都空着，外面的椅子，越是靠近亭子，在道上的名头就越响，相应实力也就越强，心肠也就越黑越狠……
<br/>　　比名头，比实力，比心肠……自己好像都不如周狗周鼠这两兄弟。
<br/>　　张四乖乖让了，众人在众人面前被周狗周鼠逼到这境地有些丢人，但委实还是不要和他们有过多恩怨才好。
<br/>　　“两只坟耗子，今天也耍起威风来了！”
<br/>　　在风雨亭上，忽然响起一声尖笑，众人一愣，大家都在此地，竟没人发觉他是什么时候爬上去的，或者说，他早已在上面。
<br/>　　今日幽州绿林道的匪类几乎到齐，所谓狗有狗路，鼠有鼠道，各人都有各人特别的本事，但能有这手漂亮轻功的却是只有一个。
<br/>　　于是，这人都名字也就出来了——赛燕子单飞。
<br/>　　“既然你们都在争，那我也捡张椅子坐。”
<br/>　　单飞飘然落地，自顾自的走进风雨亭内，准备在主座右手边的椅子上坐下。
<br/>　　风雨亭内三张太师椅，中间是主人位，以左为尊，以右为卑，单飞敢坐这张太师椅，就代表敢应这绿林道第二的位置。
<br/>　　嗖！
<br/>　　一支鱼叉凭空而来，单飞速退一步，鱼叉险之又险的落在单飞脚边。
<br/>　　抬起头，只见五个人从远处过来。
<br/>　　“姓单的，滚开！”
<br/>　　野人渡的马家父子。
<br/>　　单飞从腰上拔出一柄匕首，在手指上旋转。
<br/>　　“马老五，你想争这个位置？”
<br/>　　马五膝下四子，有胖有瘦，有高有矮，一样的是眼中同样凶恶的目光。如饿极的狗。
<br/>　　“绿林道第一的位置自然是熊老大坐，可第二还轮不到你。”
<br/>　　“你们爷们不服，可以刀上见个真章，在场谁要是不服，都可以上来试试。”单飞指尖的匕首，旋转飞快。
<br/>　　“哼，你要是不跑，谁都可以打你！”
<br/>　　
<br/>(本章完)
