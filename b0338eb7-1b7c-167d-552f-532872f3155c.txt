　　第532章
<br/>　　作为西蜀巫医一脉的唯一传人，孟玄青总有孟玄青的体面和尊严。
<br/>　　曾经年少时，孟玄青也是个风流人物，家中宾客盈门，身边美妾如云，多少达官贵人，世家公子以结交孟玄青为荣。
<br/>　　当然，因为后来发生一些事，孟玄青满门被害，他躲到大雪峰上。多少年过去，孟玄青一直不去想曾经的事，潜心研究毒术，借此麻痹自己。
<br/>　　但这次下山，在人世间走了一遭，那颗尘封的心又活泛起来。
<br/>　　他当然可以就地取材，采荒野中的药草配出毒药，但这一来无法显出西蜀毒医的手段，二来，程大雷毕竟不是泛泛之辈，冒昧行动未必可以成功，反而有可能打草惊蛇。
<br/>　　孟玄青想了又想，最后还是决定取消这次行动。他在大树上直起身，捂着胸口咳嗽一阵，拄着木杖一步步往东去了。
<br/>　　在他头顶，几只黑鸦盘旋一阵，发出呱呱的叫声。
<br/>　　黄家大宅中，欢宴仍然在继续。
<br/>　　当程大雷放开之后，便无人能在他面前夺得话语权。酒桌上的谈话节奏，已完全被他掌握，至于刘发财，发现自己几次插口都无法打断程大雷的话，于是只有闷头喝酒的份儿。
<br/>　　黄老汉一双三角眼烁烁放光，他几次开口试探程大雷，却被程大雷轻松将话题转到一边。
<br/>　　程大雷完全水泼不入，到现在也没能对程大雷有个大概估计，但自己的底细已快被程大雷盘问清楚。
<br/>　　“咦，怎么不见黄少爷？”程大雷突然开口问，像是现在才发现这件事。
<br/>　　黄老汉早就准备好说词，呵呵笑着道：“他老母亲病重，今天早上去请大夫了。”
<br/>　　程大雷闷了一口酒：“不是说附近没有名医么？”
<br/>　　“呃……”黄老汉一时语结，尴尬的笑笑道：“只是随便找个大夫，不是壮士口中说的名医。”
<br/>　　“一去一回，到现在还没回来，算路程……他去了兖州城？”
<br/>　　“哪有，哪有，来，壮士喝酒。”黄老汉借着倒酒的机会，擦掉额头的冷汗。这人外表粗犷，内里却心细如线。自己一句话被谨慎，便被他看出破绽。
<br/>　　程大雷心思电转，只看出黄老汉有些古怪，但确实也猜不透对方有什么打算。当然，他心里确实也没将黄老汉看在眼里，一个土财主又能做什么。
<br/>　　咚咚咚！
<br/>　　一名家丁急匆匆慌张张跑到大厅，焦急道：“老爷，大事……大事不好，山贼杀来了！”
<br/>　　“山贼，什么山贼！”
<br/>　　黄老汉一激灵，事情怎么和自己想得不一样。他下意识看向刘发财和程大雷，难道山贼是这二人引来的。
<br/>　　并不是没有这种可能啊，毕竟那位可是天下第一山贼。
<br/>　　程大雷和刘发财对视一眼，也被眼前突入起来的情况搞得有些懵。
<br/>　　正这时候，又有下人跑进来，叫苦连天道：“老爷，山贼闯进镇子了，见人就杀，一路奔着咱们宅子来了。”
<br/>　　黄老汉坐在太师椅上，一张脸煞白无比。陪坐的客人也都如热锅上的蚂蚁，有人站起来，不知该逃该留，逃，又能逃到哪里去呢。
<br/>　　“黄老伯，莫非平日得罪了附近的山大王？”
<br/>　　“没有啊。”黄老汉苦着一张脸道：“老汉向来不敢和他们打交道，每年要米要钱，也不敢有半分亏待，哪里来的胆子得罪他们。”
<br/>　　程大雷问那下人道：“山贼来了多少人？”
<br/>　　“看不清楚啊，黑压压一片，足有上千人吧。”下人明显已被吓破了胆，说话也语无伦次，模糊不清。
<br/>　　程大雷倒也能理解，一个普通下人，能不被吓死过去已经不错。
<br/>　　黄老汉此刻突然来了精神，冲程大雷和刘发财道：“二位好汉，黄家上下八十几条性命可都拜托在二位身了，二位一定要救我黄家一救。”
<br/>　　在确认程大雷二人与这伙山贼没有瓜葛时，黄老汉还是把所有希望寄托在二人身上。
<br/>　　刘发财眼珠打转，心中默默想着脱身之策。黄家八十余口和他当然没多少关系，还是自己的性命要紧，当然，如果能将那水灵灵的丫鬟带走，就更好了。
<br/>　　“大家都不要急，山贼未必是冲着黄家来的。”刘发财。
<br/>　　“老爷，老爷……”第三个下人跑进来：“山贼已经把院子围了，门房老李被他们杀了，山贼说要把黄家全部杀光，一个也不放走。”
<br/>　　“这……”黄老汉彻底崩溃。
<br/>　　刘发财无语的坐回原位，心中呜呼哀哉一声：这下自己要走也难了。
<br/>　　“莫要慌。”程大雷道：“将前后门关好，先让女人孩子躲起来，他们只围住院子没有闯进来，看来目的并不是杀人。让人出去传话，看他们想要什么？”
<br/>　　“对对对。”一席话说开了黄老汉的心窍：“快按这位壮士说的办，无论他们要多少钱粮都给他们，只求他们不要杀人。”
<br/>　　夜里的黄土镇被吵醒了，一队人举着火把涌进安静的镇子，最后在黄宅大门口汇聚。所有人散开，将黄家里三层外三层的围住。
<br/>　　长须鬼孙山起，饿死鬼朱刚，白面鬼张猛骑马挡在大门前，面前倒着门房的尸体。鲜红的血流了一片，腥气扑鼻。
<br/>　　之所以没有着急攻入院子，当然不是因为他们不想杀人，大概是出于某种对程大雷的尊重，或者是出于对自己的尊重。
<br/>　　“程大雷听真！某乃鹰嘴岭张猛，特意来取你的人头。有种的，赶紧与我出来！”
<br/>　　这声音从大门口传入院子，穿过回廊花园，一直传到大厅，响在大家耳边。
<br/>　　程大雷和刘发财同时看着彼此，程大雷的眼睛眨了眨。
<br/>　　黄老汉也是一激灵，第一眼下意识看向程大雷，随之才意识到自己认错了人，所以又看向刘发财。
<br/>　　“程当家，这帮山寨是冲着您来的，以您的本事自然不惧他们，肯定挥挥手就吓得这帮山贼魂飞魄散。”
<br/>　　“哈哈哈。”刘发财仰天大笑：“本当家走南闯北，又怕过谁来。黄老汉……”
<br/>　　“在！”
<br/>　　“黄家有没有向外的密道？”
<br/>(本章完)
