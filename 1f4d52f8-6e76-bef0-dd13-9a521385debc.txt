<br/>　　脸上带着笑眯眯的表情，刘哔从商城里兑换了十颗筑基丹取出，拿在手里。

<br/>　　最多也就是冒些风险。

<br/>　　这些半人类，如果成长起来，几乎都是SSS级别的能量体，每一个人，都能给他提供整整一万点积分。
<br/>　　而和他原本的世界相比，两个位面之间的时间流逝速度并不相同。
<br/>　　也就是说，这边花费十年的时间才可以培养出一批SSS级别的搜查官，而对他而言的话，只需要两三年就足够了。
<br/>　　或者更短。
<br/>　　况且，等到白日庭这边形成一定规模了，培养逐渐就会循环起来，那样效率会比现在高了不知道多少。
<br/>　　对他而言，这里简直就是一个天然的养猪场啊。
<br/>　　可惜这种方式，相对来说很难复制，毕竟像半人类这样的能量体十分少见，即使有，数量也肯定不会太多。
<br/>　　说起来，能想到这样的点子，还是拜和修一族所赐。
<br/>　　初代喰种之王，之所以组建CCG，何尝也不是有圈养人类的想法。

<br/>　　他也没指望他们会协助自己战斗。
<br/>　　很快，他就会变成这些人杀之而后快的存在了。

<br/>　　最终，他还是叹了口气，起身将刘哔送到门外。

<br/>　　看来，接下来的日子里，他要更加小心的面对这件事情了。

<br/>　　将钥匙扔到门口的鞋柜上，刘哔脱下外套，坐在什造身边，语气有些关切。

<br/>　　这么一个优秀的种子，如果在这样的事情上出现什么意外，就着实太过于可惜了。
