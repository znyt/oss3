　　比武台上。
<br/>　　萧逸看着气宗三长老，语气有些不耐烦。
<br/>　　“怎么？不过是些许反震，气宗三长老稳下气息要这么久？”
<br/>　　“还剩一剑，气宗三长老不打，我可就走了。”
<br/>　　那般轻蔑，再次让得气宗三长老脸色涨红，证明着他已愤怒到了极点。
<br/>　　而且，萧逸的话，同时亦让得气宗席位上的一众气宗强者面露怒色。
<br/>　　萧逸言语中，没有任何一句冲撞、谩骂，或者挑衅。
<br/>　　但他的语气和态度，却处处透漏着对气宗的不屑与轻蔑。
<br/>　　“臭小子，你找死。”
<br/>　　“剑域无数年来，还从未有外来武者敢在剑域放肆，更别说是一个猖獗的毛头小子。”
<br/>　　“宗主，还请下令立刻拿下这小贼，否则我剑域剑者声名，必定受损。”
<br/>　　气宗席位上，一道道愤怒的目光，齐齐剐向萧逸。
<br/>　　萧逸怡然不惧，只是冷笑一声。
<br/>　　如今他剑力充沛，战个一天一夜都够了。
<br/>　　气宗六剑侍他已不必放入眼中，更别说这气宗席位上的一个个气宗长老和气宗武者，
<br/>　　若气宗非要发难，他也不介意杀个痛快。
<br/>　　“宗主。”气宗席位上，一个个长老，已然起身，齐齐看向气宗宗主。
<br/>　　很显然，只要气宗宗主一声令下，这些个长老，就会毫不犹豫地对萧逸出手。
<br/>　　气宗宗主，也就是那中年人，皱了皱眉。
<br/>　　恰在此时，云渊长老如云般轻淡的声音，缓缓响起。
<br/>　　“从无外来武者敢在剑域放肆？不见得吧。”
<br/>　　轻淡的声音，并不多响亮，却传遍全场，清晰地传入到所有剑修耳中。
<br/>　　在这天下剑修云集的盛事之上，当那一言传遍全场，可没几人。
<br/>　　周遭剑修，这才想起来，比武台上的这个‘嚣张’的年轻人，背后可是站着两位巅峰剑修。
<br/>　　而同一时间，气宗剑者也霎时想了起来。
<br/>　　确实，曾经有过别的外来武者，敢在剑域之内放肆，甚至放肆得可怕。
<br/>　　而这个人，此刻正站在云渊剑的身旁。
<br/>　　不错，正是讳无视。
<br/>　　多年前的一次剑域盛事中，三大巅峰剑修之战，震惊中域。
<br/>　　可也便是那一战中，讳无视这个当时还是天冥府第一杀手的强者，忽然而出，无视剑域各大剑者，一剑将其中一个巅峰剑修封溅血封喉。
<br/>　　这也是讳无视中域杀神之名崛起的一战，亦是讳无视成为新的三大巅峰剑修之一的一战。
<br/>　　念及此，气宗一众剑者的脸色，陡然难看到极点。
<br/>　　唯气宗宗主，脸色仍旧淡然。
<br/>　　“这毕竟是两宗大比。”气宗宗主淡淡道，“如何处置，也不是我气宗一脉说了算。”
<br/>　　“是吧，器宗诸位。”气宗宗主，轻笑着看向器宗席位。
<br/>　　不愧是能当上宗主的人物，果真非泛泛之辈。
<br/>　　轻飘飘的一句话，便将这个如何处置的‘难题’，交到了器宗手上。
<br/>　　“自然是再战。”丁家族长，猛地喝了一声。
<br/>　　“秋月，不得给我丁家庄丢脸，更不得给器宗一脉丢脸。”
<br/>　　“是，族长爷爷。”丁秋月呐呐地回答了一句。
<br/>　　气宗宗主闻言，脸上泛过一丝冷笑。
<br/>　　萧逸，霎时皱起了眉头。
<br/>　　“你没胜算，何必战。”萧逸皱眉看着丁秋月。
<br/>　　不错，这本就是一场毫无悬念的战斗。
<br/>　　丁秋月再战，不过是送死罢了。
<br/>　　而且，他很清楚，气宗对于他上台干扰比拼，反应如此大，甚至咄咄逼人，不过是因为他阻止了气宗打算让丁秋月意外身死一事。
<br/>　　丁秋月再战，桌万宗必定不会再放过机会。
<br/>　　丁秋月摇了摇头，苦涩道，“这不是我一个人的战斗，我必须继续战。”
<br/>　　“可是…”萧逸想说些什么。
<br/>　　器宗席位上，丁家族长沉声道，“萧逸小友，老朽很感激你刚才救秋月一命。”
<br/>　　“不过，我器宗与气宗两脉的比拼，你不懂，还请不要再多加扰乱。”
<br/>　　萧逸皱眉看向丁家族长，“我是不知晓你们两脉的规矩，也不知晓其中情况。”
<br/>　　“但我很清楚，秋月的性命，比什么规矩和比拼都重要。”
<br/>　　“小友言重了。”器宗席位上，一老者沉声道。
<br/>　　“不过是一场比拼罢了，之前那等意外，应当不会再发生。”
<br/>　　这老者，便是器宗老宗主，亦是丁秋月的师尊。
<br/>　　“小子，还不滚下去？”桌万宗冷喝一声。
<br/>　　“器宗老宗主尚且认可丁秋月一战，何时轮到你来操心？”
<br/>　　“莫不是你对丁秋月早便心怀不轨吧？”
<br/>　　“呵，小贼便是小贼，祸害别的女子还不够？还要觊觎一个不过16岁的小丫头？”
<br/>　　此言一出，周遭看向萧逸的目光，霎时变得鄙夷。
<br/>　　确实，萧逸一个外人，这般‘护’丁秋月，还处处操心，不得不让人怀疑是否心怀叵测。
<br/>　　“呵。”萧逸不屑一笑，并未理会周遭目光，而是直视桌万宗。
<br/>　　“你也知道丁秋月不过是个小丫头？”
<br/>　　“值得你这般咄咄逼人，乃至下杀手？亏你还是气宗第一天骄，这等心怀，也配当剑修？”
<br/>　　“我都替你汗颜。”
<br/>　　“混账，你胡说些什么？一派胡言。”桌万宗脸色一变。
<br/>　　萧逸身旁。
<br/>　　丁秋月天真而明亮的眼眸中，泛过一丝疑惑。
<br/>　　“大哥哥，为什么这般护我？”
<br/>　　那双天真无邪的眼眸中，有着的，仅仅是单纯的疑惑。
<br/>　　萧逸笑笑，摸了摸丁秋月的脑袋，“我家侍女，当年走时，也如你这般年纪，豆蔻年华。”
<br/>　　“你还年轻，路也还长，记住，活下去比什么都重要。”
<br/>　　说罢，萧逸转过身，缓缓下台。
<br/>　　下台前，萧逸淡漠地看了眼桌万宗。
<br/>　　“你可以当我胡说，我也可以现在下台。”
<br/>　　“但记住，收起你那等心思。”
<br/>　　“否则，你可以试试是你的剑快，还是我的剑更快更狠。”
<br/>　　冰冷的话语，让得桌万宗身躯一颤。
<br/>　　“慢着。”气宗三长老怒喝一声。
<br/>　　“你说下台便下台？扰乱了两宗大比，岂是你一言可盖之的。”
<br/>　　“敢问城主府的裁判，一个外来武者，扰乱两宗大比，该当何罪？”
<br/>　　气宗三长老，显然在咄咄逼人。
<br/>　　不错，替两宗大比担任裁判的，乃是城主府的武者，便是林家一脉。
<br/>　　作为拥有魂剑力血脉的林家，当之无愧乃是剑域第一家族。
<br/>　　只是，气宗三长老之前刁难萧逸时，却并未询问林家裁判的意见。
<br/>　　直至此时，他明知自己奈何不了萧逸，方才搬出林家以及大比的规矩。
