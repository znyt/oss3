　　这地魔界，就是地魔门的老巢，跟北邙学院一样，是一处独立的空间。
<br/>　　地魔门，是这北域最大的宗门之一，虽然隐藏在地下，但是但凡是北域之人，那个没听说过地魔门的存在。
<br/>　　北邙学院，便是地魔门的死敌之一，双方一在明，一在暗，便是争锋相对。
<br/>　　如今这一次任务，居然是跑到了地魔门的老巢来了。
<br/>　　若是被地魔门的一些强者知道了，只怕是楚风眠一行人，肯定会被直接灭杀的。
<br/>　　每年之中，都有着不少弟子，死在对方的手中。
<br/>　　毕竟北邙学院跟地魔门之间，可以称得上是不死不休的仇敌。
<br/>　　“闭嘴！”
<br/>　　雷绝瞪了华新一眼，冷哼一声说道。
<br/>　　“乖乖跟着我，不会有什么事的，这里只是地魔界的外围，人烟稀少，我们这次的任务，便是前往一处洞窟之中，捣毁其中的一座祭坛。”
<br/>　　“跟我走。”
<br/>　　说着，雷绝身形便是一动，再次化为一道遁光飞走。
<br/>　　楚风眠一行人，急忙跟随了上去，这里看不是北邙学院了，在地魔界中，只怕是危险重重，只能是一路跟随雷绝才行。
<br/>　　一路上，众人都是小心翼翼，一个个都是极为紧张，生怕是被人发现。
<br/>　　终于是两个时辰后，众人在一处洞窟前，降落了下来。
<br/>　　这洞窟中，蕴含有着无尽魔意，一股股魔意，便是从这洞窟中，散发出来。
<br/>　　这些魔意，已经是浓郁到了极点，只怕是圣者，进入其中，都会被魔意侵蚀魔化。
<br/>　　整个地魔界，便是就像是一方魔域一般，对地魔门的魔修来说，是绝对是宝地，一方乐土。
<br/>　　但是对楚风眠这些武者来说，这种魔意，却是极为致命的，一旦是被侵蚀到了身躯之中去，便是有可能受到魔意影响。
<br/>　　轻则心魔深重，重则可能直接会被魔性控制，堕落成魔，成为只知道杀戮的魔头。
<br/>　　看着洞窟中涌现出的无数魔意，众人的脸上都是露出了几分惧色，这魔意足以侵入武者身躯去，令众人直接都想要逃离这里。
<br/>　　但是却没有任何一人，敢真的逃，这里是地魔界，就算是逃，也无处可逃，没有雷绝的指引，他们就连回去的道路都不一定可以寻找的到。
<br/>　　不逃，还有活路，一旦逃了，就算是回到北邙学院，也要落得一个临阵脱逃的罪孽，会直接被斩杀。
<br/>　　所以就算是如今危险重重，也无人敢临阵脱逃。
<br/>　　“走。”
<br/>　　雷绝看了一眼洞窟，直接是一步踏出，直接是走了进去。
<br/>　　“你们快点进来，要是招惹到了地魔门的魔头，你们也都要一起死！”
<br/>　　雷绝进入其中的一刻，也不忘冷声威胁道。
<br/>　　“谁逃！谁死！”
<br/>　　“走吧，进去吧。”
<br/>　　秦莽的脸色虽然恐惧，但是也只能是硬着头皮进去了，的确这个时候，已经是无路可逃。
<br/>　　“没事，只要是完成了任务，我们就可以直接离开了。”
<br/>　　楚风眠轻笑一声，倒是直接步入其中。
<br/>　　这洞窟中，虽然看起来危险重重，但是楚风眠倒是丝毫不惧。
<br/>　　这点危险就惧怕，还修什么武道，那一尊强者，不是历经无数次威严，才蜕变而成的。
<br/>　　“走！”
<br/>　　其他三人对视了一眼，也是坚毅一声，全部踏入到了这洞窟之中。
<br/>　　这洞窟一眼望下去，深不见底，其中无数分支，看起来复杂无比，不知道里面到底是有着多大。
<br/>　　“地魔门的人，都喜欢挖掘这种洞窟，在其中培养魔意，借此修炼，这里应该就是地魔门的一处分部。”
<br/>　　楚风眠步入到洞窟之中，心中暗自想到。
<br/>　　这一次的任务，楚风眠大概也已经明白了，便是应该就在这洞窟之中，有着一座祭坛。
<br/>　　而楚风眠一行人的任务，便是负责在其中的祭坛捣毁。
<br/>　　楚风眠灵识打开，发现在这洞窟的最深处，的确是有着一股庞大的灵力波动，不过其中到底是什么，楚风眠却无法看清。
<br/>　　他的灵识，在这魔意之下也都受到了不小的影响，不过应该就是那祭坛的所在。
<br/>　　“这洞窟中的魔意，倒是极为棘手啊。”
<br/>　　楚风眠跟随这雷绝，在这洞窟深处走去。
<br/>　　在洞窟边沿外围的魔意，虽然恐怖，但是以他们的实力，却还是可以轻易对付的。
<br/>　　但是随着楚风眠一行人越来越走入到这洞窟的深处，其中散发出来的魔意，也越发强大。
<br/>　　已经是快要到了一种，不得不消耗灵力，才可以抵挡的地步。
<br/>　　“这里的魔意，必须要以灵力来抵挡才行了。”
<br/>　　雷绝突然出声，指着莽通道。
<br/>　　“你，走在最前面，动用灵力，来抵挡魔意！”
<br/>　　“什么？让我独自一人来抵挡？”
<br/>　　莽通听到雷绝的话，脸色不由一变。
<br/>　　这里的魔意，简直是在每时每刻都在侵蚀这武者的内心。
<br/>　　莽通抵挡他自己一个人的魔意，倒是不成问题，但是现在雷绝居然是让他，负责抵挡五个人的魔意，这令他脸色不由一变。
<br/>　　要是抵挡五个人的魔意，只怕是很快就要被消耗光灵力，倒时候在这如此危险的环境中，没有灵力，只怕是随时可能陨落。
<br/>　　这雷绝的意思，便是要让莽通当做炮灰，让他在前面开路。
<br/>　　“怎么？你不愿意？”
<br/>　　雷绝看着莽通的反应，脸上露出几分冰冷道。
<br/>　　“废物，让你做什么就做什么，你要是不去抵挡魔意，我现在就杀了你！”
<br/>　　说话之际，雷绝的眼神中便是不满杀意。
<br/>　　这话中的威胁之意，可不是假的，只怕是随时可能出手。
<br/>　　众人也想不到，雷绝居然在这个时候，直接翻脸。
<br/>　　莽通脸色露出几分慌张，结结巴巴的道。
<br/>　　“你，你可是北邙学院的弟子，你杀了我，也会受到学院的惩罚的。”
<br/>　　听着莽通的话，雷绝脸上露出几分不屑道。
<br/>　　“杀了你们几个废物，学院也不可能处罚我，更何况在这地魔界，谁有能知道。”
<br/>　　“小子，快点过去抵挡魔意，完成了任务，还能给你几分好处，若是你不识相的话，现在我就杀了你！”
