　　他望着只有几个月大的自己，内心简直翻江倒海，忍不住就往前探了探身子，想要看得清楚一点，到了一定的角度之后，他就感觉到了脸上皮肤有些发紧，好像被一层无形的膜给挡住了，脑袋越往前伸，这层膜就兜得越紧。
<br/>　　砰……
<br/>　　叶少阳感觉膜被撕开了，自己几乎要掉下去，就在这时，被少妇抱在怀里的小叶少阳正好抬头看着上空，哇的一声哭了起来。
<br/>　　就在叶少阳几乎要掉下去时，一只手从后面拽住他，把他拉了回来。
<br/>　　叶少阳转头一看，一个满脸都是胡子的老道士站在面前。
<br/>　　徐福！
<br/>　　虽然在这里不可能看到别人，能看到的也只有徐福，但乍看到他，叶少阳还是很吃惊，“你不是在昭狱里吗，怎么会在这儿？”
<br/>　　徐福笑道：“你应该问，是哪一个昭狱。”
<br/>　　哪一个……叶少阳冷不丁听出了这句话中隐含的意思，惊道：“你是真徐福还是假徐福？”
<br/>　　“徐福只有一个，哪里有什么真假。”
<br/>　　“别跟我打机锋，你知道我说的是什么，你不是我那个世界的徐福，或者你是，昭狱那个不是。”
<br/>　　徐福还是笑。“小天师，你能想到这一层，也是开了窍，不过你还是错了，我徐福只有一个，你而今看到的，是我一缕魂魄，我的本尊正在昭狱中，前段时间，你不是刚去见过我？”
<br/>　　叶少阳一惊，“你……你在这做什么？”
<br/>　　“自然是等你，当年，我因为机缘得到这山海印，在这里琢磨了几十年，才总算找到操控它的法门，还好几次酿成大祸，我不想让你重蹈覆辙。”
<br/>　　“几十年啊！”叶少阳震惊，简直不可思议。
<br/>　　徐福摆手道：“有我教导你，用不了几十年，十年就行。”
<br/>　　“十年！”叶少阳吓的要哭了，“你逗我玩呢！”
<br/>　　徐福想一下，道：“你这么聪明，或许用不了十年，三年就行。”
<br/>　　“那也不行啊，三个月也不行啊！”叶少阳嚷起来，摆手说道，“那我不玩这个了，我走了。你快送我回去吧。”
<br/>　　徐福上来拉他，道：“三年也好，五年也罢，是我相对人间的时间，做的大致估算，等你出去，也不会就是一瞬间，这个地方，是时光轴，在这里时间是不会流逝的，不管你在这里呆多久，出去都是一瞬间。”
<br/>　　叶少阳目瞪口呆，“这里没有时间流逝，怎么可能。我们……我们现在对话，时间不也在流逝吗？”
<br/>　　“流逝的只是这里的时间，但这里是独立在时间规则之外的，你不管在这里呆多久，等你回到原来的世界，都不过是一瞬间，这一瞬间不是在这里消耗的，而是你来往穿越虚空消耗的。我这么说，你可懂了？”
<br/>　　叶少阳勉强点头，“就这样吧，我也不是科学家，你跟我说说……这些线都是怎么回事，我刚在里头看到了小时候的我。”
<br/>　　“这些，就是时间轴，所有一切纵横交错的，是你的时间轴，”徐福招呼叶少阳来到一根时间轴前，招呼叶少阳来看。
<br/>　　叶少阳凑近了看，看到了十几岁的自己，在茅山上跟青云子一起吃饭，青云子一边抠脚一边看着电视，“自己”则在捧着一本书看。
<br/>　　从这个角度看下去，看不到书的名字。但是叶少阳看到这一幕就想起来了，那是一本《东周列国志》，赶集的时候从书摊上买的，这书看了差不多有一年，因此记得清楚。
<br/>　　那是十五岁的自己啊……
<br/>　　“你再看看这边。”徐福招呼他。
<br/>　　叶少阳摆摆手，道：“我不看了，你有事说事。”
<br/>　　能看到自己的过去，当然是一种十分新奇的体验，但是……“你是不是有一种自己不是自己了的感觉。”徐福一语道破他内心想法。
<br/>　　叶少阳摇摇头，“我不知道怎么说，如果这些都是录像，我会看一个晚上也不停，但是……他们都是活生生的是吧，那我到底是不是我，还是说，无数个时间里有无数个我？哪那一个是真的？”
<br/>　　“没有什么真与假，在无数条时光轴中，有无数个你，也有无数个别人，都是一样的，在你的世界，你就是真的，别的你如果闯进你的世界，他们就是假的。反之……”
<br/>　　徐福随便指了指时间轴上的一段，道：“如果你进入人家的世界，那你就是假的。你懂吗，叶少阳，你只活在当下，只有在当下的世界，你才是真实的，任何人都一样。”
<br/>　　活在当下……叶少阳缓缓点头，道：“可是没人像我这样，真实地看到过去的自己，而且那些自己都是真实的。这种感觉……反正很不爽。”
<br/>　　“这个倒是真的。”
<br/>　　徐福也承认，然后感慨地叹了口气，道：“我第一次到这里来的时候，也跟你一样，看着……看着这么多的时间轴，全都是我，注意，我当时可是一个人，在这里研究它们，寻找驾驭山海印的法门，我一个人啊，研究了几十年，我几乎要精神分裂了！”
<br/>　　“精神分裂？徐老祖，你说话怎么一点不像古人。”
<br/>　　“我是古人，但我两千多年没有去投胎，漫长的岁月里，我连，包括你们这个时代的，我连外语都会一些，你要不要听几句？”
<br/>　　“算了吧。”叶少阳苦笑，抬头看着那近乎无数的时间轴，喃喃道：“这些都是跟我有关的？”
<br/>　　“你每做一次选择，或者被意外干扰，时间轴就会多出一条，无穷无尽。”
<br/>　　叶少阳沉默半晌，道：“那这里都是与我有关的时间？别人的时间轴呢？”
<br/>　　“那自然在别的地方，每个人只能看到自己的时间轴，看不到别人的。”
<br/>　　叶少阳皱眉想了下，道：“不对吧，那你之前为什么带我去了民国，你又去到我的未来。”
<br/>　　“那是因为，过去有我，未来也有我，我用的是我自己的时间轴，你如果想看到别人的过去，例如民国十年，那就顺着自己的时间轴找到那个年代，进去之后，再去找那时候的他，去未来也一样，你不可能去一个没有你的世界。”
<br/>(本章完)
