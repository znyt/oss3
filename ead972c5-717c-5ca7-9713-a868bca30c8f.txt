　　林峰听到厕所开门的声音边舔边看着她。
<br/>　　“老师，你洗好了。”
<br/>　　“嗯，你这是干嘛呢。”
<br/>　　苏云有些不怎么自然的看着林峰这个姿势。
<br/>　　林峰觉得疑惑，“老师，你脸怎么红红的。”
<br/>　　“哦，没什么，刚洗澡水有点热。”苏云明显有点慌。不过林峰很纯洁，倒也没有多想。
<br/>　　“我这不是在吸收基因原液嘛，我担心这个九阶的基因原液太强大了，我身体无法承受，要是爆体而亡那可就死得太冤了。”
<br/>　　“这样啊，可是你不会拿个吸管嘛，为什么要用舌头这样子舔，搞得像是……”
<br/>　　苏云说到这就没有说下去了，反而直接走了。有些落荒而逃的样子。
<br/>　　林峰一时半响都没有回过神来苏云老师这是怎么了，搞得像是什么什么，怎么就不说完呢。
<br/>　　林峰没有多想，而是把注意力全部放在这一锅的基因原液，酸酸甜甜的，非常好喝。
<br/>　　“之前的基因原液都是养乐多的味道，这个和蒙牛酸酸乳没区别。”
<br/>　　林峰一锅全给他干完了，没有任何意外直接突破到了九阶。
<br/>　　九阶，基础力量就是四万斤。
<br/>　　林峰一突破就是足足四万八千斤的巨大力量。远超基础力量，这自然因为内力的缘故，值得一提的是，十阶，似乎是个遥不可及的东西，他没有一个力量的标准。
<br/>　　似乎五万斤也是九阶，十万斤也是个九阶，一切就瞬间失去了目标一样。
<br/>　　林峰一鼓作气把身上的晶石全部用了。
<br/>　　握了握拳头，九阶巅峰的力量。他的力量也从四万八千斤爆增到了恐怖的五万斤。
<br/>　　“为什么，我感觉九阶和十阶之间似乎存在一个看不见的门槛，难道是还差了点什么吗？”
<br/>　　林峰暗自皱眉。
<br/>　　“对了，问系统这个随身小百科。”
<br/>　　“系统，为什么我无法突破十阶。”
<br/>　　系统：要想突破十阶，需要把内力提高到一定的门槛。
<br/>　　“怎么提升？还有这个内力到底是什么。”
<br/>　　林峰问道。
<br/>　　系统：你体内的能量罢了，来源于你体内的细胞，你的细胞无时无刻都在散发能量，其中一部分，被转化为了所谓的内力，内力，是地球上的叫法，内力强大才可以突破十阶。你其实已经有内力了，只是内力还不足够壮大，若是能够捡到天上地下唯我独尊八荒六合第三重，那么你就可以突破十阶了。
<br/>　　“原来如此。”
<br/>　　林峰恍然大悟。
<br/>　　他的内力修炼的是天上地下唯我独尊八荒六合，这个功法目前第二重了，这么看来，他有必要要弄到第三重了。
<br/>　　“这天上地下唯我独尊八荒六合好像是开C级空投开出来的。”
<br/>　　林峰想到此这几天就专门去开C级空投。一些D级空投也都留着合成C级空投。
<br/>　　转眼一个星期就这样过去了。
<br/>　　林峰开的C级空投也有好几个了，但就是没有开到天上地下唯我独尊八荒六合第三重。
<br/>　　“难道这玩意没有第三重？怎么可能，什么功法就特么两重。算了，这东西估计和打麻将插一张牌就胡一个道理，运气不好，死也摸不到，反正我现在也是九阶了，算是这个基地内等级上数一数二的存在了。”
<br/>　　林峰想到此就把小妲己给放出来，让小妲己帮他按摩按摩。
<br/>　　“主人，舒服吗？”
<br/>　　妲己眨巴着两个小眼睛道。她还用了精油，精油这东西自然是在一些按摩店里捡到的。
<br/>　　“舒服。”
<br/>　　林峰实话实话的确很舒服，这小妲己的小手就好像是有魔力一样的。
<br/>　　林峰一抬头就可以看到小妲己的酥胸，透着一股淡淡的奶香味，在自己头顶上摩擦来摩擦去，也不知道是不是这个小狐狸故意的。
<br/>　　“林峰，你干嘛呢。”
<br/>　　苏云不知道从哪里冒了出来看到林峰趴在那里好奇道。
<br/>　　她的视线之中她只看得到林峰躺在那里。小妲己隐身了。
<br/>　　“主人，是你的苏云老师来了哦。”
<br/>　　小妲己眨巴着眼睛道，眼睛之中露出了一抹坏笑。
<br/>　　她低下了身子让自己的酥胸彻底的可以撞到林峰脸上。
<br/>　　那一下一下的就好像果冻一样，特别的软嫩。
<br/>　　“这小狐狸，仗着苏老师看不到他，故意这样子挑逗我吗？”
<br/>　　林峰都快要被胸部撞晕了。
<br/>　　“没什么休息一下。”林峰道。
<br/>　　“哦，那你好好休息吧。”苏云虽然觉得奇怪，但是也没有多想就离去了。
<br/>　　“主人，刚才舒服吗？”
<br/>　　小妲己眼神之中露出一抹狡黠的目光道。
<br/>　　“舒服的很。”
<br/>　　林峰一口就是咬了过去，顿时小妲己一声尖叫。
<br/>　　不过苏云自然是听不到了。
<br/>　　到了下午的时候。
<br/>　　林峰去逛了下店里，几天前上架的尸体已经被买走了，其中最大的买家依然还是军部的科研部门，那个变异穿山甲就是他们买走的，甚至买走的时候还非常高兴。
<br/>　　又是几千晶石到手。
<br/>　　林峰现在完全是土豪了，放眼整个曙光基地，他估计也是非常土豪的存在了。
<br/>　　林峰来到了基地门口看了下排行榜，不知道这军部怎么知道自己实力的，自己的实力已经更新到了八阶。
<br/>　　出现在了前十的行列。
<br/>　　第一依然还是周华，第二还是寒鸦。
<br/>　　前几名基本没有怎么动过。
<br/>　　倒是后面变的非常激烈，现在基本上六阶高手也很多了。七阶的也不像之前那么稀少。
<br/>　　“咦，这不是那个寡妇吗？”
<br/>　　林峰忽然看到了被妲己控制的寡妇。
<br/>　　寡妇虽然被妲己控制了，但不代表他是个傀儡人什么不会做，实际上她什么都会做和之前没区别，只是有关于林峰的一切，都被设定为以林峰为优先的原则来处理。
<br/>　　“寡妇。”
<br/>　　林峰走了过去拦住了她。
<br/>　　“主人。”
<br/>　　寡妇看到林峰立即道。
<br/>　　“你怎么在这里，对了，王家人跑哪里去了你知道吗？”
<br/>　　林峰问。
<br/>　　“不知道，王家人走的很着急，只带走了王家的几个骨干，是战门的人亲自来接他们的。具体去了哪里我只是梅姐的一个下属罢了，无全知道。”寡妇道。
<br/>　　“那骷髅会现在归谁管。”林峰问。
<br/>　　“骷髅会的会长已经彻底的掌管了骷髅会，我被他们给架空了。”寡妇道。
<br/>　　“架空？”林峰皱着眉头道。
<br/>　　“他们知道了王家跑掉的事情，所以立即反水了，这个骷髅会会长是个八阶的高手，组织内还有很多七阶的，我不过六阶不是他们对手，所以他们看到王家不在了后就把我架空了。甚至还要杀我灭口，好在我跑得快。”寡妇道。
<br/>　　“那你现在去哪里？”
<br/>　　林峰问。
<br/>　　“我不知道。”寡妇道。
<br/>　　“干脆去我的老龙口吧。”
<br/>　　林峰道。
<br/>　　“好。主人说什么我就做什么。”寡妇没有拒绝。
