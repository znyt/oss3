　　“就是这里？”苏重看着眼前只有半米宽，一米多宽的裂缝。
<br/>　　此处是一个小山破南面，四周长满参天大树，地下草木丛生。根本就没有下脚的地方。如果不是有地头蛇提供的情报，苏重根本就找不到这里。
<br/>　　裂纹就在地面上，向内看去漆黑一片，丝毫不见底。好似连接着无尽深渊。
<br/>　　看了看周围密密麻麻的藤蔓草木，苏重眉头一皱。
<br/>　　嗡！
<br/>　　气魄猛然一放一收，红色光膜骤然扩散，接着迅速回归自身。
<br/>　　簌簌簌……
<br/>　　四周绿色植被瞬间焦黑，化作黑色粉末无声落地。
<br/>　　以苏重为中心，一个半径三米的黑色空地，突兀形成。
<br/>　　蹲下身，苏重把手探入裂缝，一股股温热气体从地下不断冒出。
<br/>　　有门！
<br/>　　“这种热量，可不是简单的地热！”不再犹豫，苏重纵身一跃，直接跳进漆黑裂纹之内。
<br/>　　跳进去之后，苏重才发现，裂纹下面变得匡阔了一些，纵向也在不断变长。
<br/>　　而且角度极其陡峭，苏重跳下去之后，发觉这是一个角度极其陡峭，近乎垂直的大裂缝。好似地震时候，裂开了口子的大地一样。只不过，大部分裂纹在地下，延伸到地面的只有一个小口，这才让人难以察觉。
<br/>　　越往下越宽阔，苏重伸手抓向身旁墙壁，趁着现在靠近侧壁，用来借力改变自己的方向。
<br/>　　入手一片滑腻。苏重眉头一挑。尽管视线很差，但他的身体练到现在，已经发生了不可思议的改变。即便是在漆黑一片的地下，苏重依旧能看清周围景物。
<br/>　　“似乎是些苔藓。”四下看去，周围石壁上也分布着很多不知名植被。
<br/>　　感受着越来越闷热潮湿的空气，苏重了然：“正是因为这种环境，才催生了布满整个地下的苔藓植被。显然，这些植被以空气中的热量为能源生长。”
<br/>　　依靠地热生长的苔藓植被，苏重以前也听说过。
<br/>　　“这不就说明，地下肯定有岩浆？”苏重眼睛一亮。挥手甩出一根念线钉进石壁，苏重身体急速下坠。
<br/>　　……
<br/>　　地面裂缝入口处，两个人站在苏重开辟的空地内。
<br/>　　“似乎是被高温瞬间灼烧而成。”高大魁梧之人，半蹲在地，伸手捻起黑色粉末，揉捻片刻放在眼前，鼻翼翕动。
<br/>　　“高温？怪不得他要找地下岩浆，这可能和他的念能力有关。”帕里斯通穿着一身绣着金丝的西服，点着脚尖站在灰色地面上，一脸的嫌弃。
<br/>　　“这家伙的念能力到底是什么。类似操作系的念线，可看那种震碎一切的狂暴攻击，又像是强化系。还有这种高温能力，难道是具现化系？”帕里斯通声音中带着浓浓的兴趣。
<br/>　　“世界之大无奇不有，如果有人能够精通所有系别的念能力，我一点都不会意外。”相比于帕里斯通的探究兴趣，比杨德反应很平静。
<br/>　　他一辈子都在致力于探险，见过的稀奇古怪事情一大堆。各式各样的念能力也见过不少。到了他这种地步，不管是自身实力还是见识，都已经让他能够沉稳的面对任何事。
<br/>　　所以他才对黑暗大陆那么执着。人类世界，已经不能满足他那旺盛的冒险欲望，只有凶险万分的黑暗大陆，才能激发他的热情。
<br/>　　“只怪他挡了我的路，不管他有什么能力，都得死。”比杨德平静道，冷漠的好似一条人命，甚至不如一只虫子。
<br/>　　“我要下去了，你呢。”比杨德扫了一眼帕里斯通。
<br/>　　“下去？怎么可能，黑漆漆、脏兮兮，哪里可不是我这种人该去的地方！”帕里斯通大叫着。
<br/>　　“你真不像一个猎人！”比杨德收回视线，看着黑沉沉裂缝。
<br/>　　“我可是有你老爹亲自颁发的猎人执照，怎么就不是猎人。”帕里斯通嬉笑道，旋即变得郑重：“再说，我的猎物和你的可不一样！”
<br/>　　比杨德挑了挑眉，每个猎人，都有着自己的猎物。珍兽、魔兽、财宝、遗迹，总有一个明确的，为之奋斗终生的目标。
<br/>　　“或许吧……”不再理会帕里斯通，比杨德毫不犹豫的跳进漆黑裂缝。
<br/>　　黑色裂缝好似深远入口，刚刚跳进去，身影便消失不见。帕里斯通静静的看了片刻，微风拂过额头发梢，身影一闪消失不见。
<br/>　　有比杨德出手，结果已经注定。他可不想再这一片黑灰的地方，不远处可是就有一个天然温泉，去哪里等着才舒服。
<br/>　　……
<br/>　　苏重身影不断下降，每隔一段距离，就重新射出一条念线，修正自己的位置。不让自己偏离地下岩壁太远。
<br/>　　不一会儿，脚下出现了实地。心念一动，一根念线从脚底射出。咄的一声，两端扎入周围岩壁。苏重蹲在念线之上，打量脚下实地。
<br/>　　地下全是麻绳一样粗的藤蔓，黑褐色一片，延伸到远处。仔细一看，苏重顿时觉得有些熟悉。
<br/>　　“黑褐、粗粝、充满褶皱的纤维，这不就是树木根须的外皮吗？”
<br/>　　“难道这是世界树的根须？这么深？”这一会儿，他可是下降了数千米了！
<br/>　　据说世界树根须能够穿透地幔，进入地下深处吸收岩浆热量，供给自己生长。尖端甚至能够弹入太空中，却依旧能正常生存！
<br/>　　抬眼大量已经看不到边的巨大裂缝，苏重心理一惊。
<br/>　　“这个地下裂缝，不会就是被世界树的根给撑开的吧？”尽管吃惊，可苏重知道，这大概就是实情了。
<br/>　　“顺着根须下潜，一定能找到地下岩浆！”
<br/>　　拽着念线拔高身形，仔细分辨根须走势。很快就找到了根系下行方向。
<br/>　　来到根须与土壤接触出，并未发现裂缝存在。
<br/>　　“看来还是需要自己动手挖掘。”在来之前，苏重就已经有了准备。
<br/>　　收起念线，苏重走到墙壁边缘。眼睛眯起，一股凶悍之气骤然爆发。气魄毫无征兆的冲出体外。
<br/>　　嗡！
<br/>　　黑红色光膜瞬间冲进裂缝侧壁。好似热刀切凝固的油脂，气魄瞬间就撞入侧壁。泥土岩石纷纷化作黑灰落地。
<br/>　　苏重面无表情，缓步而行。黑红色光膜笼罩全身。随着他的步伐前进，触碰光膜的土壤，不断化作黑灰。地下深处，巨大压力造就的坚硬泥土岩石侧壁，竟丝毫无法阻挡苏重脚步。
<br/>　　顶着气魄光膜，苏重顺着世界树根须，向着地下缓慢而坚定的前进。
