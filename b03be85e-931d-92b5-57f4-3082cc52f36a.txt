        晚餐就她和童筱颖还有刘婶三个人吃饭，傅夜沉并未在家。
<br/>
<br/>        饭菜都上了桌，童筱颖见连星儿头发未干，又把这丫头给轰上楼去吹头发。
<br/>
<br/>        连星儿一边往楼上走时，一边说道：“妈咪，你明天给我做佛跳墙呗！”
<br/>
<br/>        “这顿饭还未吃，你就想着明天的饭了。真是个小吃货！”童筱颖宠溺地笑道。
<br/>
<br/>        连星儿俏皮地向童筱颖吐了吐舌头，快速跑上了楼。
<br/>
<br/>        另一边，明司寒回到自己的别墅里时，一直住在他别墅里的裴萱妍，已经替他做好了晚饭。
<br/>
<br/>        上次她跟他去北城，帮他办事，不但把事情办砸了，还害他赔了人家一辆出租车的钱。
<br/>
<br/>        后来，裴萱妍就一个人跑回来了。
<br/>
<br/>        明司寒也没多说什么。
<br/>
<br/>        而吃饭的时候，裴萱妍见明司寒一直盯着手机看，忍不住抱怨了一句：“你爸妈没教过你，吃饭的时候不许看手机吗？”
<br/>
<br/>        “……”明司寒抬眸瞥了裴萱妍一眼，懒得理会她，继续盯着自己的手机屏幕。
<br/>
<br/>        不是说好了要给他发短信报平安的吗？
<br/>
<br/>        连星儿那丫头怎么还没有发？
<br/>
<br/>        她到底搞什么去呢？
<br/>
<br/>        “喂，你这是在等你心上人给你发短信？”裴萱妍一边夹菜，一边问道。
<br/>
<br/>        明司寒没吭声，不过倒是放下了手机，开始端起碗筷准备吃饭。
<br/>
<br/>        裴萱妍又滔滔不绝起来：“我跟你说，连星儿那丫头从小就没心没肺。你就别指望她会喜欢上你了。”
<br/>
<br/>        听完裴萱妍这话，明司寒顿时不可思议地抬眸盯着她。
<br/>
<br/>        被明司寒盯得浑身不自在后，裴萱妍又问道：“你这么看着我做什么？我是秀色可餐吗？”
<br/>
<br/>        “你和星儿从小就认识？”明司寒诧异。
<br/>
<br/>        裴萱妍意识到自己说漏嘴后，不得不坦白道：“是我认识她，只是她不认识我罢了！更何况，我们组织里的人都认识她。傅夜沉的宝贝女儿嘛！谁不认识。”
<br/>
<br/>        “所以，上次星儿在饭店里被人下了催眠药，你才会在第一时间知道？”明司寒淡然道。
<br/>
<br/>        裴萱妍咧嘴一笑：“不然呢？”
<br/>
<br/>        “……”明司寒没再吭声，淡定地继续吃饭。
<br/>
<br/>        裴萱妍受不了这安静的气氛，又按捺不住地接着说道：“我跟你说，你喜欢谁都行，就是不要去喜欢连星儿。连星儿那丫头从小到大就被她父母给宠坏了，一身的坏毛病。她……”
<br/>
<br/>        “你在我家里还要住多久？”明司寒直接打断了裴萱妍的话。
<br/>
<br/>        裴萱妍咧嘴一笑：“住到上级对我委派新的任务为止。”
<br/>
<br/>        “那这房子，我给你住好了。我去我爸妈那儿住。”明司寒不痛不痒地说。
<br/>
<br/>        裴萱妍立马阻止道：“诶！你别走啊！好不容易回来一趟，就当是给这房子增加点人气嘛！”
<br/>
<br/>        “孤男寡女共处一室，你觉得妥吗？”明司寒很没好气地说。
<br/>
<br/>        裴萱妍嘴角微扬：“我都不怕，你怕什么？更何况，你这别墅这么大，我两也不算是共处一室吧？还有……”
<br/>
<br/>        话说到一半，裴萱妍又想起了什么，不禁看着明司寒，呵呵地笑了起来：“噢——我知道了！你是怕连星儿误会我两有个什么是吧？你放心好了，她肯定巴不得我两真有点什么。”
<br/>
<br/>        “这饭你自个吃吧！”明司寒顿时心烦气躁地放下了碗筷，从餐桌前起身，拿起手机去了楼上。
<br/>
<br/>        此刻已经快八点钟了，但是连星儿仍旧没有给他回短信。
<br/>
<br/>        她是不是真的一点都不把他放心上？
<br/>
<br/>        要不，主动给她打个电话？
<br/>
<br/>        明司寒心里想着，刚准备拨打连星儿的手机号码，随即又犹豫了。
<br/>
<br/>        他为什么要给她打电话啊？
<br/>
<br/>        她都不给他打一个电话。
<br/>
<br/>        明司寒踟蹰了片刻，还是将手机给收了起来。
<br/>
<br/>        另一边，连星儿吃过晚饭后又陪着童筱颖去散了一会儿步，回来后，直接趴在床上看起来漫画书。
<br/>
<br/>        她从来就不用去考虑柴米油盐酱醋茶的问题，不管在哪里，都能随心所欲地去做自己喜欢的事。
<br/>
<br/>        当她看漫画，看着看累了以后，便直接趴床上睡着了，完全不记得要给明司寒回电话或者是短信的这回事。
<br/>
<br/>        而明司寒等她连星儿的消息，等了一晚，虽然明明知道她一定已经平安到家了，可是他却莫名其妙地堵着一口倔气，就是不主动给她打电话。
<br/>
<br/>        第二天一大清早，连星儿躺床上倏然睁开了双眼坐起身来。
<br/>
<br/>        下一秒，她立马从床上跳起来直奔卫生间。
<br/>
<br/>        卧室门外正在打扫走廊的刘婶，听到屋内的动静后，叩门走了进来。
