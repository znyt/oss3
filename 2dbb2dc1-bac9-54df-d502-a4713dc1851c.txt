　　“这……这是——”
<br/>　　经过一番仔细探查，功夫不负有心人，终于，他在这精血更深处的因子之中，发现了一段神秘的因子。
<br/>　　这因子平淡无奇，初一看，并没有什么不对，但万劫魔尊乃何许人也，尽管它很普通，但他还是从中发现了一丝不寻常之处，那是属于轮回独有的气息。
<br/>　　那是一小段轮回之力形成的因子，只不过，这点轮回之力实在是太少了。同时，不知道是不是错觉，他还从中隐约发现，这段轮回因子中蕴含着一股苍茫古老之力。但等他仔细再看之时，那种感觉却没有了。
<br/>　　之后，他也没放在心上，仔细开始研究起这三滴精血。
<br/>　　……
<br/>　　数十年过去了，这一天，当帝俊、太一等妖族高手还沉浸在建立天庭，凌驾苍生之上的快感之中的时候，陡然间，整个洪荒大地，剧烈颤动，山河倒巅，乾坤颠覆，山摇地晃中，十二股强悍的气势直冲九霄，这气势弘磅礴，其煞气滔天，轰隆隆崛起。
<br/>　　整个洪荒所有的修士脸色狂变，大神通者也都纷纷将目光投到这十二股惊天气势之中，此时他们看到，在洪荒大地东方、南方、北方三个方向，有着十二道通天黑色光柱，升空而起。此光柱贯穿天地，好似蕴含着法则之光，而光柱之间，煞气弥漫，让众准圣瞳孔直缩，他们能够看到，每一道光柱之内，都有一尊强悍的存在，那赫然是十二祖巫的本体形象，他们仰头望天，彼此之间仿佛相连，形成一个宏大的阵势。
<br/>　　“嘶！？这是——？十二祖巫？怎么回事？”各路强者面面相觑，不知道他们要干什么。
<br/>　　“这巫族居然有十二尊堪比准圣的存在，这……这怎么可能？看他们的样子，一个个煞气滔天，气势惊人，莫非是要对付妖族？”
<br/>　　“哈哈哈，前有妖族立天庭，今有巫族出世，天道之下，果然万物制衡啊。”
<br/>　　“这巫、妖两族，皆不可小觑，就是不知道谁强谁弱？”
<br/>　　一时间，洪荒宇宙，各路大神通者，纷纷反应过来，神念在虚空之中碰撞，议论纷纷。
<br/>　　与此同时，太华山劫魔殿深处，一处充满劫煞之气的空间中，盘坐在十二品黑莲之上的万劫魔尊睁开双眼，眼中闪过滔天煞气，这杀气比之十二祖巫还多，他看了一眼十二道惊天光柱，道：
<br/>　　“要开始了么？呵呵！巫妖之战，不可避免，大劫即将来临，本尊能够感受到，天地之间的劫煞之气已经开始增加了。哈哈，这对于别人来说，是灾，是难，是劫，但对于本尊来说，这就是一场豪华大餐。哈哈哈！好！巫族已经出世，不知你妖族又该如何应对？帝俊？太一？女娲？哈哈哈！”
<br/>　　这个时候，众目睽睽之下，却听到十二祖巫大声喝道：
<br/>　　“盘古在上，今有盘古父神血脉巫族出世，理应执掌洪荒大地！以盘古殿镇压气运，巫族，立！”
<br/>　　“巫！巫！巫！……巫族无敌！巫！巫！巫！”
<br/>　　顿时，无数巫族强者仰天大吼，煞气冲天而起，直达九霄，惊天动地。
<br/>　　便在这时，那苍穹之上，一道金光灿灿的功德之光降落而下，分成十二份，落在十二祖巫身上，虽然十二祖巫没有元神，不修道行，只修肉身，但那是曾经，现如今他们修炼的万劫魔尊传授的巫神法门，已经开始开辟法则空间了，这些天道功德降下，刚好落在他们的法则空间之中，加快空间的开拓。
<br/>　　每个祖巫的肉身强度也提高许多，甚至已经有两个堪比准圣中期之修了。一时之间，巫族的实力大涨。
<br/>　　不周山，女娲与伏羲的道场中，女娲与伏羲虽然在三十三天有属于自己的宫殿，但他们并没有在天庭居住。此时女娲俏脸阴沉地望着洪荒之中的那十二道惊天的光柱，他的眼皮忍不住狂跳起来，惊声道：
<br/>　　“我妖族有如此大敌，之前我们竟然不知？嘶？十二祖巫，十二个准圣，甚至有两个突破了准圣中期？为何会如此？”
<br/>　　“大劫？噗噗！我看到了，尸山血海，血流成河……”就在此时，一旁掐指演算的伏羲大圣，大叫一声，突然两色变得狰狞无比，狂喷一口鲜血。
<br/>　　“哥哥！你怎么样？为何会如此？你看到了什么？”女娲大吃一惊，他知道自己的哥哥演算之道无匹，但如今竟会遭到如此反噬，马上关心道。
<br/>　　伏羲脸色苍白无比，神色之间带着一股强烈的震撼，看着自家妹妹女娲道：“我只是算到未来的一角，但，仅从我看到的那一丝，便可推断出。劫，这是一场浩大的劫，虽然没有看到最后的结果，但，我知道，妖族前途堪忧。”
<br/>　　说着，他脸色见透露着强烈的不甘，突然，他仿佛下定什么决心，对着女娲叮嘱道：“妹妹，你听我说，以后妖族之事，你尽量少管，为兄算到你不就将会有大机缘，这关系到你的圣道。此时与妖族有太多牵连不好，至于帝俊、太一那边，交给我来应付。”说着，伏羲脸上流露出凌厉之芒。
<br/>　　“哥哥，你到底看到了什么？怎会遭如此反噬？”女娲焦急的关心道。
<br/>　　伏羲却摇头不语，默默地看着远处，那里正是十二祖巫出世之地。直到女娲问急了，他才缓缓说道：“不可说，天机不可泄露。”
<br/>　　就在此时，那苍穹之上，三十三天方向，陡然间一道凌厉的剑光，带着势不可挡的寒光，滔天的凶煞之气，砍向十二祖巫。速度之快，仿佛跨过无尽空间，斩向虚无。
<br/>　　“嗯？这是……东皇太一出手了？”各路大神，包括万劫魔尊，都将目光投向苍穹，一个个目光凝聚，要看看这巫妖之间的争斗。
<br/>　　“本皇倒要看看，汝等何能，竟敢执掌洪荒大地！”一个充满霸气的声音响彻云霄，刹那间，回荡在苍穹八方。剑光凶煞，凌厉无比，朝着巫族方向，砍了下来。
<br/>　　无尽空间，尽皆破碎，一个个小世界被开辟出来但又很快被破灭，混元剑带着无以伦比的凶煞剑光，让苍穹都为之失色。
<br/>　　“太一小二，休得猖狂！！阵起！吼！”帝江猛然大喝。
<br/>　　顿时十二祖巫，纷纷脸露狰狞，蓦然间星辰一道阵法，聚拢天地煞气，这阵法一成，立刻带着一股强大的凶煞之气，冲天而起，轰隆隆之间，迎向太一。
<br/>　　这是都天神煞大阵的雏形，他们还没有参悟透大阵，凝聚盘古真身。但仅此，对付同样没有周天星辰大阵的妖族，已经足够了。
<br/>　　“妖族何能？安敢立天庭！”帝江祖巫猛然大吼一声，厉声道。
<br/>　　阵光与大阵煞气对碰，消散于虚空，两者相互僵持起来。各路大神纷纷倒吸一口气，立刻明白，这巫族果然有所准备。能与妖族对碰，岂是简单之辈。好生厉害！
<br/>　　“既然如此，那便让尔等看看，我妖族能力天庭的能耐！”就在此时，九天之上，妖帝帝俊那浑厚的声音在洪荒大地之上响起，回荡在四面八方。
