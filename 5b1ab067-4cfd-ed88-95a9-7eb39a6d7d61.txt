　　山中无老虎，猴子称大王。
<br/>　　看着老头子乘坐的战舰远去，李承乾的心中一下子轻松了不少。
<br/>　　这段时间以来，老头子虽然禅位了，但对于他来说总是觉得事情有些不大对头，或者说不管干什么，下什么样的命令，都觉得像是有一双眼睛在盯着自己。
<br/>　　现在，老头子离开了，那种被盯着的感觉一下子就消失了，只觉得浑身轻松，似乎有使不完的劲。
<br/>　　看来还是老头子给自己的压力太大，虽然心里不想承认，但潜意识中一直很有压力吧。
<br/>　　李承乾如是想着，回头看了看那些跟着出来的朝臣：“都回吧，明日别忘了上朝！”
<br/>　　“是！臣等告退！”一群或老或少的家伙们彼此对望一眼，躬身退去。
<br/>　　时间不大，原本人山人海的码头已经只剩下李承乾和他的几个大小老婆，再有便是一群木头一样的禁军。
<br/>　　“陛下，您……”
<br/>　　“你们也回去吧，我想一个人静一静。”打断了苏玫的话之后，李承乾再次将目光转向河面，看着那流淌了不知多少年的渭河默默的发起呆来。
<br/>　　苏玫无声的抿了抿嘴，转头又看了看另外几女，摇了摇头，没人知道李承乾现在想的是什么，但却知道在这个时候不应该去打扰他。
<br/>　　……
<br/>　　黄昏，日影西斜。
<br/>　　在河边站了一天的李承乾终于有了一丝反应，只见他默默的转身，然后迈着机械的步子一点点的向马车那里走去，小白和夜魅想要扶他，但被其拦住。
<br/>　　自从李承乾来到大唐这个世界上开始，李二的影子就一直笼罩着他，而现在那影子突然远离，又让他有些不舍。
<br/>　　纵然是从今以后他得以尽情施展自己的抱负，那种快活的感觉也无法压制心中的失落。
<br/>　　兴奋与失落两种情绪纠结在一起，让李承乾一时间很难说清楚自己到底在想法什么，整整一天的时间，其实他的脑子完全就是一片空白，没有任何的思想存在。
<br/>　　……
<br/>　　半年之后，茫茫南海。
<br/>　　曾经的帝国主义头子，现如今的太上皇李二陛下站在巨大战舰的船头上，极力眺望着远方，半晌之后回过头：“青云啊，你说程老货发现的那个大岛在什么位置来着？”
<br/>　　“太上皇，继续向前再有两个月左右也就到了。”独孤青云站在李二的身后，一切就像回到了二十年前一般。
<br/>　　“还要两个月？朕记得似乎从长安出来半年多了了吧？”李二皱了皱眉头。
<br/>　　“六个月零九天。”林松海与独孤青云尴尬的对视了一眼，报出了一个准确的数字。
<br/>　　“六个月零九天，呵呵，想不到，朕的江山竟然如此庞大，算下来如此从最北边走到最南边怕不是要一年才成。”李二看着远处波涛起伏的海面，喃喃说道。
<br/>　　“太上皇，如果用走的，怕是一辈子也到不到。”长孙无忌不知什么时候从船舱里走了出来，听到李二的话之后，半直半假的开起玩笑。
<br/>　　“无忌，你这可是属于抬扛。”李世民听到长孙无忌的声音，转过头用手点了点他，脸上露出一抹笑容。
<br/>　　“臣这可不是抬扛，而是实话实说。”长孙无忌笑着来到李二的身边，落后半个身位站定有些感慨的说道：“算来自大业十三年到现在已经快要三十年了，沧海桑田，物是人非喽！”
<br/>　　“大业十三年……”李世民眼中闪过一抹怀念之色。
<br/>　　那年五月，老爹李渊杀王威、高君雅起兵，带着他们兄弟几个集三万大军南下，斩宋老生，收孙华义军，而后攻取长安……。
<br/>　　而后便是长达十年的征战沙场，他李世民也从一个十八岁的懵懂少年成长为二十八岁的青年。
<br/>　　薛仁杲、刘武周、窦建德、王世充、薛举……，一个个名字闪过脑海，奸诈的，狂妄的，亦或是卖国求荣的，这些人全都是成为他的踏脚石。
<br/>　　玄武门那一夜刀光血海，老大建成，老四元吉尽数毙命，子嗣死绝，现在想想，当时过于冲动了。
<br/>　　而接下来的日子过的就有些莫名其妙了，随着家里大小子年龄的增长，各种奇思妙想层出不穷，老一代渐渐被新生代所接替。
<br/>　　突厥之战，薛仁贵与苏定方的崛起；西域之战，尉迟宝琳崛起；辽东之战，李承乾亲自带兵……。
<br/>　　不知不觉间，大唐的疆域越来越大，到了现在，可能连那小子自己都不知道自己掌握着多大的国土了吧。
<br/>　　想着想着，李二不觉有些痴了，他的一生经历过生离，也经历过死别，已经足够精彩。
<br/>　　不过，难道这样就够了么？这个世界很大，大到几乎没有边界，虽然那小子说这个世界其实是一个球，一直向一个方向走最后一定会回到原点。
<br/>　　但李二不信，这个世界怎么可能是个球呢？如果那样的话，球另一边的人不是掉下去了！
<br/>　　所以李二要亲自看看，看看这个世界到底有多大，到底有没有神仙，若有机会自己也学学当年老祖宗出函谷关，来一次登仙之旅，到时候看自家那小子如何超越。
<br/>　　想到这里，李二不自觉得露出一抹笑容，抬手指了指不远处隐隐一条黑线说道：“青云，那是是什么地方。”
<br/>　　“应该是一座无名岛吧，太上皇，南海这里岛屿众多，有很多根本没有名字。”独孤青云向远处看了看，又打量了一下周围的景物，摇了摇头，表示自己也不知道那是什么地方。
<br/>　　“唔，无名岛，朕刚刚用望远镜看，那里似乎有烟，想必是有人在岛上活动，走，我们也上去看看。”李世民听了独孤青云的解释之后，点了点头，做出了一个让众人哭笑不得的决定。
<br/>　　要知道，这一路上，这位前任皇帝陛下已经不知道登了多少次岛，几乎只要是看到有岛便会上去看看，而且理由也是层出不穷。
<br/>　　这一次用岛上有烟还算是不错了，上上一次的时候理由才算奇葩，竟然是在船上拉不出屎来，也不知道他是怎么想到的。
