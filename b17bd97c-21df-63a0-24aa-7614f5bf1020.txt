　　轰隆隆隆隆！
<br/>　　随着叶尘这道淡漠的声音落下，一股惊天气势，陡然自他的身体中迸发而出，直冲九天云霄！
<br/>　　蹬蹬蹬！
<br/>　　周围所有人全都大惊失色，不由得纷纷朝四面倒退而去。
<br/>　　嘭嘭嘭！
<br/>　　天台周围的那些禁制力量，在这股强大的气势冲击之下，竟然全部破碎开来。
<br/>　　继而，
<br/>　　咔嚓嚓嚓！！
<br/>　　众人发现，他们脚下的整座大厦，都瞬间全部龟裂开来，但是诡异的是，却并没有立即崩塌。
<br/>　　所有人全都忍不住瑟瑟发抖，这是一种发自本能的畏惧，就犹如兔子面对雄狮，是一种来自等级上的压制！
<br/>　　不但天台上的所有人，被这股强大到逆天的气息震慑，就连网络直播间里的观众，此刻也全都被镇住了，继而纷纷议论了起来，
<br/>　　“什么情况？”
<br/>　　“这个年轻人是谁？貌似很强的样子！”
<br/>　　“我记得华夏武道界年轻一代中，向来以祝小白为首，可是这人的气势，似乎比祝小白还要强大很多啊！”
<br/>　　“看来我们华夏还真是卧虎藏龙！”
<br/>　　“等等，听他刚才说的那话，似乎也是狂仙门的人！”
<br/>　　……
<br/>　　台下的鱼瑶瑶，望着那道犹如天神般令人心悸的瘦削背影，顿时美眸瞪得老大，玉手捂着嘴巴，满脸的不可思议，喃喃自语道：
<br/>　　“他……他竟然……这么强！！”
<br/>　　而那位之前，完全没有将叶尘放在眼里龙长老，此刻也两眼瞪得老大，简直无法相信眼前看到的一切，指着叶尘，结结巴巴，道：
<br/>　　“你！你究竟是……什么人？！！”
<br/>　　“呵呵！”
<br/>　　叶尘发出一道充满嘲讽的笑容，一字一顿，缓缓开口道：
<br/>　　“鄙人，叶，狂，仙！”
<br/>　　说完这话的同时，叶尘缓缓抬起头来，终于在众人面前，露出了自己的面目。
<br/>　　轰！
<br/>　　这一刻，
<br/>　　不但天台上的众人，全都看清楚了叶尘的脸庞，空中的无人摄影机，也自动锁定了画面，传送到了直播间平台上，
<br/>　　“叶狂仙？这名字好像在哪听过？”
<br/>　　“楼上小学生啊！连叶狂仙的大名都没有听过，还好意思说自己了解武道界？”
<br/>　　“还是让哥给你普及一下知识吧！叶狂仙便是在十多年前，被誉为全球武道第一人的叶仙人！狂仙门的开创者！”
<br/>　　“什么！他就是当年那位，凭着一己之力，压高丽，平岛国，横扫东南亚六国军队，令米国俯首，全球都为之颤粟的叶仙人？”
<br/>　　“真的是叶仙人吗？没想到他居然这么年轻啊！”
<br/>　　“传说他早在十多年前就已经失踪了，有人说他已经陨落，怎么今日突然活过来了？”
<br/>　　“放屁！叶仙人在当年，可是公认的全球第一，怎么可能会这么容易陨落？”
<br/>　　“哈哈！这下有好戏看了！”
<br/>　　……
<br/>　　在亿万双眼睛的瞩目之中，叶尘身上的气息也平复了下来，再次恢复了之前的普通模样，然后朝唐清雅缓步走了过去。
<br/>　　“叶……叶尘，真的……真的是你吗？我不是在做梦吧？”
<br/>　　唐清雅望着眼前那张，令她朝思暮想的脸庞，眼眶瞬间通红，眼泪犹如决堤的洪水，哗哗的流了下来。
<br/>　　叶尘走到唐清雅跟前，伸手抹去那绝美脸庞上的泪痕，冲着她咧嘴一笑，
<br/>　　“你当然不是在做梦！清雅，我回来了！”
<br/>　　说完这话，叶尘张开双臂，
<br/>　　唐清雅微微一呆，旋即犹如飞鸟入林般，直接扑进叶尘的怀里，紧紧埋在了叶尘的胸膛之中，沉浸在你久违的温暖感觉片刻之后，忽然又想到了什么，顿时轻哼一声，道：
<br/>　　“叶尘！你这个大混蛋！明明说好十年之内回来的，却说话不算话，你知道我们等你等的有多辛苦吗？混蛋！混蛋！混蛋！”
<br/>　　唐清雅一边说着，秀拳在叶尘的身上不停的捶打着，似乎要将这十多年的思念，全部用这种方式发泄出来。
<br/>　　直播间里的观众们，看到这一幕，顿时嚎叫起来，
<br/>　　“我的女神啊！竟然就这么被人抢走了！”
<br/>　　“这个叫什么叶狂仙的，我要和你单挑！”
<br/>　　“得了吧！那可是昔日全球第一人，早在十几年前，连世界第一强国的米国都被他压得抬不起头，就凭你也配？”
<br/>　　……
<br/>　　直播间里的观众，毕竟隔着屏幕，虽然被叶尘的出场惊到，但还不至于感到畏惧，可以肆无忌惮的调侃。
<br/>　　可是现场的众人，可就没有这个胆量了，所有人全都连大气都不敢喘，尤其是那些曾经见识过叶尘手段的老一辈武者们，更是满脸的忐忑之色。
<br/>　　此时京都，
<br/>　　当华夏众将看到叶尘突然出现在擂台之上，顿时全部霍然站起身来，一个个虎目通红，满脸激动之色，
<br/>　　“叶帅！”
<br/>　　“真的是叶帅吗？”
<br/>　　“哈哈！叶帅回来了！我们华夏有救了！哈哈哈！”
<br/>　　……
<br/>　　这些平日里，不苟言笑的钢铁男儿、国之栋梁，此刻却全都高兴的如同孩子，就连易山河，如今已经贵为华夏军方的头号人物，此刻也激动的直搓手，两行老泪不由自主的滑落了下来，低声喃喃道：
<br/>　　“叶仙人！叶先生！叶小子！真的是你吗？没想到老朽有生之年，竟然还能再见到你！”
<br/>　　而此刻的叶尘，似乎早已经把周围的人全都忘了，任凭唐清雅那轻柔的拳头，砸落在他的身上，等到她发泄的差不多了，方才拍了拍她的香肩，温声道：
<br/>　　“对不起，都是我的错，是我连累大家了！对了，清雅，岚姨他们没有和你在一起吗？”
<br/>　　“岚姨……”
<br/>　　唐清雅正要开口回答，忽然一眼瞥见后方，青龙派的那位龙长老，正悄悄祭出一件法宝，朝叶尘攻击而来，顿时大惊失色，连忙提醒道：
<br/>　　“小心后面！”
<br/>　　然而，
<br/>　　面对唐清雅的提醒，叶尘连头都没有回，脸上依旧挂着淡淡的笑容，
<br/>　　“区区一只蝼蚁而已，还伤不了我！”
<br/>　　说话间，龙长老已经挥舞着一把金色巨斧，朝叶尘狠狠劈落下来，
<br/>　　“叶狂仙！我今日便为死在你手下的药王宗前辈们报仇雪恨！！”
<br/>　　龙长老显然没有丝毫留手，趁着叶尘和唐清雅叙旧的时候，突然发出了自己的最强一击。
