　　只一下子，巨浪将所有人都拍到了水中，虽然这无色海中可以呼吸，但当众人试图往上游的时候，才发现根本不是之前那样了：水没有了浮力，反而脚下产生了一股吸力，将他们生生往下面拽去。
<br/>　　越是往下，海水越凉，而且随着海水的流淌，从他们身上带走修为的速度也越快，甚至不光是修为，连他们的神识意念，也在一点点被瓦解带走。
<br/>　　众人拼命挣扎起来。
<br/>　　小马和吴嘉伟一人拉着叶少阳一条胳膊，朝上面拖拽，他们法力深厚，一时半会还能抵抗得住吸力，努力浮升到水面上。
<br/>　　这时候，四个僧人中的一个开始念咒，声音穿过水域传到下面那些生灵的耳朵里：“文殊师利，导师何故、眉间白毫，大光普照，雨曼陀罗、曼殊沙华，栴檀香风，悦可众心……”
<br/>　　众人在下沉过程中听到这咒语，先是困惑，继而明白，这是让他们跟着念，于是互相看去——水很清，也不像人间的水那样能阻挡视线，在这里一切都看得清清楚楚。
<br/>　　沉默了半晌，终于之前那个和尚先张口念了出来，才念到一半，身下的吸力立刻消失，浮上了水面，周身那种冰冷的感觉也没有了，反而很舒服，继续静心念了下半句。
<br/>　　接着身边又冒出了几个脑袋，都跟着一起念，念完之后，偷偷看叶少阳，一脸的愧疚。
<br/>　　他们也不傻子，大家都明白，只要跟着念了经文，就等于是向轩辕山宣誓了效忠，也就等于背叛了叶少阳。
<br/>　　“以是因缘，地皆严净，而此世界、六种震动。时四部众、咸皆欢喜，身意快然，得未曾有。”
<br/>　　这时候那盘古僧又念了第二句咒语。
<br/>　　“叛徒，不要念了！”小马望着身边几个人大喝。
<br/>　　那几个人感到羞愧，一时犹豫，又沉了下去，第一个念经的和尚却义无反顾地跟着念。
<br/>　　小马松开叶少阳，游到那人面前，道：“不要念了，听到没有，你真愿意给他们当狗啊！”
<br/>　　和尚脸红了一下，随即说道：“马王爷，蝼蚁尚且偷生，况且……如果真能到轩辕山，倒也是一场机缘，你自己不去，为何要阻挡别人？”
<br/>　　小马一怒之下，要用板砖拍他，被吴嘉伟拉住，劝道：“人各有志，你管他们做什么，还是想想办法怎么出去！”
<br/>　　小马环顾左右，喃喃道：“也真是奇怪啊，这法阵怎么说成就成，还如此厉害，这到底是什么法阵！”
<br/>　　水中的吸力越来越强，他们三个又沉了下去，在这过程中，看到身边不断有人念着咒语浮起来，突然心生一计，两只手分别抓住两个人的头发，借着他们的浮力升起来，到水面上，反倒把两个人按下去，自己浮起来。
<br/>　　两个人在下面挣扎，小马全然不顾，嘎嘎笑着。
<br/>　　那两人又不敢不念咒，这样一来，等于是两人用自己的浮力将小马托了起来。
<br/>　　“哈哈，你们奈我何啊，我不还是浮起来了！”
<br/>　　小马说着，把叶少阳捞起来，正好按在了那个和尚的肩膀上，让叶少阳骑在他身上，那和尚念咒产生的浮力，负担不起两个人，反倒把叶少阳托出水面，自己在水中蹬着两腿，挣扎起来。
<br/>　　鲲鹏他们看到这一幕，内心也是无语的。
<br/>　　这种不按套路出牌的办法，也真是没谁了。
<br/>　　鲲鹏一怒之下，双手结印，射出一道神光，穿过电弧的缝隙，朝小马打过来。
<br/>　　小马灵机一动，从身前抓起了一个人，当做盾牌挡在自己身前，神光打在这人身上，只听一声惨叫，这人身躯立刻四分五裂，化成一堆血肉，但也替小马阻挡了这一下。
<br/>　　鲲鹏大惊，骂小马无耻，用别人挡刀。
<br/>　　“我就是恶人啊，我从来也没说自己是好人，你要再动手，我还拿他们挡，反正杀人的是你！”小马这混不吝的态度，倒让鲲鹏张口结舌。
<br/>　　“你们这是当了叛徒，我也不为难你们，但如果人家动手，我就拿你们挡刀，你们死了别怪！”
<br/>　　小马这么一说，这些生灵都面带愧疚地朝四周游过去，躲得远远的。
<br/>　　鲲鹏没再动手，反而冷笑一声，双手飞快结印，口中念咒，水面再一次起了惊涛骇浪，一个浪头接着一个浪头砸下来，将所有生灵分开，小马和吴嘉伟自己是没问题，但要照顾叶少阳，反而三个人一起沉了下去。
<br/>　　突然，一道紫气从水中飞射而出，砸在结界上，带起了一大片的电弧火花，结界剧烈地摇晃了一下。
<br/>　　鲲鹏大惊，定睛看去，紫光消散之后，是一把剑，重新落入水中。
<br/>　　七星龙泉剑！
<br/>　　鲲鹏立刻明白过来，哈哈大笑：“叶少阳啊，你而今只有一只手能用，还是左手，你如何能发挥七星龙泉剑的威力，想冲开这剑阵，简直是蚍蜉撼树！”
<br/>　　影魅突然转头看了他一眼，冷冷道：“话太多了！”
<br/>　　鲲鹏被训，急忙低头，十分恭敬地说道：“师伯教训的是！”
<br/>　　接着继续作法，在水中搅起了惊涛骇浪。
<br/>　　叶少阳方才卯足力气、用左手捏决，使出了一剑，虽然失败了，但鲲鹏的话，却被他听到了“剑阵”两个字，什么剑阵？
<br/>　　波涛一浪接着一浪，疯狂地砸下来。
<br/>　　盘古僧的咒语也还在念着。
<br/>　　许多生灵都耐不住生死的考验，选择了屈从，跟着念咒，一个个浮到了水面上，留在水中的，没有几个人了。
<br/>　　只有叶少阳几个人坚持不念咒，在水中沉下去。
<br/>　　他们倒不是不能吃眼前亏的，但眼下，这种念咒的行为，象征意义太强了，以他们的性格，是不可能屈服的。
<br/>　　在下沉过程中，小马和吴嘉伟还紧紧抓着叶少阳。
<br/>　　他们当然不可能放开叶少阳。
<br/>　　叶少阳刚才用左手试了一下，失败之后，更加明白，光凭着自己一只手的实力，根本无法冲开结界，只好继续吐纳，用真气一点点将法令虫朝手腕方向逼。
<br/>　　已经逼到了手肘部位，但是法令虫死死抓着他的经脉，这种将他生生推动的过程，牵动经脉所造成的痛苦，根本没法形容，叶少阳死死撑着。
<br/>　　（法国队果然赢了……开心，明天晚上荣耀堂发红包庆祝！）
