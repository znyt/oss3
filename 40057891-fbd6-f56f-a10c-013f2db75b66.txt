陆寒霆站在那里，目光落在陆茵茵的脸上，那双狭眸黑而沉，里面像晕染着可怕的风暴，阴寒冷鹜，都能将陆茵茵给吞没在这个风暴里。
<br/>
<br/>        陆茵茵不怕陆寒霆的，毕竟自己是长辈，但是这个时候她也头皮一麻，对陆寒霆生出几分畏惧感来。
<br/>
<br/>        更或者是心虚，她正要将他那些不堪的过去悉数告诉夏夕绾，他就赶到了。
<br/>
<br/>        夏夕绾转身也看到了陆寒霆，她出声道，“陆先生，你来啦，刚才我没有找到你。”
<br/>
<br/>        陆寒霆迈着稳健的步伐走了过来，他一张俊脸平静无波，就连声线都平仄的没有丝毫起伏，“陆太太，去给奶奶打个电话，告诉奶奶今晚我们回家吃饭，奶奶好几天没见你了，一定想你了。”
<br/>
<br/>        他这是要将她给支走。
<br/>
<br/>        夏夕绾怕自己走了会出事，所以她踟蹰的看着陆寒霆，用柔白的手指拽了一下他的衣袖。
<br/>
<br/>        陆寒霆将薄唇勾出了一道浅浅的弧线，“我就跟姑姑说两句话，陆太太，乖，听话。”
<br/>
<br/>        夏夕绾收回小手，“那我先去打电话，你快点过来。”
<br/>
<br/>        夏夕绾离开了。
<br/>
<br/>        现在回廊里就剩下了陆寒霆和陆茵茵这姑侄俩，这压抑的气氛透着一股诡异，陆寒霆看着陆茵茵，嗓音不动声色，“姑姑，你想跟我太太说什么？”
<br/>
<br/>        陆茵茵脸色不太好，陆寒霆是她的侄子，但是在陆寒霆强大的气场里她感觉自己就像是受审的犯人，“你不是都猜到了吗，何必明知故问，我要将你那些过去都告诉夕绾，我要让夕绾知道你是怎样一个可怕的人。”
<br/>
<br/>        陆寒霆抬脚，一步步的上前，“姑姑，你不该将我对你的仁慈当成纵容。”
<br/>
<br/>        他高大的身躯随着那迈开的优雅步调慢慢的欺近而来，陆茵茵往后退了两步，“寒霆，这件事瞒不住的，夕绾早晚会知道的。”
<br/>
<br/>        很快，陆茵茵的背抵到了墙壁。
<br/>
<br/>        视线里一黑，陆寒霆已经将她逼到了墙角里，男人狭长的眼角染上了几分猩红，薄唇溢出嗜血森然的冷笑，“陆茵茵，我叫你一声姑姑别以为自己真的是我姑姑了，如果你嘴巴闭不上的话，那我可以帮你把嘴巴闭上！”
<br/>
<br/>        陆寒霆一个结实的拳头狠狠的砸了过去。
<br/>
<br/>        当凌厉的拳风疾驰过来时，陆茵茵吓得脸色煞白，她迅速用手抱住了自己的头，尖叫了一声。
<br/>
<br/>        那个拳头没有砸到陆茵茵的身上，而是砸到了墙壁上。
<br/>
<br/>        轰一声，震耳欲聋。
<br/>
<br/>        陆茵茵惊魂未定，浑身颤抖，她亲眼见过陆寒霆发病的时候，他就是一个可怕的怪物！
<br/>
<br/>        这时一只白皙修长漂亮的大手伸了过来，一把扣住了陆寒霆的健臂，耳畔传来了一道清冷的嗓音，“哥，够了！”
<br/>
<br/>        陆子羡来了。
<br/>
<br/>        陆寒霆没有看陆子羡，只是从喉头里逼出了森然的音节，“把手拿开！”
<br/>
<br/>        “你不顾念姑姑，不顾念奶奶，不顾念整个陆家，那你能顾念一下夏夕绾吗，她傻傻的听你的话去打电话，现在估计在等你了！”
<br/>
<br/>        提到夏夕绾，陆寒霆高大英挺的身躯倏然一僵。
