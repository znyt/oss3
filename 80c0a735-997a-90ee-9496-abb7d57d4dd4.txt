　　这鲜血虹桥凝聚出来的一刻，血袍老祖的身躯落在其中，便是在他的面前，凝聚出的一条隧道。
<br/>　　这血袍老祖，毕竟也是成名已久的人物，临危不乱，如今感觉到楚风眠的实力，便是直接打算离开，保存力量。
<br/>　　踏入到这鲜血虹桥之上的血袍老祖，几乎是刹那之间，他的身躯便是即将消失。
<br/>　　“还想要逃？什么鲜血虹桥，统统都给我破！”
<br/>　　楚风眠凌空猛然一爪，他的手臂，便是化为了战龙之爪，群龙围绕在上面，猛然之间便是一爪抓了过去。
<br/>　　这血袍老祖的身躯，刚刚落入到了鲜血虹桥之外，突然之间便是一股可怕的力量，笼罩在了他的身上，令他甚至都是无法再移动一步。
<br/>　　滚滚龙威笼罩而下，这血袍老祖身上的灵力，都被死死的压制住了，他站在这鲜血虹桥上，甚至都无法再迈出一步。
<br/>　　“轰隆！”
<br/>　　楚风眠的战龙之爪落下，这鲜血虹桥在一瞬之间，都是被战龙之爪，直接握在了手中，下一刻，就在楚风眠的手心之中，被直接捏碎！
<br/>　　就在这众人不可置信的目光中，这血袍老祖凝聚出的鲜血虹桥，就在楚风眠的手心之中，被轻而易举的给捏碎。
<br/>　　而这血袍老祖，如今便是落入到了楚风眠的手心之中，一脸惨白，如何挣扎都无法从楚风眠的手中脱身。
<br/>　　“楚公子，还请手下留情。”
<br/>　　坐在王座之上的洛汐突然开口道。
<br/>　　一个吕狼，死了就死了，不过这血袍老祖，可是一尊超越生死的老祖，让这洛汐不得不开口道。
<br/>　　在她开口一刻，这洛汐也是缓缓出手，一股魔意突然笼罩下来，就连楚风眠都是一时间感觉到了无比巨大的压力。
<br/>　　这洛汐的实力，比起看起来的，还要强大得多，虽然是生死台阶第六重，炼心之境的境界，不过她的真正实力，可是要比起那古家家主，还要强大得多。
<br/>　　甚至是隐约之中，都足以是媲美那生死台阶第七重，天命之境的人物。
<br/>　　这洛汐出手，就连楚风眠都足以个暗觉到一种忌惮。
<br/>　　“既然洛汐姑娘开口了，那今天就放过你一命吧。”
<br/>　　楚风眠轻声开口道，便是手中灵力一散，将那血袍老祖的身躯，直接给放开。
<br/>　　血袍老祖急忙退后，看向楚风眠的目光之中还都是忌惮，不过他却是不敢再继续出手了。
<br/>　　“多谢楚公子了。”
<br/>　　洛汐看着楚风眠，笑着开口道。
<br/>　　“无妨，不过洛汐姑娘今日叫楚某前来，到底是所为何事？也该说清楚了吧。”
<br/>　　楚风眠嘴角露出几分冷笑，看向洛洛汐开口道。
<br/>　　这洛汐的突然前来，肯定不是没有任何的目的的，楚风眠可不相信，身为六道魔门尊主之一的洛汐，会白白过来一趟。
<br/>　　“其实小女也是在刚刚，才知道楚公子居然是来到了中域的。”
<br/>　　洛汐目光闪烁，看向楚风眠，在她的目光之中，似乎都是露出了几分惊讶。
<br/>　　楚风眠之前在归墟天之中的所做作为，她自然也是早已经是听说过的，在杀了如此之多的七大宗门弟子之后，楚风眠居然还敢来到中域。
<br/>　　这绝对是就连她都没有想象过的，毕竟是任何武者，在招惹到了七大宗门的时候，几乎都不会敢于踏足中域的。
<br/>　　毕竟七大宗门之四，可全部都是立于中域之中去，来到中域不过是自投罗网，自找麻烦。
<br/>　　不过楚风眠倒是不管这一切，他既然是想要来，就算是这中域蕴含有着无数危机，楚风眠的心中都不会有这任何忌惮，变化。
<br/>　　“我六道魔门，从新出世，这一次的威势，自然是远远无法跟数万年前的强大相比。”
<br/>　　洛汐看着楚风眠，缓缓的开口道。
<br/>　　“所以现在，正是用人之际，所以不知道楚公子，有没有想要加入我们六道魔门的想法呢？”
<br/>　　“以楚公子的资质，实力，只要是加入到了我们六道魔门之中，做个一宗尊主，不成问题，如何？”
<br/>　　“加入六道魔门？”
<br/>　　楚风眠听到这洛汐的话，倒是有些意外，没想到这洛汐今日打算，居然是想要拉拢他。
<br/>　　“那还是有些让洛汐姑娘失望了，我楚某，从没想过加入魔门。”
<br/>　　楚风眠轻描淡写的开口道，毋庸置疑的直接拒绝了。
<br/>　　六道魔门？
<br/>　　先不要说从古籍之中楚风眠所记载看到过的，他自己本身，也没有想要加入魔门的兴趣。
<br/>　　“从未想过？”
<br/>　　洛汐听到楚风眠的拒绝，似乎也没想到，楚风眠居然会拒绝的如此干脆，她的目光都是显现出几分怒意。
<br/>　　他们六道魔门，前去拉拢，就算是许多天命之境的老祖，想要加入其中，都不可能成为一宗尊主。
<br/>　　开出这样的条件来，楚风眠都好不接受，也是让她心中出现几分怒意，不过瞬息之间，洛汐的眼神之中便是恢复了过来，嘴角从新是露出了几分冷笑道。
<br/>　　“既然如此，楚公子既然不愿意加入，那我等也不会强求，若是楚公子有朝一日，想要加入我们六道魔门的话，倒是可以随时来找我，想必少主也会愿意，像是楚公子这样的弟子，加入到我们六道魔门之中来的。”
<br/>　　洛汐的这一副转变，倒是让楚风眠有些刮目相看，看来这洛汐的确是今非昔比，现在跟楚风眠撕破脸，绝不是一件好事。

<br/>　　如今化解，对双方来说，都没有任何坏处。
<br/>　　楚风眠身形一动，便是准备离开，不过正当他要离开之际，突然楚风眠停下了脚步，转头开口道。
<br/>　　“这段时间，我会在宁州古城之中，我希望洛汐姑娘，不要来打扰宁州古城，毕竟楚某，还不想要跟洛汐姑娘战场相见。”
<br/>　　“小子！你！”
<br/>　　听到楚风眠的话，站在宫殿之中的几尊武者的脸色都愤怒起来。
<br/>　　楚风眠一口拒绝也就算了，如今这话，根本就是在威胁，若是他们六道魔门，再去对付宁州古城的话，那楚风眠也就会出手，对付他们。
