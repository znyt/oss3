　　当时间到达28分钟的时候，场上的人头比已经是来到了8：6，皇族这边3个人头，叶帝他们这边则是两个人头。
<br/>　　中间对方倒是利用狮子狗这个英雄打出了几波优势，在人头方面已经领先了叶帝这边，只不过如果从场上的经济来讲，这个时候却是叶帝他们这边领先的。
<br/>　　而且最关键的是，这个时候叶帝他们已经是拿下了两条火龙，这两条火龙在后期的作用可是非常关键的。
<br/>　　而到了这个时候香蕉的武器已经是非常的恐怖了，手中已经有了三相、破败和振奋铠甲几件装备，单路带线根本就没人能够拦的住他。
<br/>　　“过来打团，香蕉你继续单带，肉装大树的伤害并不高，你可以顶着他拆塔。”看着自己这边的装备基本都已经成型了，叶帝开口指挥道。
<br/>　　这一局因为对方的紧密防守，他们这边倒是一直没有机会去偷大龙，每当他们在地图上消失的时候，对方都会立刻向着大龙赶来，这一点让叶帝几人非常的烦。
<br/>　　不过好在他们现在发育的都非常好，即使没有大龙buff也是可以继续打下去，香蕉的武器装备已经非常的好了，现在别说是大树一个人了，恐怕就算是狮子狗去了也要被他给1v2反杀的。
<br/>　　这就是到了后期的武器的强大之处，真正的单挑王，在线上根本不需任何对手。
<br/>　　看着叶帝几人的动作，皇族这边的教练也是皱起了眉头，他最担心的事情终于出现了，对方的武器装备已经成型，现在已经是打起了41的套路。
<br/>　　这个时候虽然他们这边的大嘴同样已经装备成型了，但是如果大树不过来参团的话，他们的团战还是开不起来。
<br/>　　让布隆过去开团，那么后面就没人保护大嘴了，他们这个阵容只要大嘴一死，那么基本上就直接gg了，不过有一个好消息就是，对面的白队的打野并不是刺客英雄。
<br/>　　当然了到了这个时候雪人的w技能释放到女警身上，为女警带来的攻速加成同样是令他们非常恶心的，到了这个时间段他们的大嘴起来了，同样的白队的女警也起来了，而且仗着射程远，加上输出高，女警同样是非常恐怖的存在。
<br/>　　只是眼下这个情况，让大树过来参团的话，那么只要被对方给拖住了，他们这边的下路肯定是要没的。
<br/>　　因为上路的外塔已经被带光了，此刻香蕉正在下路带线，奔着对方的下路二塔而去。
<br/>　　而也正是如此才会让皇族这边如此纠结，这个时候他们如果敢过去抓香蕉的话，不用多想，叶帝几人肯定会立刻去开大龙的。
<br/>　　有着雪人、蛇女还有女警这几个英雄在，他们打大龙的速度简直不要太快，恐怕十秒钟的时间这个大龙就要落入他们的手中了。
<br/>　　而就在皇族这边纠结的时候，众人的屏幕上已经是浮现出了武器推掉了他们下路二塔的消息，原来就在刚刚香蕉的武器带着一大波兵线进入到了对方的防御塔下，硬顶着大树的伤害直接推掉了防御塔。
<br/>　　这个时候武器的推塔速度已经是非常的快了，而等到这个防御塔被推掉的时候，香蕉的血量还保持在三分之二左右，武器对他并没有造成太多的伤害。
<br/>　　“怎么办？我守不住啊，一会这个武器就要上高地了。”看着这一幕，苹果连忙开口说道。
<br/>　　听着苹果的话，皇族教练的眉头顿时皱了起来，他没想到武器的推塔速度会这么的快，等他看向苹果的电脑屏幕的时候，此刻香蕉的血量在清理过一波小兵后已经是回满了。
<br/>　　“这么弄下去会被他们耗死的，大树的伤害太低了，连兵线都清不了。”司马贼同样一脸凝重的开口说着。
<br/>　　“大树跟团，白衣去下路守！”没有继续思索下去的时间了，眼看着武器快要推上高地了，皇族教练连忙开口说道。
<br/>　　这也是他目前为止唯一能够想出来的办法了，机械先驱这个英雄的e技能清兵速度非常的快，让楚白衣来下路清兵，只要对方的兵线进不了防御塔，那么武器就不能推塔。
<br/>　　只不过这样的话，这条大龙就是他们非常关心的事情了，少了楚白衣这么一个c位，他知道叶帝他们绝对是不会放过这个机会的。
<br/>　　“如果他们开大龙的话，找机会上，他们的武器在下路，只要保护好大嘴，这一波有机会赢。”皇族教练继续开口叮嘱着。
<br/>　　这个时候他只能期待司马贼的大嘴能够有亮眼的发挥，眼下的大嘴装备已经成型，有着大树和布隆的保护，他完全有机会创作出奇迹的。
<br/>　　看着机械先驱在下路出现，大树则是在中路出现，叶帝几人连犹豫都没有犹豫直接奔着大龙这里走去。
<br/>　　“开龙开龙，萝卜布好夹子，茄子做好视野。”叶帝连忙开口说道。
<br/>　　片刻之后，叶帝几人便是已经来到了大龙这里，而在边上此刻已经布置好了五个夹子来阻挡对方的推进。
<br/>　　来到大龙圈子后，几人直接选择开龙，蛇女的技能、雪人的技能加上女警的普攻，大龙的血量以众人肉眼可见的速度在那里下降着。
<br/>　　同一刻皇族这边的几人也是在那里缓慢的向前推进这，虽然这一波他们是要开团的，不过眼下大龙的血量还算多，这个时候他们还是打算利用大龙来消耗一下叶帝几人的。
<br/>　　当大龙的血量还剩下半血的时候，几人的速度终于是快了起来。
<br/>　　“茄子大招！”看到对方的动作，叶帝开口喊道，下一刻娜美的大招便是朝着对方几人汹涌而去。
<br/>　　眼下这种情况叶帝他们并不打算用娜美的大招来开团，现在大龙的血量已经很少了，只要拖延对方片刻，那么这一波他们就可以拿下大龙了。
<br/>　　只不过茄子的这个大招才刚刚将最前方的大树击飞起来，便是已经消失不见，这一刻布隆依然是手持护盾挡在了前方，茄子的大招直接被挡了下去。
<br/>　　虽说如此，不过茄子的这个大招还是拖延了对方片刻时间，而就在这片刻之间，大龙的血量已经是不到五分之一。
