　　第45章
<br/>　　“黄管家，黑石城一别，已经好久不见，没想到你竟会在此地落草为寇。”李甲。
<br/>　　“我不是什么管家，现在是蛤蟆寨的总管，以前的事不必提了。”黄三元。
<br/>　　“可惜了，可惜了。”李甲叹息：“怎么不见程当家……”
<br/>　　“大当家要事缠身，这种小事是不会过问的。”黄三元道：“你有什么事，和我谈也是一样。”
<br/>　　国人谈判的特点就是避实就虚，好像谁先说出目的，谁就技不如人。黄三元和李甲在一起磨，谁也不聊正题，好在黄三元做这些事情是擅长的，他也不怕和李甲磨叽。
<br/>　　李甲心底还是想见一见程大雷的，这被传得神乎其神的蛤蟆大王，也不知道真实面目如何。
<br/>　　现在只派出徐神机、黄三元与自己谈判，很显然是没把自己当会事，那蛤蟆大王果然好神秘。
<br/>　　再磨叽无数次后，终于聊到今天的正题：究竟要多少赎金拿会寿礼？
<br/>　　“薛城主愿意拿出两万贯，赎回寿礼，这笔钱却也是薛城主大恩大德，又有我的面子作保。”
<br/>　　“哈，薛城主想得真好，那批寿礼卖出去，怕是不只这个数目吧。”
<br/>　　“黄总管莫要忘了，这批寿礼是你们抢来的，本来就不属于你们。”
<br/>　　“李帮主也莫忘了，我们是山贼，抢到的东西就是我们的。”
<br/>　　“你们蛤蟆寨只有六个人，这笔钱不给，想平了蛤蟆寨也不是不行。”
<br/>　　“可我们就六个人，也抢了陆哼的一个百人队。”
<br/>　　二人你来我往，耍一些语言上的诡计，一方想抬高数目，一方想压低数目，可以说是唇枪舌战。
<br/>　　黄三元心底不屑的瞥了李甲一眼，心道：你和我聊这些，还差得远呐！
<br/>　　“那么黄总管想要多少？”李甲想要摸黄三元的底。
<br/>　　“那要看李帮主有多大权力。”黄三元的心理价位是三万贯，但以他的狡黠是不会轻易暴露出来的。
<br/>　　李甲骂了一声老狐狸，叹口气道：“黄总管如果做不了主，还是把程当家请出来吧。”
<br/>　　“我来和你谈就好。”黄三元含笑。
<br/>　　嗵！
<br/>　　屋门忽然被一脚踹开，李甲一惊看到一满脸络腮胡子的大汉，扛着一柄大斧头闯进来。
<br/>　　难道这人就是那顿顿都要吃人肉喝人血的蛤蟆大王，果然好凶恶的相貌。
<br/>　　李甲心里一激灵，难道对方想要对自己下黑手！
<br/>　　“你……你想干什么！”
<br/>　　砰！
<br/>　　“我不喜欢讨价还价，赎金要十万贯。”程大雷一斧头砸在桌面上：“你答应不答应？”
<br/>　　李甲额头的汗珠冒出来，果然是吃人肉的主儿，自己要是不答应，这一斧头就会砸在自己脑袋上。
<br/>　　“我答应，我答应……”
<br/>　　嘟，收到恐惧值199
<br/>　　嘟，收到恐惧值288
<br/>　　嘟，收到恐惧值266
<br/>　　“好了，黄三元继续和他们谈！”
<br/>　　程大雷扛着大斧头离开屋子，刚回到自己房间，他就打水把脸上的黑灰和假胡子洗掉，其实心里已经乐开了花。
<br/>　　果然有效，这一波就收到小一千的恐惧值，看来是找到正确的法子。这个样子发展下去，收集10000恐惧值任务并不难完成。
<br/>　　土屋内，程大雷离开后，李甲和两个随从才擦了擦额头汗珠。
<br/>　　黄三元眯眼笑道：“那就如此说定，你们带来十万赎金，蛤蟆寨把寿礼还会。”
<br/>　　“等一等。”李甲身边一人道，他不是野狼帮的人，确切来说是薛半川的人。
<br/>　　“怎么，想反悔？”徐神机道。
<br/>　　这个时候三人那敢反悔，脑袋不想要了。李甲和两个同伴嘀咕一番，李甲道：“问题是……我们拿不出十万贯。”
<br/>　　一贯是一千文，十万贯就是一百万枚铜钱，莫看薛半川是一城之主，他拿不出这么多现金。刚刚得到黑石城城主的位置，许多地方都要花钱，现金这种东西，他是没有多少的。而那批寿礼满打满算也就值三万多贯而已，十万贯这个数目确实是蛤蟆寨狮子大开口。
<br/>　　说出这句话，李甲心里有些打鼓，该不会触怒对方吧。
<br/>　　“没问题，我们是可以分期支付的？”
<br/>　　“分期支付？”李甲三人一头雾水，没听过这个词。
<br/>　　“我们都帮你们考虑好了。”黄三元打了个响指，拿出一副三流理财经理的态度：“没有现金，可以用粮食、盐、布抵债，只要交够首付就可以。至于利息么，以后凡是从蛤蟆岭下经过的车队，我们抽取二成利息，直到本金全部还清为止。”
<br/>　　李甲听不太懂黄三元的话，包括他身后两个薛半川派来的人。也不怪他们，要怪只能怪这个时代的教育水平太差。怎能奢望一个帮派头目听得懂什么叫等额本金，等额本息。不过李甲倒是听明白了一件事，就是只要交三成，也就是三万贯就可以取走所有寿礼。至于后面的欠债，我不还你能怎样……
<br/>　　不得不说，李甲的思维方式很简单。
<br/>　　“没有问题的话，我们就签约吧。”黄三元和徐神机取出厚厚一叠纸，一副早有准备的样子：“合约一式两份，你们签完字后再拿给薛半川签字盖章，等凑齐首付，就可以把寿礼取走了。”
<br/>　　之后，双方又谈了一些细则，比如一石粮食等于多少铜钱，比如盐铁又作价几何，具体交付时应怎么执行……
<br/>　　这些繁琐的事情聊到很晚，最后李甲捧着厚厚一叠合约离开，当离开蛤蟆寨的时候，李甲想不太明白，薛半川托付的任务，自己究竟算完成没有。
<br/>　　有些事程大雷以琢磨很长时间，山寨自然靠打劫为生，连续两场战斗，蛤蟆寨也打出了自己的名声。在很长一段时间内，蛤蟆寨不用再考虑安全问题。
<br/>　　但是涸泽而渔焚林而猎，一直打劫没人从山下过怎么办。所以还是要放长线钓大鱼，程大雷可不想把落叶城与黑石城的交通要道给切断。
<br/>　　程大雷虽然蜗居蛤蟆岭，但心里可还是牵挂着黑石城与落叶城的经济问题呢。
<br/>　　当然，这种躺着挣钱的方式，是需要实力支撑的，而现在蛤蟆寨的实力显然不够。
<br/>　　所以，发展才是硬道理。
