　　天下没有免费午餐这个道理，卡尔很清楚。
<br/>　　“卡尔总统是个聪明人，跟聪明人说话就是省力。”
<br/>　　电话那头的陌生男声说道：“GHOST和SHADOW同归于尽，一起死了，他们手中所掌握的资源也随之失踪这是一个巨大的损失。”
<br/>　　陌生男人叹了口气，随后说道：“但是GHOST留下了一个学生，就在这次在K国的那些人中。我要你把他们拦截下来，不要让他们离开K国！”
<br/>　　听到陌生男人的要求，卡尔忍不住倒吸一口冷气，这个要求太难达到了！
<br/>　　K国是一个总人口不过五千万的小国，因为它丰富的油气资源带动了K国经济，不然的话，K国不过是一个地域狭小的贫瘠之地。
<br/>　　油气资源带给了K国经济发展，但是K国自己本身的生产力却很低，很多东西都依赖进口，根本无法跟作为联合国“五大流氓”的中国相比。
<br/>　　联合国常任理事国的强大能量不是K国这种自身缺乏生产力的小国所能对抗的，作为K国总统的卡尔很清楚这一点，所以他犹豫了。
<br/>　　电话那头的陌生男人很有耐心，他并没有去催促。过了好一会儿，卡尔才开口说道：“先生，你的要求太困难了。”
<br/>　　“你确定要拒绝我的提议吗？”陌生男人很肯定的说道：“你不会甘愿放弃你现在的位置，还有目前所掌握的权利，但是如果没有我的帮助，你肯定会遭到弹劾，你肯定不能继续当你的总统！看着你的对手上位，然后把你踩在脚底下，你甘心吗？”
<br/>　　卡尔嗫嚅着没有回答，他很纠结，也很犹豫，如果他真的把赵天宇他们这些人拦截在K国不让他们离开的话，K国势必会承受来自一个超级大国的怒火。这股怒火未必会是军事上直接开战，更多的会是经济领域的多项合作，而这会对K国经济发展所造成的影响不亚于一场战争！
<br/>　　所以卡尔不敢答应，他需要承诺，对方的一个承诺！
<br/>　　“你能为我保证什么？”卡尔缓缓开口说道：“除了我现在的总统之位，你还能为我保证什么，那样做，我要承受的风险很大。如果真的引来中方的怒火，造成对K国经济领域的冲击，针对我的弹劾案还是会被摆上国会议桌，我还是会遭到弹劾，到那个时候，我的下场会比现在遭到弹劾更惨，到那时，我就是想要一个平凡生活都是奢望！”
<br/>　　“我代表洛克希德公司。”
<br/>　　电话中，陌生男人缓缓亮明身份，说道：“以后我们会在经济、军事领域有更好的发展，我们会在K国建立洛克希德的分公司，建立生产线，让K国的军备不再仅仅只是依靠进口。只有自己掌握生产力，那才是真的强大，不是吗？”
<br/>　　洛克希德公司！
<br/>　　卡尔倒吸一口冷气。他太清楚这个公司代表着什么了，美国五大军火巨头之一，现在是洛克希德公司的代表给他电话寻求合作，那么这是不是也代表着洛克希德公司背后美国政府的意见？如果真是这样，那卡尔就要好好权衡了，如果拒绝，就因为着K国会得罪美国这个庞然大物，一个能够绕开联合国安理会直接对他国开战的流氓，K国怎么得罪的起？又如何得罪的起？
<br/>　　“这是你身后‘老板’的意思吗？”卡尔问道。这是他目前急需要弄清楚的事情，也是会影响到他决定的事情。
<br/>　　“我没有这么说。”陌生男人回答道：“但是你可以这么去理解。毕竟现在局势微妙，就算是‘老板’也有他的顾虑。”
<br/>　　陌生男人这句话虽然没有明说，但是字里话间却是透露给卡尔一个信息，陌生男人只是帮人传话，背后的人不愿意露面，到底该如何选择就看卡尔自己怎么琢磨。
<br/>　　良久，卡尔终于做出了选择，他答应了电话中陌生男人的要求；同时，陌生男人也给了卡尔相应的承诺，第一份合作合同发到了卡尔的私人电子邮箱中。
<br/>　　这通电话虽然双方没有做过多的交谈，但是却打了足足有十五分钟，打完电话，卡尔只觉得自己浑身的力量都被抽空了一样，就连他握在手里的电话听筒都有千斤重，让他险些拿不稳。
<br/>　　卡尔放下电话，用手搓了搓脸，静静的做了好一会儿，重新拿起电话，拨通了一个号码，问道：“赵将军他们什么时候离开？”
<br/>　　“明天中午的飞机。”
<br/>　　“将他们拦下来，暂时不要让他们离开！”卡尔语气冰冷的说道，将总统的威势毫无保留的释放了出来！
<br/>　　“可是这样的话，会不会……”
<br/>　　“按照我说的去做！”
<br/>　　卡尔没等电话那头说完，就打断了他的话，然后直接挂断了电话啊。
<br/>　　打完电话，卡尔靠在椅子上，重重的叹了口气，他仿佛看到了在自己头顶上悬着一柄利剑，只要这柄利剑落下，就能将他轻而易举的劈成两半。
<br/>　　而他屁股下这张用犀牛皮制作的椅子此时也不再像往日那样的舒适，让他有种恨不得立刻站起身来逃离的冲动，如坐针毡，或许就是现在对卡尔而言最好的形容。
<br/>　　“但愿，我的选择不是错的。”
<br/>　　卡尔看着天花板，轻声呢喃，这是一步险棋。正所谓一步天堂，一步地狱，究竟会如何卡尔决定不了，只能看他身后“老板”的手腕！
<br/>　　……
<br/>　　第二天上午，赵天宇再次来到了医院。
<br/>　　云熙挂了一晚上的点滴，高烧是退了，但是人还没醒，并且不时会说梦话，她还没有从那个梦境中挣脱出来，但愿回去后，云岳和家里那两个小的，能让她尽快好起来。
<br/>　　赵天宇联系医院方面，要了一辆救护车帮忙把云熙送去机场；可是救护车还没离开停车场，几辆黑色轿车突然闯进停车场，将正欲离开停车场的救护车给拦了下来。
<br/>　　为首的那辆黑色轿车上走下来一个穿着黑色西装，戴着墨镜的男子，走到赵天宇面前，给他出示了调查令，说道：“赵将军，很遗憾你们现在暂时不能离开，有些事情需要你们配合调查！”
