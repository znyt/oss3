<br/>				第二日，<br/>　　<br/>　　随着叶尘一声令下，众人正式动身返回华夏。<br/>　　<br/>　　这一次岛国之行，他先是将杉口组连根拔起，而后逼退第七自卫队，再然后斩杀羽鹤大仙，更逼迫岛国相低头服软！<br/>　　<br/>　　如今残卷既然已经得手，此行也算是功德圆满了，自然没有继续待下去的必要。<br/>　　<br/>　　众人来到机场，欧阳寒为了避免被岛国政府追踪，早已经为叶尘等人分别办了一个假身份。<br/>　　<br/>　　不过，欧阳寒五人要返回京都述职，叶尘则准备直接返回天海，所以并没有同行。<br/>　　<br/>　　叶尘带着小林舞依、雪姬以及江口樱子三人，通过安检之后，直接登上飞机头等舱。<br/>　　<br/>　　江口樱子依旧对叶尘十分依赖，始终死死的搂着叶尘的胳膊，上了飞机之后，便挨着叶尘坐了下来，依偎在他的肩头，在外人看来，两人犹如恋人一般。<br/>　　<br/>　　这让叶尘也很无奈，面对一个智力只有七八岁的大美女，打不得，骂不得，让他堂堂狂帝，竟然也有些束手无策。<br/>　　<br/>　　而一旁的小林舞依见状，竟然也不甘示弱，也在叶尘旁边坐下，然后悄悄依偎在叶尘的另外一个肩头。<br/>　　<br/>　　唯独雪姬孤零零的坐在一旁，看上去好像被三人冷落了一般。<br/>　　<br/>　　不一会，旁边座位又上来了几个人，似乎是一起的。<br/>　　<br/>　　为的是一名器宇不凡、相貌颇为英俊的青年男子，一看就知道不是寻常人物，身后还跟着一名气势很强的老者，看样子应该是一名武道高手，后面还有四名身材壮硕的中年男子，应该是他保镖。<br/>　　<br/>　　那青年男子在旁边座位上坐下之后，很快便注意到了叶尘等人，当目光在江口樱子三人身上扫过的时候，顿时两眼一亮，继而眼眸中透着浓浓的嫉妒之色，<br/>　　<br/>　　“御姐！萝莉！女神！而且个个都是极品，这小子还真是艳福不浅啊！”<br/>　　<br/>　　小声感叹了一句之后，青年男子两眼微微一转，像是在打什么主意，然后起身来到叶尘对面坐了下来，笑吟吟道：<br/>　　<br/>　　“这位朋友，认识一下！我叫郑同宇，天海人，我爸爸是……”<br/>　　<br/>　　青年男子的话还没有说完，叶尘连眼皮都没有抬一下，直接冷冷道：<br/>　　<br/>　　“滚！”<br/>　　<br/>　　青年男子先是一愣，顿时脸色狂变，<br/>　　<br/>　　就连青年男子的那帮手下，听到叶尘竟然敢对他的主子如此无礼，也顿时一个个全都霍然站起身来，冲着叶尘等人怒目而视。<br/>　　<br/>　　“小子！你知道我是谁吗？竟然敢和我这么说话？”<br/>　　<br/>　　青年男子此刻早已经没有了最初的优雅之态，已经被叶尘的态度气得险些暴走。<br/>　　<br/>　　他父亲在天南省位高权重，他的家族更是在整个华夏都排的上号，因而自小到大，他都是在众星捧月中长大，还从来没有人胆敢和他这么说话！<br/>　　<br/>　　然而，叶尘依旧连眼皮都没有抬一下，冷冷道：<br/>　　<br/>　　“我不管你是谁，收起你那点小心思！再敢和我废话，信不信我把你从飞机上丢下去！”<br/>　　<br/>　　“你敢……”<br/>　　<br/>　　青年男子顿时暴怒，忍不住就想要朝叶尘扑上去，却被一旁眼明手快的那个老者，直接一把拉住，然后附在他耳边，轻声道：<br/>　　<br/>　　“公子，不可鲁莽！此人很不简单，连我都无法看透他的修为！另外三个女人，应该也都不是寻常之人，我们现在毕竟是在国外，还是低调点的好……”<br/>　　<br/>　　听到老者这么说，那青年男子这才一脸不甘的退到一旁。<br/>　　<br/>　　叶尘却从始至终连看都没有看他一眼，如果方才不是那老者拦着，那青年男子胆敢冲上来，此刻已经是一个死人了！<br/>　　<br/>　　开玩笑，连岛国相都被叶尘逼的低头服软，在岛国杀个把人，而且还是华夏人，谁敢多说什么？<br/>　　<br/>　　不一会，飞机正式起飞，<br/>　　<br/>　　那青年男子显然一直心有不甘，时不时的望向叶尘等人，眼眸中透着浓浓的恨意和嫉妒之色。<br/>　　<br/>　　很快，青年男子郑同宇的目光，转移到一旁的雪姬身上。<br/>　　<br/>　　他现，这个犹如仙子般清纯冰冷的大美女，似乎和另外三人不是一起的，又或者，是被另外三人冷落，孤零零的坐在一旁，这样一个风华绝代的大美女，如此孤单的模样，让人不由自主的就生出了几分保护欲。<br/>　　<br/>　　一念及此，郑同宇又起身来到雪姬旁边的座位上坐了下来，然后脸上露出一抹自认为很迷人的和善笑容，缓缓道：<br/>　　<br/>　　“这位美丽的小姐，我看你应该是岛国人吧？不知道你去天海市做什么？莫非是天海大学的留学生？”<br/>　　<br/>　　雪姬一脸冷漠，一言不，似乎根本没有听到郑同宇在说什么。<br/>　　<br/>　　郑同宇还以为雪姬听不懂华夏语，当即又用岛国语讲了一遍。<br/>　　<br/>　　可惜，雪姬依旧默然，甚至连眼皮都没有抬一下。<br/>　　<br/>　　郑同宇顿时感觉面子上有挂不住了，他自问自己也算是一个大帅哥，而且家世了得，向来都是大把大把的女人投怀送抱，如今好不容易主动出击一次，竟然被对方直接无视，这让他如何能忍？<br/>　　<br/>　　“美女，相逢即是有缘！不如交个朋友可好？”<br/>　　<br/>　　雪姬继续一言不。<br/>　　<br/>　　郑同宇用华夏语和岛国语，一连几次试探之后，见雪姬始终一言不，顿时感觉有些颜面扫地，继续喋喋不休道：<br/>　　<br/>　　“美女，我叫郑同宇，我爸爸是郑光明，他可是天南省的二把手！以后你在华夏遇到任何问题，都可以尽管来找我，我保证没人敢招惹你！”<br/>　　<br/>　　被雪姬的冷漠态度刺激到有些气急败坏的郑同宇，直接报出了自己的家底。<br/>　　<br/>　　他本以为，对方听到自己的家世背景之后，肯定会有所反应，肯定不可能再无动于衷。<br/>　　<br/>　　然而很快，郑同宇再次失望了！<br/>　　<br/>　　雪姬依旧犹如未闻，仿佛一尊石像般端坐在那里，丝毫没有任何反应。<br/>　　<br/>　　“莫非是个聋哑人？或者连眼睛也是瞎的吗？”<br/>　　<br/>　　郑同宇见雪姬始终没有任何反应，不由得小声嘀咕了一句，两手还特意在她的眼前晃了晃。<br/>　　<br/>　　除此之外，他实在早不到其他更好的理由解释。<br/>　　<br/>　　然而，<br/>　　<br/>　　他这话一出口，一直如同冰雕般的雪姬，终于有了反应，不过却是转头望向一旁的叶尘，眼眸中透着杀气，语气冰冷道：<br/>　　<br/>　　“主人，我请求杀了这个人！”<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
