　　生活的富足让越来越多人走出国门去旅游，而国家的强大，也让得出境旅游的游客在安全上有了最大化的保障。
<br/>　　说实话，慕容飞心中还是担忧的，可担忧归担忧，他还是决定尊重佟千雪自己的意见。
<br/>　　“你想去T国？”慕容飞对佟千雪问道。
<br/>　　佟千雪轻咬着嘴唇，犹豫了一下，最后还是点了点头，说道：“其实我一直想去T国旅行，本来这是我明年的旅行计划，可是现在因为他（她）……”
<br/>　　佟千雪用手指轻轻戳了戳自己尚未隆起的肚子，有些苦恼的说道：“彻底打乱了我原来的计划。再过段时间肚子凸起来，行动就不方便了，等到这个小的出来就更没有时间了，这次正好姐姐学校里有组织，又正好多了一个名额，我就想跟着一起去；不过，我还是尊重你的意见，如果你不答应，我就不去了。”
<br/>　　“你都这么说了，我还能不答应吗？”慕容飞说道：“反正也就只有一个礼拜，出去散散心也好，被怀孕得了抑郁症。”
<br/>　　“谢谢你。”
<br/>　　佟千雪笑得很开心。
<br/>　　“那我就把小雪的护照拿去办签证了。”佟千莹站起身来对慕容飞说道：“既然你来了，我这个姐姐就不在这里当电灯泡了。夫妻小别胜新婚，但是也要注意，小雪现在还不满三个月，什么事情能做，什么事情不能做，你们自己清楚。小雪你自己是医生，这方面也不用我多提醒你。”
<br/>　　尽管是自己的亲姐姐，可是涉及到夫妻间的私密之事，佟千雪还是被闹了一个大红脸，赶紧起身去找自己的护照去了。
<br/>　　佟千莹拿了佟千雪的护照后就离开了，慕容飞回来一次不容易，佟千莹可不想做那个打扰他们夫妻团聚的“恶人”。
<br/>　　看着佟千莹走进电梯后，慕容飞从背后一把抱住佟千雪，在佟千雪的娇嗔和拍打中，一起窝进沙发中。刚坐下，慕容飞就直接对着佟千雪“啃”了下去，一直吻到佟千雪缓不过气来，在她推搡中才把她放开。
<br/>　　一通热吻，让佟千雪双唇粉嘟嘟的十分可爱，也十分诱人，就当罗昊还想着要再次靠近的时候，佟千雪却是一把推开了他，手指着他警告道：“挑起火来了，你自己去浴室解决，别指望我帮忙。今天你妈和我妈都不会来，你有足够的时间，不用担心有人会来打扰你。”
<br/>　　佟千雪的话立刻就让慕容飞又退了回去，的确，现在一时占了便宜，等到被挑起火来了，慕容飞估计就只能去冲凉水澡了。
<br/>　　“吃饭了吗？”佟千雪对慕容飞问道。
<br/>　　“没呢。”
<br/>　　“你说你这么大一个人了，怎么还跟个孩子一样，吃饭还要人盯着？我外甥都比懂事。”佟千雪用手指戳着慕容飞，说道。
<br/>　　“吃饭也要钱啊，我身上一分钱都没有，怎么吃饭？”慕容飞说得理直气壮。
<br/>　　佟千雪白了慕容飞一眼，站起身来，走向厨房，不一会儿就给慕容飞端了一碗饭出来，米饭上各种菜堆得跟小山似得。
<br/>　　慕容飞消灭食物的速度超出了佟千雪的想象，很快一大碗饭菜就被消灭的一干二净。
<br/>　　“你这是多久没吃饭了？”佟千雪皱眉道：“跟饿死鬼投胎似得。锅里还有，还要不要？”
<br/>　　“饭是吃饱了，再吃就要吃你了。”慕容飞眼神不善的看着佟千雪。
<br/>　　佟千雪轻笑一声，露出一个妩媚的笑容，还对着慕容飞眨了下眼睛，细声说道：“你舍得下手，那就来吧。女人三十如狼四十如虎，我正好是如狼似虎的年纪，求之不得。”
<br/>　　慕容飞抓起沙发上的抱枕，把头埋在里面，他现在真想抽自己两下，让自己嘴贱，自己怎么可能玩的过佟千雪嘛，纯粹就是找虐！
<br/>　　下午的时候，佟千雪拉着慕容飞陪自己看军事电影，佟千雪是看的津津有味，但是以慕容飞这个专业人士的角度去看，里面的作战行动实在太假了。
<br/>　　特种部队的狙击手都是管杀不管埋，哪儿有那么多顾忌，瞄准了半天都没有开枪；还有那个负责整个行动的指挥官也是个不合格货，作为一个特战指挥官一切以任务为第一优先，进入任务区域就是为了任务，完成任务就是果断撤退，哪来那么多事情，还把整支队伍搭在里面去救难民，特种部队不是红十字会，不是联合国维和部队，是绝对不会去做行动目标以外的事情，尤其是以整支特战小队作为代价。
<br/>　　“真是的，太没劲了。”佟千雪抓起抱枕扔在慕容飞身上，说道：“本来看的好好的，你倒好，这不专业那业余的，一部这么精彩的电影就被你说得一无是处。”
<br/>　　“本来就很业余嘛。”慕容飞撇嘴道：“这种电影也就骗骗你们这些小姑娘，对我来说这东西简直就跟催眠曲没任何区别。”
<br/>　　“不看了，不看了！”佟千雪拿起遥控器关掉了电视，嘀咕着说道：“既然你那么专业，那你倒是说说你的故事啊。”
<br/>　　话刚出口，佟千雪意识到自己的话欠缺考虑，以慕容飞所在部队保密性，他们的行动内容怎么可能跟故事一样随意拿出来说呢？
<br/>　　看到慕容飞陷入思考中，佟千雪立刻开口道：“如果不方便就不要说了，你就当我刚才胡言乱语好了。”
<br/>　　“龙隐部队的事情不能说，龙隐部队的行动内容绝大多数都是无期限的，最短也是几十年。”
<br/>　　慕容飞抱着佟千雪说道：“不过在我没进入龙隐部队之前的事情，我倒是可以跟你讲讲，虽然不如龙隐部队中的行动那么刺激，但是也比这种业余电影要精彩。”
<br/>　　慕容飞将自己在侦察营服役时，一次在边境配合武警部队的行动告诉给佟千雪听，这是慕容飞第一次正式行动，也是慕容飞第一次开枪杀人，一般来说，对于第一次军事行动都会刻意的去遗忘，但是慕容飞却还是在行动结束后，偷偷把行动内容记录了下来，因为对于慕容飞来说，是从大学生到职业军人的重要转折处。
