　　五彩神蛤顿时大怒，
<br/>　　“人类小子！你不要得寸进尺！我都已经不追究你的罪过了，你还想怎么样？”
<br/>　　叶尘淡淡一笑，
<br/>　　“五彩神蛤，你先不要激动嘛，让我离开也行，只不过外面还有药王宗的人，我们若是这么就出去，难免会被他们围攻！”
<br/>　　五彩神蛤肚子咕咕作响，没好气道：
<br/>　　“那是你的事情，这和我有什么关系？”
<br/>　　叶尘笑道：
<br/>　　“你先别紧张，我其实就是想向你借用一下避毒丹而已！”
<br/>　　根据古籍中记载，五彩神蛤自幼体内便有一枚妖丹，名为避毒丹。
<br/>　　而五彩神蛤生性喜欢吃毒，也正是以为这避毒丹的功效，所以才能百毒不侵。
<br/>　　常人只要得到这避毒丹，便可以抵御各种毒物。
<br/>　　而对于五彩神蛤而言，失去了避毒丹之后，只要休养一段时间，很快体内便会再次生出一颗避毒丹出来，所以对于它来说，这东西并不算多珍贵，只不过却有些羞辱。
<br/>　　叶尘虽然不惧药王宗的众高手，可是对方擅长用毒，苏蔓难免会再次中招。
<br/>　　纵然他手里还有四颗扶摇仙果，可是扶摇仙果的能量太强，苏蔓已经不可能再继续服用，而且拿极品仙果来解毒，未免太浪费了些。
<br/>　　当然了，除此之外，其实还有一个更为重要的原因。
<br/>　　倘若他现在带着苏蔓离开万毒窟，难保这五彩神蛤，不会秋后算账，和药王宗的一起，联手对付他们。
<br/>　　而只要五彩神蛤没有了避毒丹，便不敢再对他轻易出手。
<br/>　　叶尘这么做，也是为了彻底消除后顾之忧。
<br/>　　……
<br/>　　五彩神蛤听完叶尘的话，顿时气得不行，两眼都要喷出火来了，
<br/>　　“人类小子！我对你一再忍让，你可不要以为我怕了你！想要我的避毒丹，这绝对不可能！”
<br/>　　叶尘两眼微微一眯，
<br/>　　“是吗？既然是这样的话，那就没什么好谈的了，不如我先毁了你这颗仙树如何？”
<br/>　　说着，叶尘举起紫琼剑，便冲着扶摇仙树，作势欲斩。
<br/>　　五彩神蛤果然急了，
<br/>　　“别别！有事好商量，不要毁我仙树！”
<br/>　　叶尘冷冷一笑，
<br/>　　“五彩神蛤，这事没得商量，我只要你的避毒丹，只要你交出避毒丹，我不但离开就走，而且保证以后，再也不会踏足万毒窟，如何？”
<br/>　　五彩神蛤听到这话，两只巨眼飞速转动，显然是在思量其中利弊。
<br/>　　最终，
<br/>　　五彩神蛤经过一番考虑之后，两眼死死盯着叶尘，
<br/>　　“你说话算话？”
<br/>　　叶尘傲然道：
<br/>　　“我叶狂仙一言九鼎，说到自然就会做到，你若是不信，我这就立下灵魂誓言！”
<br/>　　所谓灵魂誓言，是修真界之中，比较常见的一种灵魂咒术。
<br/>　　立誓者以自己的灵魂起誓，一旦违背誓言，灵魂就会遭到反噬，轻则灵魂重创，重则直接毙命，十分的恐怖。
<br/>　　五彩神蛤听到这话，那一双大眼之中，露出一抹意外之色，
<br/>　　“人类小娃娃，想不到你年纪轻轻，竟然还知道灵魂誓言！”
<br/>　　“好！只要你立下灵魂誓言，我便给你避毒丹！”
<br/>　　叶尘也知道，自然如果不做点牺牲，五彩神蛤肯定不会轻易交出避毒丹，当即缓缓闭上眼睛，口中念念有词，似乎是一道奇怪的咒语。
<br/>　　很快，
<br/>　　叶尘立下灵魂誓言之后，五彩神蛤身上的杀气也缓缓退去。
<br/>　　“呱呱~”
<br/>　　五彩神蛤忽然扬天叫了两声，继而张开血盆大口。
<br/>　　嗖！
<br/>　　自五彩神蛤的体内，竟然飞出一枚，散发着金芒、宛如夜明珠一般的珠子。
<br/>　　那金珠自五彩神蛤的体内飞出之后，便迅速飞到叶尘跟前，
<br/>　　“这便是避毒丹吗？果然不凡！”
<br/>　　叶尘虚手一抓，便将避毒丹吸入掌心，然后递给一旁的苏蔓，
<br/>　　“蔓姐，吃了它！”
<br/>　　苏蔓顿时两眼一瞪，
<br/>　　“什么？你让我吃这个怪物吐出来的……呕~~”
<br/>　　苏蔓又望了一眼相貌无比恶心的五彩神蛤，一想到这东西是从它的口中吐出来的，顿时忍不住干呕起来，继而连连摆手，
<br/>　　“我才不要！打死我也不吃！”
<br/>　　叶尘不由得一阵摇头叹息，
<br/>　　这避毒丹，哪怕是在修真界，也是最顶级宝贝！
<br/>　　当初在前世，为了对付了一个擅长用毒的门派，叶尘就曾经一直想要得到此物，可是却始终未能找到五彩神蛤的踪迹。
<br/>　　后来他修为大进，成为了狂帝之后，避毒丹对他已经可有可无，他这才放弃寻找此物。
<br/>　　“也罢，你若是不吃的话，就先把它带在身上，一样可以避毒！”
<br/>　　苏蔓还真害怕叶尘逼她吃下此物，听到这话，顿时忙不迭的点头，双手接过避毒丹。
<br/>　　“人类小子！避毒丹我已经给你了，你现在应该遵守你的誓言了吧？”
<br/>　　五彩神蛤口吐人言的同时，庞大的身躯也开始缓缓变小，然后退到了一旁，给叶尘两人让出了一条通道。
<br/>　　叶尘哈哈一笑，
<br/>　　“那当然！五彩神蛤，咱们后会有期！”
<br/>　　说着，叶尘一手拦住苏蔓的芊芊细腰，身体一晃，便已经从五彩神蛤的跟前穿过，直奔外面而去。
<br/>　　很快，后方传来了五彩神蛤充满愤怒的咆哮之声，
<br/>　　“卑鄙的人类小子！不要让我再见到你，否则我一定不会放过你的！”
<br/>　　叶尘再次哈哈一笑，根本没有把五彩神蛤的话放在心上，心情大好。
<br/>　　他也没有想到，这次误打误撞闯入万毒窟，不但给苏蔓解了毒，而且还得到了扶摇仙果这等极品灵物，外加避毒丹和一堆的灵花灵草。
<br/>　　他这一次，可谓是因祸得福，收获颇丰了。
<br/>　　甚至，
<br/>　　因为之前服用了半颗扶摇仙果，叶尘隐隐觉得，自己许久没有进步的真元，竟然隐隐出现了突破的征兆！
<br/>　　不过眼下，显然还不是突破的时候。
<br/>　　“药王宗！接下来，也该来算一算我们之间的账了！”
<br/>　　口中喃喃自语了一阵，叶尘两眼之中，露出一抹浓浓的寒意！
<br/>　　片刻之后，
<br/>　　轰！
<br/>　　叶尘揽着苏蔓，冲出了万毒窟，
<br/>　　然而，
<br/>　　出乎叶尘意料之外的是，此时万毒窟之外，竟然空无一人！
<br/>　　“难道他们觉得我必死无疑，所以干脆就不管我了？”
<br/>　　叶尘暗暗纳闷，
<br/>　　轰隆隆！
<br/>　　这时，
<br/>　　远处却传来一道巨响，似乎是药王宗中，出了什么大事。
