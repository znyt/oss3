　　“嗯？这是……”
<br/>　　玉虚宫中，一直在闭关之中的原始圣主突然睁开了眼睛，眼中闪过一丝莫名之色，紧接着，他再仔细感受了一遍，才发现自己并没有感应错。这才惊声道：
<br/>　　“这怎么可能？”
<br/>　　原始圣主声音之中带着浓浓的惊异，满脸不可思议。原始自从成圣后，便一直在闭关之中，炼化天道本源空间中的天道法则。天道空间寂静无声，一直如此。
<br/>　　但，就在共工撞断不周山的那一刻，天道突然之间变得衰弱下来，这就好似一个凶悍的大汉一下子虚弱得如同大病一场的病秧子。这让原始觉得不可思议，但紧接着，他面上则露出狂喜之态，因为他发现，这一刻天道空间中的法则变得前所未有的虚弱，炼化速度快了十倍不止。
<br/>　　原始不再去管其他，赶紧抓紧时间炼化自己熟悉的天道法则，他知道，这种状态不可持久，天道迟早会恢复过来，若不抓紧时机多炼化一些法则，那就太对不起自己了。
<br/>　　同时，他心中也有些隐隐猜测到了，不周山为何会被共工撞断了。
<br/>　　他之前就一直在疑惑，不周山是何等坚固，那可是盘古大神脊梁所化，就算经过无尽岁月的洗刷，变得不再如之前那般坚固了，但这也不是区区一个祖巫便能撞断的，如今看来，这其中果然有鬼。
<br/>　　天柱不周乃盘古意志最浓的地方，天柱不断，天道又怎会变得虚弱下来？天道不虚弱，别人又怎会有机会炼化其中的天道法则？
<br/>　　至于这其中的幕后黑手，原始除了想到自己的便宜师傅道祖鸿钧之外，便再无其他人选了。毕竟，鸿钧给他的印象，一直都是高深莫测。
<br/>　　当初天道不全，他以身补天道。当时原始便觉得不对劲了，对于一个修道者来说，追求的都是大道超脱，越是强大的人，这种信念便会越强。鸿钧又怎会无缘无故去以身补天道呢？原始才不会相信他是真正的大公无私，这其中必然有什么他不知道的秘密。
<br/>　　果然，如今天道出现这种虚弱情况，也许就是他计划中的一环，毕竟，鸿钧以身补天道，他自己就成了天道的一部分，对天道的感悟，其他人自然是比不上他了。
<br/>　　如今看来，鸿钧道祖的目的一直是洪荒天道。如今天道虚弱，也定是他法杰作，而原始只是他计划之中的一个变数而已，毕竟当初没有人能想得到，原始能够以力证道，突破天道空间，炼化天道法则。
<br/>　　此刻感受到天道的虚弱，原始不再去想其他，他才不想去管鸿钧的计划，而且就算他想去管，也管不了。洪荒之中，一切都是虚的，唯有实力最真实。眼下有如此提升实力的良机，他又怎会去多管闲事？能多增强一分实力，就多增强一分，想其他的干什么。
<br/>　　他不再多想，赶紧将自己的圣人意识引入天道空间中，捡着自己熟悉的天道法则，赶紧炼化。为什么要选自己熟悉的，那自然是因为熟悉的法则是他自己已经领悟的法则，这样的天道法则炼化有个过程，会更加容易。
<br/>　　至于不熟悉的，还需要重新领悟，而且还要防止天道意识的反噬，炼化难度大增。若是平时还好，在如今这种争分夺秒的时刻，那是无比愚蠢的做法。
<br/>　　……
<br/>　　在一处无垠无边的空间之中，没有天，没有地，亦无光明与黑暗之说，奇妙的大道韵律回荡在空间中，弥散整片空间之中，如同滚滚大道。
<br/>　　一个身着玄色道袍的身影，此刻正盘坐在中央，一处道台之上，冷眼向四方看去。
<br/>　　他身上散发着无穷无尽的道韵，仿佛，他，就是苍穹，他，便是天地意志，……他，便是，道！
<br/>　　这个老者神魂玄奥，闪烁着八道彩芒，正是鸿钧道祖之魂。
<br/>　　而此处，自然也是天道本源空间之中，鸿钧以身化天，成为天道的一部分，如今，他正在吞噬天道。
<br/>　　“嗡！”“嗡！”“嗡！”……
<br/>　　道韵如波，缓缓回荡飘散着，而在这看似空旷的空间中，却有着整整三千个庞大的旋涡，好似无时无刻不在旋转，奇妙无比。每道旋涡中央，都有一道法则细线延伸而出，没入虚无之中，连通到紫霄宫中的造化玉蝶之处，供应到鸿钧身体之中，他以天道法则炼体。
<br/>　　一切看来好似完美，但美中不足的却是，靠近鸿钧道祖最近，前方一丈方位，有两道旋涡已经停止了旋转，静止不动，仿佛道韵干枯，道则细线被人早已抽走一般。
<br/>　　这两个旋涡所代表的法则细线，却正是因果法则与天罚法则，乃原始成圣的两道法则。
<br/>　　而此时，道祖还能够感受到，另外又有一道旋涡中的法则细线有被抽动炼化的迹象，虽然这种迹象非常轻微，但他还是发现了。
<br/>　　“哼！”
<br/>　　鸿钧睁开眼睛，眼中缭绕神魂的八道彩芒立刻更盛了，看着眼前停止旋转的天道旋涡，鸿钧立刻冷喝一声：
<br/>　　“元始，果然厉害，吾之前便一直没能看透你，想不到你竟能在没有成圣之前便寻得天道本源空间，以力破开空间，烙印天道法则本源，以力成圣。如今却要便宜你，混账！”
<br/>　　“不过，时间短暂，吾还是赶在天道恢复之前先争取多掌握一些天道，以后才能与天道意志抗衡，谋划更多。至于元始……哼！先让他折腾一会儿，你以为以力成圣就没事吗？只要你用过鸿蒙紫气，便不能超出贫道的掌控！哼！”
<br/>　　言及于此，鸿钧脸色变得有些发冷，目中好似有寒芒闪烁。
<br/>　　“嗡！”“嗡！”“嗡！”……
<br/>　　鸿钧这般想着，面色正难看之际，陡然间，天道本源空间嗡嗡作响，下一瞬，空间里，天道意志一下子虚弱下来，道之气息开始变得萎靡起来。
<br/>　　此刻，正是外界，洪荒不周山被共工撞断之际，这对天道产生了影响。
<br/>　　“嗯？”
<br/>　　鸿钧神魂双目一凝，陡然一声大喝：
<br/>　　“给贫道炼化！”
<br/>　　一声大喝回荡，传遍整个空间，鸿钧身躯一震，头顶之上，造化玉牒在缓缓旋转，释放出造化玄光，玄之又玄，牵引着三千天道本源，配合着鸿钧的炼化。
<br/>　　“很好，不周山已断，盘古意志消散，天道意志虚弱，贫道的计划完成一大半，虽有变数，但一切顺利进行，天道法则淬炼贫道之身躯，定能让吾肉身再破极限，到时贫道变更突破混元后期，向着混元圆满进发，至于之后的事，那就容易许多了，若是完全炼化盘古天道本源，贫道定能突破天道境！”
<br/>　　鸿钧神魂微微一笑，眼中好似闪过一丝满意一般，沉声开口说道。
<br/>　　对于自己的“吞天”计划的进程，道祖鸿钧颇为欢喜，认为大道就在眼前，心中大定！
<br/>　　毕竟，在他看来，已经快要成为混元圆满的他，洪荒之中已经没有谁能威胁他了，除非是盘古复生。但，盘古早已身化洪荒，他怎么可能还活着？
<br/>　　至于洪荒六圣，除了元始能够给他一点意外之外，其他人他都没有放在心上。
<br/>　　……
