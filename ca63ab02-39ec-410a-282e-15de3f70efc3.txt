　　哗——
<br/>　　黑与白的相撞光芒四射，像是最原始的较量，在中心划出一道分界线，颜色分明。
<br/>　　尚九娣和陆玟丞被这刺目的光线所逼的双双退后，无法在这片空地上持续观战，在这快速发生的变数中也根本无法看清前方发生了什么，只知道又一恐怖如斯的人物，大驾光临。
<br/>　　楚涵眯起眼睛，低头看了眼紧握修罗战斧，却在微微颤抖的右手。
<br/>　　月影斩很强势，不需要依靠他太过于消耗精神的能量控制，便能自主形成攻击角度，所有能量的聚集和散开，都与这个世界相融合，浑然天成。
<br/>　　这让楚涵明白一味的模仿白允儿的手段并不明智，每个人都有自己最合适的方向，而月影斩，才是他最贴切的招数！
<br/>　　与爆炎斩一样，强势的月影斩所需消耗的能量同样不小，每一次的出手都是楚涵体内本源能量的消耗，再在瞬发出击中与四周的能量结合，从而逐渐壮大或递减。
<br/>　　所以月影斩同样需要看环境，这也就是为何当时在木屋内的月影斩很小，只是通过出其不意斩下了丧尸王的几根手指，在室外的月影斩却强大到一刀将树林都劈成两半的原因。
<br/>　　只是再如何强大，都有一个极限，已经在短时间内多次使用月影斩的楚涵，当下能量耗竭，本以为能通过这最后一击将丧尸王解决，却没想到突生异变。
<br/>　　前方的月影斩威力还未消散，正在与那道白昼般的能量较量，双方僵持不下在时间的流逝中渐渐衰弱。
<br/>　　光线逐渐递减中，那道刺目的光辉不在，也将当下了月影斩的身影显露而出。
<br/>　　不是刚才那个扔下亲弟弟先走的异种王木叶，也并非楚涵见过的人，更不是这末世纪元里任何出名的高战力新人类。
<br/>　　而是一个坐在轮椅上，双鬓泛白的中年人！
<br/>　　看到此人的一瞬间，楚涵便猛然精神紧绷到极限，嘴唇抿成了一条直线，漆黑如墨的双眸直视前方。
<br/>　　他的另一只手缓缓抬起，将修罗战斧用双手握住。
<br/>　　刷！
<br/>　　巨大的黑斧抬起上扬，绷直在前方呈现出一种明显的防御姿态。
<br/>　　楚涵已经没有月影斩可用，接下来的战斗唯有近战！
<br/>　　尚九娣和陆玟丞也一下子紧张无比，他们知道苏醒后的楚涵又一次变强，甚至强大到如同君王降临，但能让已经强大无比的楚涵瞬间做出防守姿态，还如此谨慎的人……
<br/>　　“咳——”此时的丧尸王木天忽然咳出一口血，趴在地上狼狈至极。
<br/>　　而那个坐在轮椅上的中年人，就这么淡淡的挡在楚涵与丧尸王之间，两条空空荡荡的裤腿看上毫无战力，但他的姿态和神情却淡然镇定，在这满目苍夷，甚至到处都是月影斩留下的幽幽黑火中，不受丝毫影响！
<br/>　　他穿着一身白色的唐装，身上无任何突显身份的东西，唯有袖口和领口纹着银色的纹路。
<br/>　　他的皮肤很白，苍白的白，像是许久没有见过阳光。
<br/>　　无论是气场、风格还是品味，这个人似乎都低调无比，但又透露着无上的奢华感。
<br/>　　太耀眼了，此人耀眼到，让楚涵一下子不得不让出半个场地的主宰权。
<br/>　　黑与白，势均力敌！
<br/>　　“你就是楚涵。”中年人开口，声音带着微微的沙哑。
<br/>　　他的第一句话没有任何情绪和表情，让人看不出一丝深浅，更无法猜测其身份。
<br/>　　楚涵在此抿嘴，极少数会让他如此被动的状况，让他由心底散发至周身都大写着不爽。
<br/>　　尤其是此人还这么忽然出现，随手挡下了本该杀死丧尸王的一击！
<br/>　　看到楚涵没有回应，中年人的唇角微微一勾，毫不在意的笑容彰显着大度，好似根本不与楚涵这等小辈计较。
<br/>　　“我是白晟，白家家主。”下一句话，中年人如是说。
<br/>　　轰！
<br/>　　楚涵的瞳孔剧烈收缩，情绪好似一颗巨石，狠狠的砸向湖面。
<br/>　　顿时波涛汹涌，风起狂澜！
<br/>　　白家，家主！
<br/>　　那个神秘至极，多年来从未曝光过，更未踏出过白家禁地一步的白家家主！
<br/>　　白晟，他叫白晟！
<br/>　　中年人，残疾，强大，高贵……
<br/>　　一瞬间重重标签重合，全部集中在了这个中年男人身上！
<br/>　　尚九娣和陆玟丞也双双楞在原地，惊愕的说不出话，谁也没有想到在这场斗争的最后，会冒出来这样一个人物。
<br/>　　这段时间无论是大战还是暗杀，统统都是神秘家族的几个嫡系子弟在活跃，似乎那三个尚存的神秘家族中，家主亦或是长老这等级别的人，压根就不参与到其中。
<br/>　　是看不上，是不屑？
<br/>　　而今时今刻，死了一片的白、罗两家死士，打退了异种王，差点杀掉了丧尸王，白家嫡系大小姐身受重伤之时，那几个神秘家族的人一个没来。
<br/>　　却直接来了位家主！
<br/>　　楚涵一瞬间的惊讶和错乱，白晟全部看在眼里，他依然镇定淡然，与白忧似乎除了有些相似的外貌之外，其他地方却完全不同。

<br/>　　这个人……
<br/>　　太危险了！
<br/>　　这是楚涵的第一想法，能够仅此一个人轻而易举改变极端到边缘的气氛，此人的心境到底有多强？

<br/>　　哗啦啦！
<br/>　　木质的轮椅就这么滑动了起来，带着这个残疾人小范围的随意闲逛。
<br/>　　这期间楚涵不敢有任何动作，只能任凭白晟到处看，就像是来观光。
<br/>　　在后方重伤又受到极大惊吓的丧尸王木天抬起头，显然也被白晟的忽然出现所惊，神色满是看不懂现场局势的惊愕。
<br/>　　白晟的出现毫无预兆，他也压根没有接触过这个白家家主。
<br/>　　为什么救他？
<br/>　　为什么救下了他又什么都没说？
<br/>　　太奇怪了！
<br/>　　木天漆黑的双目快速的转动了一下，而后在楚涵全部注意力都放在白晟身上时，用上了自己最后的爆发，猛然将速度提升到极限。
<br/>　　咻！
<br/>　　一下子消失，跑的无影无踪！
