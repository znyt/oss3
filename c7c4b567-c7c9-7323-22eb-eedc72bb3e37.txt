        连华生见傅夜沉进来，连忙把刚准备好的两个不同的资料袋递给了他。
<br/>
<br/>        “第一件事情，马智杰的尸体和解剖资料全部丢了，上头让你派手下协助调查，与之有关的线索都在资料袋里。第二件事情，你的堂叔冷岩峰，私底下有大动作，你得提前做准备了。”他一脸严肃地说道。
<br/>
<br/>        “现在召集大家，去秘密基地开会。”傅夜沉剑眉微蹙，接过连华生递过来的资料袋，一边打开，一边命令道。
<br/>
<br/>        连华生颔首应了声：“是！”抬眸之时，他见他喜上眉梢，脸上露出了久违的笑容，聊起了题外话，“童筱颖把你的心病治好呢？”
<br/>
<br/>        “你觉得呢？”傅夜沉嘴角微扬，意味深长地反问。
<br/>
<br/>        连华生却不识趣地泼冷水：“童筱颖这个女人，你最好不要让外界那群人知道她的存在。不然，她很容易成为你的弱点，会牵制到你的行动。”
<br/>
<br/>        “你有什么好的建议？”傅夜沉笑容渐敛，转瞬间凝眉看向连华生。
<br/>
<br/>        连华生有板有眼地提议：“这段时间是非常时期，你和她的婚事不能公开，甚至你跟他已经领证结婚的事情，也得从电子系统中移除，冷岩峰盯你盯得紧，还有那位幕后大Boss也是。所以，你得把童筱颖藏着掖着，然后还得找其他女人来混淆视听。苏漫雪的存在，倒是一个不错的利用对象。”
<br/>
<br/>        “苏漫雪那种女人不可靠，到头来，指不定会卖了筱颖。”傅夜沉深思熟虑地说，他不能让筱颖也牵涉其中。
<br/>
<br/>        连华生知道，那个童筱颖是他傅夜沉的宝贝，既然不能利用苏漫雪，他思忖了一下，又有了一个主意：“不如，从组织里找一个机敏又能自保的女人，来掩饰童筱颖的存在。”
<br/>
<br/>        “这件事情，交给你去办好了。”傅夜沉淡淡地说道。
<br/>
<br/>        连华生点了点头。
<br/>
<br/>        虽然，他现在真不大看好童筱颖，但是，傅夜沉对她爱的执着，而且木已成舟，他作为他的挚友，要做的不再是阻拦，而是想办法替他保护他想保护的女人。
<br/>
<br/>        傅夜沉和连华生两人出去后，童筱颖泡完澡，身上穿着浴袍，走出浴室门时，看到傅夜沉的那张大床，倏然间，红了脸。
<br/>
<br/>        大哥回来后，会不会继续把刚刚在浴室里的那件没做完的事情做完？
<br/>
<br/>        不行不行！
<br/>
<br/>        不能这样轻易地把自己的身体交付给他！
<br/>
<br/>        她跟他才领证没多久，对他还不够了解，而且也还没有……
<br/>
<br/>        童筱颖使劲地摇了摇头，努力甩走刚刚那个想法。
<br/>
<br/>        她回到了隔壁的卧室里，一如既往地反锁了房门。
<br/>
<br/>        等到傅夜沉回来，发现童筱颖不在自己的卧室里，再去了隔壁房间按下门上的手柄时，才意识到，自己再一次，被筱颖拒之门外。
<br/>
<br/>        他回来得有些晚了，想必筱颖此刻已经睡了。
<br/>
<br/>        抚着额头，他又叹了口气，形单影只地回到了自己的房间里。
<br/>
<br/>        傅夜沉给自己冲了个凉澡，躺床上后辗转难眠，满脑子都是起初两人一起泡玫瑰牛奶浴时的情景。
<br/>
<br/>        那个时候，筱颖到底有没有对他动情？
<br/>
<br/>        或许有，或许没有。
<br/>
<br/>        因为，她说，他是这世上，让她唯一可以依赖的亲人。
<br/>
<br/>        亲人？！
<br/>
<br/>        很可笑，不是吗？
<br/>
<br/>        她只是把他当亲人。
<br/>
<br/>        难道正是因为，她才真只是把他当亲哥哥了，才一而再、再而三地找借口拒绝跟他的做吗？
<br/>
<br/>        这一夜，他又为自己的软肋失眠了。
<br/>
<br/>        翌日，清晨。
<br/>
<br/>        傅夜沉早上起来后，直接去了洗漱间。
<br/>
<br/>        伸手往水龙头下一放，水龙头自动出水，他直接捧了把水往自己脸上扑，想让自己先清醒清醒。
<br/>
<br/>        仰起湿漉漉的一张脸，眼前镜子里的他，居然有了黑眼圈。
<br/>
<br/>        洗漱完后，他肚子又饿了，于是直接下了楼，往餐厅走去。
<br/>
<br/>        碰巧在餐厅里遇见正在摆餐盘的童筱颖，傅夜沉的目光，若有若无地在她身上打量。
<br/>
<br/>        只见她身上穿着一件宽松的白色长T恤，长长的衣摆包住了她圆俏的蜜桃臀，如白玉般洁白匀称的双腿袒露在空气里，令人情不自禁地遐想着那衣摆下是否未着贴身衣物。
<br/>
<br/>        傅夜沉剑眉微扬，朝她走了过去，似笑非笑地说道：“宝贝，我们把昨晚未做完的事情，继续做完，嗯？”
<br/>
<br/>        “欸？”童筱颖不明所以，因为他的话猛然回过神来，便看见站在自己面前的傅夜沉。
<br/>
<br/>        褪去西装的傅夜沉此刻看起来没有那么高高在上，可是那张不带丝毫笑容的脸却让她下意识地紧张起来，也许是因为两人目前的关系，在这偌大的餐厅里，让童筱颖瞬间红了脸。
