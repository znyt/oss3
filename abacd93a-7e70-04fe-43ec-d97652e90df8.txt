<br/>    凌仙自然知道大师姐对他没有什么恶意，不过大师姐对于他们的实力都是心如明镜，了若指掌，都是最大压力化，逼迫他们的修行更上一层楼。

<br/>        卓青瑶剑势凶猛，打得凌仙连连后退，凌仙也能感受到卓青瑶的进攻凶猛无比。
<br/>    
<br/>        如同是一夕顿悟，凌仙忽然想起师父数次施展搏仙术观海的模样，悍然一拳挥出，瀚海潮生！
<br/>    
<br/>        卓青瑶一剑将身前的浪潮劈开，淡然收剑。“还算不错，以你如今的搏仙术造诣，度过超脱劫应该没有什么大问题了，让师父帮你巩固一下，你就可以突破到超脱了！”
<br/>    
<br/>        凌仙又惊又喜，在卓青瑶的重压之下，忽然灵光一闪，居然施展出了搏仙术第二卷的观海！
<br/>    
<br/>        凌仙连忙拱手道：“多谢大师姐！”
<br/>    
<br/>        小胖子却是感动的一塌糊涂，在天庭，他可从来没有见过这般景象，诸位兄长争名夺利，怎么可能像是燕山这般友爱。
<br/>    
<br/>        大师姐和二师兄虽然打打闹闹，但是都没有动真格的，按照卓青瑶轻松一剑化解了二师兄的搏仙术来看，大师姐的实力，远远在二师兄之上，只是想要帮助二师兄突破。

<br/>        凌仙也是惊喜无比，准备安排小胖子住下，一路上，凌仙跟小胖子道：“燕山之上，师父行踪不定，喜欢云游四方，大师姐闭关了，所以呢，你有什么事就跟我，还有你三师兄，四师姐说，我们都会竭尽全力帮你的，谁要是欺负你，二师兄绝对打得他满地找牙！”
<br/>    
<br/>        小胖子心里一阵酸楚，在母亲死后，他还是第一次感受到这般的温暖，“多谢二师兄。”
<br/>    
<br/>        凌仙看着眼泪在眼眶里打转的小胖子，笑道：“哭什么，男子汉大丈夫，流血流汗不流泪！”
<br/>    
<br/>        小胖子毕竟才十二岁，年纪还小，听着凌仙的话，也是擦了擦眼泪。“是，二师兄，师弟记住了！”

<br/>        陈明这边，也是在炼丹的时候接到了一个任务，“叮，你触发了师门任务，凌仙深感自己的修为不够，目前凌仙已经摸到了一丝搏仙术观海境的边缘，请帮助他巩固搏仙术造诣，迎接他的天劫，成就大君！奖励：30w师德，第五个亲传弟子席位。”
<br/>    
<br/>        嗯，凌仙这小子终于要突破了！
<br/>    
<br/>        陈灵鱼都是大君了，这小子一个战斗人员，居然比后勤人员突破的慢！
<br/>    
<br/>        我可是存了两次抽奖机会啊，就等这小子突破，蹭一下他的啊！
<br/>    
<br/>        一般弟子突破大境界的抽奖，都是出好货的机会，终于可以再出一波货了！
<br/>    
<br/>        一时之间，陈明也是有点兴奋，准备前往开拓星，帮助这小子突破，正在这时，陈灵鱼找到了自己，“师父，二师兄那边传来了消息，五师弟到了！”
<br/>    
<br/>        陈明大喜道：“走，你安排一下，随我一同前去看你五师弟拜师！”

<br/>        陈灵鱼也是知道五师弟的重要性，每一个燕山的亲传弟子，都感情深厚，而且权柄昭彰，一位新的亲传弟子，是燕山的大事！

<br/>        陈灵鱼安排了一番，也是在赤潮大陆宣布了万八千拜入陈明门下，成为燕山五宗主的事，一时之间，整个赤潮大陆欢欣鼓舞，都准备庆贺一番。

<br/>        毕竟，燕山越强大，他们的生存，也更有保障。

<br/>        当然，其中拍马屁因素更重。
<br/>    
<br/>        现在燕山一统赤潮大陆，陈明新收一名亲传弟子，整个大陆自然是要做一番表态的。
<br/>    
<br/>        陈明也是清楚其中的门门道道，但是并不妨碍他高兴啊，别说，这些修行者还挺聪明的！

<br/>        带着陈灵鱼，两人来到开拓星，李素衣听说了五师弟到了，也是赶了回来，准备观看小胖子拜师。

<br/>        众多的燕山记名弟子，也是在此刻赶回了燕山分部，准备好了拜山头。
