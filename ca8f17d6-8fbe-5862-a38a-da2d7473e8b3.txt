　　元始三清经过万年游历，回到洪荒第二神山昆仑神山，三清的第一个家。
<br/>　　只是三人刚回到昆仑，还没有进去昆仑山谷，就心神不由自已地一动，像是有着什么事物吸引着自己一般。元始顿时知道，是自己的机缘到了。
<br/>　　他扭头一看，却见老子与通天也是同一副模样，他们都感受到与自己有缘的东西即将出世。
<br/>　　顿时，元始心中奇怪，是什么东西能够让三人都感受到机缘来临，莫非是……
<br/>　　元始顿时心中一震，对着二人道：
<br/>　　“看来此机缘与我等三人有关，如此，我等同去。”
<br/>　　二人自然没有意见，道了一口“大善”，于是三人就纷纷朝着感应到的方向走去。
<br/>　　不久，元始三人就来到一处山谷之中，山谷之中，迷雾笼罩，青光闪耀，却又看不透里面到底如何，显然，此处是有大阵掩护。
<br/>　　“三弟，此阵就看你的了。”
<br/>　　元始对着通天说道。三人早有分配，若是遇到大阵，通天解决，炼器之事，找元始，炼丹之事，找老子。各自有各自的任务，不至于兄弟相互争吵。
<br/>　　通天阵道天赋极强，就连元始都自叹不如，若不是元始本身对于大道的领悟深厚，一法通，万法通，顺带着阵道也不错的话，他的阵道早已被通天超越了。
<br/>　　“好，看我的。”
<br/>　　通天亦不废话，一路之上，三人早已配合得默契无间。通天阵道天眼一开，眼中青光闪耀，看向大阵，阵中种种玄妙，顿时了然于胸。
<br/>　　挥手之间，几道道法打进虚空，顿时，虚空颤动，阵光飞舞，山谷裂开一道大口，却是阵法已被通天破了。
<br/>　　“三弟，不错！阵道修为长进了不少。”
<br/>　　元始淡淡评价道，难得夸赞通天一句，要知道，平时元始一直都是懒得废话的。
<br/>　　“多谢大兄夸赞，小弟这点修为在大兄面前，还是远远的不如啊。”
<br/>　　显然，通天听了元始的话，也很是高兴。不过，他还是很谦虚的，不谦虚不行啊，想当初为破一道大阵，通天研究了半年时间都没有研究透彻。结果换了元始，神念一开，几道阵旗往阵中一放，顿时大阵破灭。他简直是自叹不如啊。
<br/>　　“我可不会随便夸人，说你不错，你就是不错。在这洪荒之中，以你的阵道修为，已经超越无数人了，可堪此界巅峰。不过，距离真正阵道大成，还真是差的有些远了。”
<br/>　　元始微微一笑，对着通天道。夸赞一句只是偶尔的，重要的是后一句，打击一下也无妨，不让他产生骄傲之心。
<br/>　　“啊？！”通天错愕，接着问元始道：
<br/>　　“那大兄，什么才是阵道大成呢？”
<br/>　　“万道殊途同归，都是一样的，阵道大成者，一阵生世界，一阵灭世界。一阵之中，包含乾坤万道，造化无数，变幻莫测。”
<br/>　　“一阵生世界，一阵灭世界……”通天喃喃道，脑海之中一道灵光闪过，好似想到了什么，又没有想到什么，总是抓不住。
<br/>　　“行了，不要再纠结了，以你的阵道天赋，差的只是对于大道的积累，若是道行上去了，这些就都不是问题了。所以，现如今，还是以大道为重。”
<br/>　　元始看着在思考着问题的通天，微微道。
<br/>　　“小弟谨记大兄教诲。”
<br/>　　通天顿时醒悟过来，是啊，自己现在纠结这些干什么呢，反倒是落了下乘，悟了就是悟了，不悟就是不悟，自己是谁？自己可是盘古元神所化的上清通天啊，自己早晚可将其悟透。
<br/>　　老子在一旁也有领悟，之前他们总感觉有些不对，现在他们明白了，对于他而言，现在最重要的是道，道行上去了，修为境界也自然上去了，至于其他的神通术法之类的，那只是护道手段，不能主次不分。
<br/>　　三人进入阵中，只见阵中自行衍化空间，迷雾阵阵，抬眼不过数百米距离。阵中迷雾有遮掩神识作用，在这迷雾之中，神识被极限压低，只能靠肉眼。
<br/>　　元始的神识强大，尽管依旧被压制，但他还是一扫便扫遍了阵中的全部变化。就连大阵最中间的宝物，他也看到了。
<br/>　　“这是什么鬼？竟然能够压制神识？”通天不爽道。
<br/>　　“跟我来。”元始淡淡道了一句，便不再说话，带着两人绕过重重阵法，向着阵中走去。
<br/>　　越是往前走，三清越感觉阵中宝物与自己有缘，这种感觉越来越明显。通天这时说道：
<br/>　　“两位哥哥，我感觉这阵中的宝物与自己有缘，你们有没有这种感觉？”
<br/>　　元始不说话，老子也感觉到了自己的缘法，道：“三弟所言不虚，为兄也有这种感觉。”
<br/>　　三清一路加速，有元始带路，绕开了重重阻拦，终于破开最后一道障碍，来到阵心。
<br/>　　只见阵心之中，青光阵阵，涌现着一股澎湃的先天灵气，一道青光卷着一朵青色莲花漂浮在阵心。
<br/>　　“十二品造化青莲！”
<br/>　　老子通天都是惊讶的大叫一声，两人都是盘古元神所化，自然知晓混沌至宝混沌青莲，混沌青莲在开天之初化为四朵十二品莲台，分别为十二品造化青莲、十二品灭世黑莲，十二品功德金莲、十二品业火红莲，四朵莲花皆为极品先天灵宝，妙用无穷。
<br/>　　其中当以十二品造化青莲最为玄妙，为极品先天灵宝，又为十大先天灵根之一。
<br/>　　元始心道，果然是十二品青莲。之前三人都有感应之时，他便有了猜测，如今眼前这一幕，正好证实了他的猜测。
<br/>　　元始看着漂在山谷池塘之中的造化青莲，尽管心中激动，但却没有如通天与老子一样，他的宝物不少，十二品净世白莲也有，多青莲一样不多，少它一样不少，没有多大关系。但老子与通天不同啊，他们如今还是很穷啊。尽管手中有几件先天灵宝，但那灵宝品级都不是很高。
<br/>　　只见池塘之中，除了一朵十二品青莲外，还有两朵九品青莲以及三朵六品青莲，培育青莲的水也不是凡水，而是十大真水之中的天一真水。
<br/>　　天一真水，与三光神水一样，造化无双，滋养万物，不仅可以提升修为，还可以通过天一真水领悟天一真意，融入到神通之中，增强神通威力。
<br/>　　可以说不管是青莲，还是天一真水，都是宝贝。
<br/>　　只是，十二品青莲只有一朵，而他们却又三人，这该如何分配，既不损害三人之间的情意，也不会破坏宝物？
<br/>　　宝物多了也是烦恼啊！
