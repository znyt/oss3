        &lt;h3&gt;第三卷第一百零二章  绾绾（感谢公子绾盟主的打赏）&lt;/h3&gt;
<br/>
<br/>        安排完公司所有的事情，陈子霖觉得时间还早，他得打电话给哥哥姐姐们让他们知道他要去北京了。
<br/>
<br/>        他们都很忙，偶尔在群里说几句话，除非有大事情，大家才在一起商议，或者是家人聚会。
<br/>
<br/>        当他发了一个“哈喽，我来了。”时，群里寂静无声。
<br/>
<br/>        看了看大家的图像，都是在忙碌的状态，而且二哥的图像是灰的。
<br/>
<br/>        于是他就打电话给二哥，提示关机。
<br/>
<br/>        他又给四姐打电话，四姐挂掉了，发了个信息：“开会中，回头联系。”
<br/>
<br/>        陈子霖叹了口气，他给潘青松打电话，响了好多声，也没有人接电话。
<br/>
<br/>        “唉，现在都忙得见不到人影，连电话都打不通了。”陈子霖自言自语。
<br/>
<br/>        陈子霖拿着电话发了一下愣，又给陈子悦打电话，还好，电话响了好多声陈子悦终于接电话了，陈子霖听见五姐的声音心里一喜，眼泪差点就流下来了。
<br/>
<br/>        “子霖，最近好吗？”虽然他们在同一个城市，但是工作很忙，又有小孩，其实见面的机会不是很多。
<br/>
<br/>        “五姐，子昂让我去北京。”陈子霖的语气真真的是个弟弟。
<br/>
<br/>        “子昂是不是要去甸城了？”陈子悦想着陈子昂可能要起甸城，才让陈子霖过去的。
<br/>
<br/>        “不是，子昂说成立耗材运营部了，让我和江俞轩调换工作。”
<br/>
<br/>        “你和江俞轩调换工作？你可以吗？”陈子悦有点担心弟弟娶北京能不能服众，毕竟北京的公司比西城大了很多。
<br/>
<br/>        “不过你过去也好，最起码可以陪在子昂的身边。”知道了妹妹的辛苦，身边有个自己的人也好。
<br/>
<br/>        “唉，我也是这样想的，你说在西城管理我还能勉勉强强应付下来，去北京我担心我管理不好。”陈子霖这几年已经成长起来了，而且自己在外面偷偷的报了好几个学习班，但是心里还是没有谱。
<br/>
<br/>        “没啥好担心的，子昂让你去你就去吧。”陈子悦转过语气来安慰弟弟。
<br/>
<br/>        “五姐，最近有和二哥联系吗？”自从上次他们聚会闹的不太美气，大家都基本不和二哥联系，其实想一想，二哥也很为难。
<br/>
<br/>        “没有呢，二哥怎么了？”陈子悦自从上次之后也很生二嫂的气，大家都不联系二哥了。
<br/>
<br/>        “没怎么，就是我打电话给二哥没有人接，关机了。”陈子霖心里有些担心二哥，二哥很少电话关机的，现在又是上班期间。
<br/>
<br/>        “估计有事情，我明天再打打。”
<br/>
<br/>        “行，姐，等我去北京之前咱坐一坐吧，或者回去看看爸妈。”去北京估计回家就不方便了，他要回去看看父母。
<br/>
<br/>        “那就回去看看爸妈吧，我也回去，我再问问四姐，就放周末吧。”陈子悦大概三个月没有回家了，到是前几次打电话，妈妈说二哥回去了，她也没有多问。
<br/>
<br/>        父母亲年龄大了，是应该常回家看看。
<br/>
<br/>        陈子霖挂了五姐的电话，又打通了陈子昂的电话。
<br/>
<br/>        陈子昂正在和赵绾绾商量信息化的事情，陈子昂看了一下电话，对赵绾绾说：“绾绾，你等一下，我接下电话。”
<br/>
<br/>        “哥，有事情吗？”
<br/>
<br/>        “我和五姐商量了一下，这周回一趟家，周一北京。”陈子霖看着外面飘落的树叶。
<br/>
<br/>        “今天周几？”陈子昂问身边的赵绾绾。
<br/>
<br/>        “周三。”赵绾绾看了一下日历。
<br/>
<br/>        “哥，你们什么时候回去？”
<br/>
<br/>        “周五，回去可能也呆不了多久，大家都忙。”
<br/>
<br/>        “行，如果大家都回去的话，我也回去。那就这样说好了啊，我挂电话了。“赵绾绾还在旁边坐着呢，大家的时间都很紧。
<br/>
<br/>        “绾绾，你也不多休息两天？”陈子昂心疼的说，赵绾绾也是工作狂，加班加点的，把自己给累倒了，住了十几天院。
<br/>
<br/>        “我是很想休息啊，跟着你这个疯子能休息吗？”赵绾绾笑着说。
<br/>
<br/>        “你要劳逸结合啊，还有你这体质不行，该锻炼锻炼了，我前几天给宁雅说了，这一段时间在筹建健身房。以后每天去跑跑步。”
<br/>
<br/>        赵绾绾生病之后，陈子昂才发现，她的这些人都是工作狂，基本都泡在工作上，很少有人参加体育锻炼，于是安排宁雅给单位建立一个健身房，让他们每天跑跑步，身体是革命的本钱，没有个好身体一切都是免谈。
<br/>
<br/>        “好，都听你的，你说了算。“赵绾绾微笑着说。
<br/>
<br/>        “西城和山东，还有青海的项目后续都得需要开接口，虽然提前预留了位置，但是后期随着社会的发展，各单位的接口会越来越多，我们得赶快升级我们的软件，要不然后期会被淘汰。”赵绾绾对陈子昂说。
<br/>
<br/>        “软件这一块你负责，该怎么样做你全权处理，资金直接要盛夏要，对了，给你们单独成立一个财务处吧，将陈思宇分给你，你们的财务支出单独核算，你觉得怎么样？”陈子昂征求赵绾绾的意见。
<br/>
<br/>        “如果真是这样，那太好了，每次申请费用麻烦死了。”赵绾绾大喜，他们每次需要什么东西，都要好久，太浪费时间了。
<br/>
<br/>        “你这贪心鬼，现在公司不是我一个人的了，花钱还是要注意点。”陈子昂笑着说，不过赵绾绾是出名的“精明女神”，和她打交道的人都知道她的那个脑袋就是一台计算器，点点滴滴，分分毛毛都给你计算的清清楚楚。
<br/>
<br/>        所以陈子昂才放心大胆的给她一个独立的财务部。
<br/>
<br/>        “嘿嘿，我贪下来的还不都是大家的。”赵绾绾高兴的说，其实她的身体还没有完全康复，只是工作上的事情太紧急了。
<br/>
<br/>        寒伯安他们交代的软件正在紧要关头，说实话，只有她知道陈子昂需要的是什么东西，她不盯着怎么成，花费那么大的力气做出来的东西要是用不成，到时候哭都没有地方哭。
<br/>
<br/>        公司正在发展中，一步都不能错。在医院她都没有闲着，远程遥控，医生都拿她没有办法。
<br/>
<br/>        “寒总他们的那个软件怎么样了？”陈子昂问赵绾绾。
<br/>
<br/>        “这几天是紧要关头了，我天天都盯着呢，很快就有结果了。到时候用我们公司的车先试验一下。”赵绾绾自信的样子真好看。
<br/>
<br/>        “绾绾，那辛苦你了，不过，我要告诉你，身体最重要，多少钱都没有你在我心里值钱，你就是我们公司的一个宝。你知道吗？”陈子昂切切叮咛。
<br/>
<br/>        “疯子，别婆婆妈妈的了，咱公司的人那个不是你的宝？你的宝太多了。”赵绾绾笑骂着。
<br/>
<br/>        “当然了，没有你们这些无价之宝，那有我陈子昂啊。”陈子昂感叹着。
<br/>
<br/>        “所以，你更是宝中宝了，宝王。我要工作去了，我们都要好好的，你也要注意身体。”赵绾绾生了一次病，确实感觉到了身体的重要性。
<br/>
<br/>        生病，特么的太难受了。
<br/>
<br/>        看着笑语盈盈离去的赵绾绾，陈子昂心里一片感动，她总是很容易被感动，而她也经常被感动！有他们这些人陪着她真好！
