　　第323章
<br/>　　程大雷一直等待的战斗终于到来。
<br/>　　莫鸣米集结兵力，从东面开始进攻，没有骑兵。前面是盾牌兵，其后是弓兵，在其后是投石机，攻城车之类的攻城器械。
<br/>　　帝国军阵和戎族是不同的，戎族擅长骑兵野战，而帝国则擅长重械攻城。这一次，程大雷面临的考验，甚至要严峻过青牛山一战。
<br/>　　只不过不同的是，戎族有不死不休战斗意志，帝国却无。
<br/>　　在对方进入射程后，徐神机下令放箭。数百架重弩蓄势待发，随着徐神机大手挥下，同时激发，一时间箭若雨下。
<br/>　　重箭携裹着巨大的冲力，由上向下冲入战阵，死伤无算。
<br/>　　“程大雷，派人杀出去吧。”李行哉道。
<br/>　　程大雷摇摇头，没有吭声。此刻就连李行哉都看得明白，蛤蟆岛没有青牛山那样的坚城，久守必失。但对方也没有戎族的战斗力，军员素质对比，蛤蟆寨的山贼是强过敌人的。
<br/>　　假若正面交锋，派一支精兵冲杀出去，直抵对方中枢。或许，这支精兵就将成为改变战斗走向的力量。
<br/>　　当然，这支精兵也是敢死队，冲入敌阵之后，究竟有多少活着，有多少会死去，这是谁也说不好的事情。
<br/>　　看着紧张关注战局的程大雷，李行哉轻轻叹口气。慈不带兵，义不行贾，程大雷如此心境，终究只是一个占山为王的山贼而已。
<br/>　　哪怕仅凭着据称坚守，程大雷也撑过了第一波攻城。随着重箭如雨，敌方的战斗意志被击溃，仓皇逃命，程大雷此刻才下令秦蛮等人出击，开始痛打落水狗。
<br/>　　最后，敌方丢下几百具尸体后，选择暂时退兵。
<br/>　　这一战，可以说是有惊无险。但程大雷也明白，今天只是双方的互相试探而已，莫鸣米还没有动用真家伙。
<br/>　　而通过这一战，程大雷大概也判断出敌方的单兵战力，团体配合，战斗意志……总的来说，可以用四个字来形容：不堪一击。
<br/>　　关键是对方人数太多了，总兵力过万，超过蛤蟆寨兵力十倍。而这还不是琴川关的全部，仍有超五千的队伍留守琴川关。
<br/>　　到晚上，蛤蟆寨灯火通明，程大雷组织人手召开战前会议。总得来说，对方虽然兵力占优，但蛤蟆寨单兵素质远强过对方，关键是如何在获得胜利的前提下，尽量减少伤亡。
<br/>　　程大雷又做了战前动员，类似明天是光明的，昨天是光荣的，但今天很残酷之类的话。
<br/>　　……
<br/>　　琴川关大营，依旧有一场重要会议。
<br/>　　大战之前，莫鸣米心情很愉悦。今天是彻底摸了摸蛤蟆寨的底，情况要比自己所想的乐观。去掉河道的天险后，蛤蟆寨几乎没有称得上防御的设施，在莫鸣米心里，蛤蟆寨的防御也可以用四个字来形容：不堪一击。
<br/>　　程大雷所不知道的是，今天攻打蛤蟆寨的敌人，只是琴川关的老弱病残，是可以牺牲的炮灰。
<br/>　　这些人几乎已没什么战斗力，活着也是浪费兵力。趁着这次攻打蛤蟆寨，正好清理一下『垃圾』，也可以顺便探出蛤蟆寨的底细。
<br/>　　会议过后，莫鸣米单独将严笛留下了。
<br/>　　“军师，攻破蛤蟆寨就在眼下，到时候定可亲手杀了程大雷这贼人。”
<br/>　　“恭喜将军大人！”严笛立刻拱手，能除掉蛤蟆寨，他其实也可以去块心病。
<br/>　　“三军未动，粮草先行，眼下还要你回琴川关坐阵，主持后勤。”
<br/>　　“定不负将军所托。”
<br/>　　“其他各地的头目，也该带过来了，他们休想闲着看热闹。”
<br/>　　“将军，云中龙，乔鹤，游九楼三人该如何办？”
<br/>　　“同样把消息传给他们，看他们是打算一条路走到黑，还是愿意弃暗投明。”
<br/>　　当天夜里，严笛便在士兵的护送下返回琴川关，他将在那里坐阵，为莫鸣米提供后勤支援。
<br/>　　同时消息已经散出去，莫鸣米手下十八家首领都得到了消息，其中包括云中龙、乔鹤、游九楼。
<br/>　　“若说来，程当家对我们不薄，十成利总能分给我们每家一二成，做事可要比莫鸣米厚道多了。”
<br/>　　在出发之前，三人聚在一起，商议这次的事情。
<br/>　　“厚道有什么用，厚道能挡得住莫老大么？咱们三个是什么人，只是莫老大屋里的臭虫而已。莫老大不愿意搭理我们，我们还可活着；莫老大想我们死的时候，一脚就可以踩死我们。”
<br/>　　“那怎么办？”乔鹤问：“咱们真去和程当家拼命，就算程当家这次必死无疑，可咱们曾经毕竟跟过他，事情过去之后，莫老大会放过我们么？”
<br/>　　三人长吁短叹，一时都不知该如何抉择。最后乔鹤和游九楼的目光都看向云中龙。
<br/>　　“龙老大，你倒是拿个主意呀，这次咱们都听你的。”
<br/>　　云中龙想了想，忽然一拍桌子：“去就去，咱们去了出工不出力，先混过这件事再说。”
<br/>　　乔鹤与游九楼叹了口气，云中龙所想，自然不是最好的办法，可却是目前唯一的办法。
<br/>　　十八家头领在三日内集结完毕，大大小小的势力在大河两岸扎营。程大雷眼睁睁看着这一幕发生，却无能为力。
<br/>　　莫鸣米原先的兵力，再加上十八家头领的兵力，总兵力已经超过两万。
<br/>　　三日后，莫鸣米再次发动攻城行动。此刻在他眼中，程大雷已经是囊中之物，随时可破，之所以目前还未被攻破，还是因为他舍不得付出太大代价。
<br/>　　蛤蟆寨可破，但是一块难啃的骨头，想要彻底攻陷敌人，必定要损失大量人力。莫鸣米舍不得牺牲自己的精锐，便让十八家头领的人去死。
<br/>　　也正好趁这个机会，削弱一下他们的实力，免得手下有人坐大。
<br/>　　而十八家头领更舍不得自己的人送死，于是能打就打，打不过就撤，在如此情况下，战斗持续了半个月有余。
<br/>　　程大雷靠着手中的重弩，在没有增加兵力的情况下，竟然真的撑了下来。
<br/>　　这一日黄昏，又一场战斗过后，大河之上布满尸体，晶莹的冰面上凝固着红色血水。
<br/>　　嘟，恭喜完成十次山寨防守……
<br/>(本章完)
