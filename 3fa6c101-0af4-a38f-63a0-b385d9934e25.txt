陆寒霆来了。
<br/>
<br/>        今天的陆寒霆穿了一身合体的黑色西装，西装口罩里叠着白色的绢巾，他下了车，然后信步走上了红地毯，步伐稳健自带霸总的气场，居高临下的高冷矜贵里透着不近人情的冷漠疏离。
<br/>
<br/>        就是这样一个男人，现在他的身上竟然贴上了多种标签性.骚扰狂，变.态狂，性.无能…
<br/>
<br/>        这强烈的反差让现场所有的记者都在疯狂拥挤，黑压压一片全是人头，大家将话筒争先恐吓的递了过去，
<br/>
<br/>        陆总，请问你跟夏夕绾究竟是什么关系？
<br/>
<br/>        陆总，夏夕绾是不是情陷你们陆氏两兄弟之间，你们会为了争夺她而反目吗？
<br/>
<br/>        陆总，你的身体还好吗，你亲锤自己被夏夕绾给顶废了，对于这种铁锅炖自己的自杀行为你是怎么看的呢？
<br/>
<br/>        这时保镖出动了，里外三层全部拉开警戒线，严毅站出来摆手道，“不好意思，我们总裁不接收任何的采访。”
<br/>
<br/>        所有媒体记者都被挡住了，他们抬头看去，陆寒霆已经进了陆氏大厅，他身后跟了一群脖间挂蓝牌的陆氏员工，他面无表情的大步而去，璀璨的晨曦透过蹭亮的落地窗镀洒了进来，将他衬出几分炫目的瑰丽。
<br/>
<br/>        很快，他消失在了所有人的视线里。
<br/>
<br/>        ……
<br/>
<br/>        总裁办公室里。
<br/>
<br/>        严毅走进去，陆寒霆坐在办公椅上正在处理文件，他垂着英俊的眼睑，手里拿着钢笔在文件的签名处“刷刷”签上自己的大名，这名字签的龙飞凤舞。
<br/>
<br/>        严毅一时也吃不准自家总裁在想些什么，外面已经满城风雨了，他和夏夕绾都站在风口浪尖上，舆论一再的发酵，但是他像个没事人一样，一点都不关心那些。
<br/>
<br/>        “总裁，要不要让公关将网上那些言论全部公关掉？”严毅试探的问。
<br/>
<br/>        陆寒霆没有抬头，没有人知道他心里究竟在想些什么，他只是掀了掀薄唇，“不用了。”
<br/>
<br/>        他没有让手下的公关出手，而是任由舆论发酵，他在想些什么？
<br/>
<br/>        严毅不敢问，“总裁，那我先下去了。”
<br/>
<br/>        “去吧。”
<br/>
<br/>        严毅走了出去。
<br/>
<br/>        总裁办公室彻底安静下来了，陆寒霆将棘手的文件全部处理掉了，将钢笔丢在了桌上，他将挺括的后背重重的倚靠进了沙发背里，然后抬手扯了一下脖间的领带。
<br/>
<br/>        他拿出手机，手机里空空的，夏夕绾没有给他打电话，就连一条短信都没有发给他。
<br/>
<br/>        她究竟什么意思？
<br/>
<br/>        吃饱了，抹抹嘴巴就跑了？
<br/>
<br/>        陆寒霆从喉头里滚出了一道森然的笑声，她这个拔X无情的样子还真是像极了大渣女！
<br/>
<br/>        放下手机，他敛着英俊的眼睑轻轻闭上了眼，这时脑海里浮现出了一个画面，夜凉如水的晚上，她将他推倒草木丛里，她一头乌黑清纯的长发都在飞舞，那水蛇般的小蛮腰就在他的掌心里…
