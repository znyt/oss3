　　这雷劫，也是分为数种。
<br/>　　其中最为寻常的，便是数十道天罚之雷的雷劫，寻常巅峰圣者，踏入生死境，引来的都是这种寻常雷劫。
<br/>　　而只有一些天才，像是北邙学院的一些地榜武者，踏入生死境的话，便是会引来百道天罚之雷的雷劫。
<br/>　　这便是百雷王劫。
<br/>　　一旦是能够度过百雷王劫，便是一大机遇，这样的天才，放在外面去，也会被无数势力所争夺拉拢。
<br/>　　而像是一些百年难遇的天才妖孽，例如许多先天神体，血脉高贵的武者，在度过生死大劫的时候，便是可能会引来千道雷劫。
<br/>　　千雷皇劫。
<br/>　　千雷皇劫，在整个北域，只怕是十年都难以出现一次，每一个能够引来千雷皇劫的存在，都是以圣者巅峰，足以击败生死境的妖孽天才。
<br/>　　一旦是有人度过千雷皇劫，只怕是其中资质，在偌大北域，都能排列前排，受人仰慕。
<br/>　　就算是在北邙学院这样的庞大势力中，要是有弟子可以度过千雷皇劫，只怕也会被学院重点培养，是绝对天才中的天才，妖孽一般的存在。
<br/>　　而在千雷皇劫之上，便是有着最后一种，万雷帝劫。
<br/>　　这种万雷帝劫，千年难遇，比起千雷皇劫出现的可能性，还要小一百倍。
<br/>　　甚至是许多人只怕都没有听说过，楚风眠熟读古籍，才会知晓这万雷帝劫的存在。
<br/>　　万雷帝劫，便是雷云之中，蕴含万道天罚之雷，一旦出现，天地变色。
<br/>　　真正能够度过万雷帝劫的人，在九域中，都是寥寥无几，每一个人将来都是绝代强者。
<br/>　　这是只有天才中的天才，妖孽中的妖孽，才会有人有可能引来的。
<br/>　　如今整个北域，都千年没有人引来过万雷帝劫了。
<br/>　　就算是前世，楚风眠的无数师兄师姐中，真正能够引来万雷帝劫的人，也不过三人，剩下的人，全都是千雷皇劫。
<br/>　　剑道之主的弟子，每一尊都是名震九域的存在，才不过三人引来过万雷帝劫。
<br/>　　楚风眠清楚，以他现在的无数奇遇，远古战龙血脉，造化之力，无上剑意，引来的天劫，绝不会小。
<br/>　　但是他也意料不到，这仅仅踏入神力境，居然是一下子引来了万雷帝劫。
<br/>　　在这劫云之中，每一道劫雷的力量，都是足以媲美寻常的生死境武者。
<br/>　　如今这万雷帝劫，便是相当于万尊生死境强者，同时出手，这种力量，简直是恐怖到了极致。
<br/>　　足以让整个北域都震动的恐怖力量。
<br/>　　“万雷帝劫又如何？我要称霸九域，这小小的劫云，又能乃我何！”
<br/>　　楚风眠抬起头，脸色平静到极点。
<br/>　　他要称霸九域，踩死七大宗门，小小劫雷这一关，楚风眠当然要过。
<br/>　　怕？
<br/>　　楚风眠前世一个没有力量的人，都不会怕，更何况他这一世归来，早已不惧一切。
<br/>　　“来吧！”
<br/>　　楚风眠看向天空之上的万雷帝劫，眼神中甚至是露出了几分挑衅。
<br/>　　“轰隆！”
<br/>　　一声震耳雷鸣，突然无数劫雷，纷纷落下。
<br/>　　这万雷帝劫，仿佛都是感觉到了楚风眠的挑衅，怒火中烧，天意一怒，绝对比起任何强者，都要恐怖的多。
<br/>　　瞬息之间，万道劫雷，尽数落下，一道道雷劫，如同雨点一般密集，密密麻麻的向着楚风眠轰击过来。
<br/>　　要将如今的楚风眠，彻底的毁灭。
<br/>　　“远古杀神！战龙之躯！”
<br/>　　楚风眠猛然咆哮一声，远古杀神率先凝聚而出，挡在楚风眠面前，抵挡劫雷。
<br/>　　同时楚风眠的身躯，丝毫不躲闪，反而是直接迎了上去。
<br/>　　这雷劫，无法躲避，躲避也会被无穷无极的追击，只能正面抗衡。
<br/>　　楚风眠的身躯上，顿时战龙决的力量，被催动到了极致，他身上的皮肤，都变的粗糙起来，看起来如同一枚枚龙鳞一般。
<br/>　　“战！”
<br/>　　楚风眠一下子踏入劫雷之中，顿时无穷无尽的雷霆，简直是要将楚风眠的身躯给硬生生撕裂一般。
<br/>　　但是楚风眠大手一挥，全身的力量爆发而出，居然是将周围的雷霆，都给撕碎。
<br/>　　“给我吞！万雷帝劫又如何？今日我就要拿这万雷帝劫，来凝炼我的身躯！”
<br/>　　楚风眠长啸一声，抓取这无数雷霆，直接向着他的身上撞了过去。
<br/>　　这撞击，令楚风眠身上的血肉，都是一下子炸裂开来，无数雷霆涌入到了楚风眠的身躯之中去。
<br/>　　但是楚风眠脸上，却是没有任何变化，反而是只有冷笑。
<br/>　　“炼化！”
<br/>　　一丝丝雷霆，涌入到楚风眠的身躯中去，却是被楚风眠身躯中的力量，彻底的化解。
<br/>　　这些雷霆，全部融入到了楚风眠的身躯之中去，令楚风眠的身躯，都开始呈现出了一种诡异的紫色纹路。
<br/>　　以万雷帝劫炼体。
<br/>　　这简直是疯狂到了极点的行为，就算在妖孽的绝世天才，碰到这万雷帝劫，也必须要全力抵挡，不敢有着任何一丝大意。
<br/>　　历代的天才之中，能够真正引来万雷帝劫的天才，都是凤毛麟角，但是这些天才中，真正能够度过万雷帝劫的，不足三成。
<br/>　　所以任何天才，碰到万雷帝劫，都是小心谨慎到极点，生怕的稍有不慎，便是被毁灭在雷劫之中。
<br/>　　还从来没有任何一人，像是楚风眠这样疯狂，直接以雷霆来淬炼他的身躯。
<br/>　　“轰隆！轰隆！”
<br/>　　万雷帝劫，甚至都被楚风眠的举动给激怒了，楚风眠的行为，简直是在藐视天威。
<br/>　　他万雷帝劫，代表天意的毁灭，居然是不被楚风眠放在眼中。
<br/>　　更多的雷劫，再一次纷纷落下，其中无数劫雷，都是化为了无数猛兽。
<br/>　　真龙，天凰，麒麟，腾蛇……
<br/>　　一尊尊雷劫化身的，都是远古异兽，纷纷咆哮着，向着楚风眠轰击而来。
<br/>　　“来的好！”
<br/>　　楚风眠看着那一尊尊远古异兽的到来，脸上甚至是不惊反喜，在他的手中，赤炎剑猛然出现。
<br/>　　“剑意化形！纷纷凝聚！”
<br/>　　“吼！”
<br/>　　这剑意，顿时凝聚，化为了一条远古战龙的样子，如今楚风眠的剑意，跟战龙决的力量，已经是彻底的合二为一，威力更甚。
