　　五千‘海狼’与大食人杀的难分难解，两个时辰高强度的战斗双方几乎都拼尽了全力。
<br/>　　但让阿里·伊本想不通的是，唐军一方虽然杀到后来已经成了强弩之末，可依旧不肯放弃近战，数千人死死与他的部下纠缠在一起，以前那种可以发出巨大声音的武器由始至终都没有拿出来过。
<br/>　　这样的情况让阿里·伊本有些犹豫，数次明明有机会将战线推到半山腰都被他犹豫再三而放弃。
<br/>　　而山巅之上，独孤玉林与自家老头子站在一起，看着原本属于他的部下一个又一个倒下，急得几乎要跳起来。
<br/>　　老独孤独有些看不下去儿子莽撞的样子，瞪了他一眼训斥道：“你能不能安静一些，毛毛躁躁成什么样子。”
<br/>　　独孤青云站定身体，指着下面杀正震天的战场，红着眼睛道：“父亲，这样下去不行啊，我们只有五千人，而大食人的后援却源源不断，这样拼下去只怕最后连最后的种子都拼光了。”
<br/>　　独孤青云顺着儿子手指的方向看了一眼，淡淡说道：“那又如何？太上皇的命令就是如此！”
<br/>　　独孤玉林不解的问道：“父亲，我们明明有枪，为什么不能用？为什么我们一定要用舍长取短与大食人近战？太上皇到底怎么了？为什么会下这样的命令！”
<br/>　　“那是因为这次大食人来的太过突然，援兵出现的也太过突然，好像所有的事情都赶到一起了一样。”老独孤说到这里顿了顿，看了一眼儿子：“话我只能说到这里，明白就是明白，不明白就是不明白，我不会再对你解释！”
<br/>　　独孤玉林愣愣看着老爹，已近不惑之龄的他再也不是当初的愣头青，听完自家老爹的话之后，心中唯一的念头就是：太上皇不相信任何人！
<br/>　　是啊！此处远离大唐国土，与大食也相隔万里之遥，那些大食人为什么会在这里拦截他们？而且援军虽然来了，但来的也有些太巧了，谁知道这些援军到底打的什么主意。
<br/>　　李世民之所以不让使用步枪，应该是不想暴露自己的位置，担心被那些援兵暗中算计。
<br/>　　想到这里，独孤玉林的背上不禁冒出一层的冷汗，深吸一口气缓缓平复了自己的心情。
<br/>　　内心深处来讲，独孤玉林觉得李世民有些太过小心，他现在已经是太上皇，离开那个位置之后，地位明显要差上一截，按道理说似乎根本没有必要担心一些有的没的。
<br/>　　援兵不管是由谁带领，能来到这里估计只有一个目的，那就是救人，至于说袭击完全就是一切实际的猜想罢了。
<br/>　　甚至可以说，今天战死在山坡上的那些‘海狼’完全就是冤死，死的完全没有任何意义。
<br/>　　……
<br/>　　贞观炮的声音越来越近了，站在山顶上已经可以看到远处扬起的阵阵硝烟。
<br/>　　近处，大食人似乎疯了一样，拼了命的向山顶上进攻，但却被‘海狼’死死的阻拦在山脚之下。
<br/>　　再看秦琼、尉迟恭、尉迟宝琪三人，此时已经杀的忘记了一切，长枪、钢鞭所到之处立刻就是一片残肢断臂，数不清的大食人在他们面前惨叫着倒下。
<br/>　　而三人此时也是浑身浴血，奋起最后的力量，如地狱中的恶鬼般扑向那些已经快要失去理智的大食人。
<br/>　　杀戮一直在继续，每一次呼吸都会有人惨叫着倒下，但已经被逼到绝路的大食人早已经忘记了生死，他们的心中只有一个念头，那就是杀到山顶，抓住大唐的太上皇，只有这样他们才有机会活下去。
<br/>　　脚下的原本就不怎么结实的泥土已经变成了暗红色的泥沼，鲜血混合着碎肉、断骨，被疯狂的人们踩在脚下。
<br/>　　杀，杀死对方，只要对方死绝了，自己就能活下去！仗打到最后，不管是大食人还是大唐‘海狼’心中都只剩下这一个念头，理智与人性已经不复存在。
<br/>　　武器断了就用石头，石头没了就用拳头，拳头折了那就用牙齿，总之只要能够杀死对方，能够给对方造成伤害，那就要继续打下去。
<br/>　　并不大的小山脚下在短短两、三个时辰内几乎成了血肉磨盘，数万人忘我的厮杀在一起，仿佛每一个人都变成了只知道杀戮的野兽。
<br/>　　秦琼已经在敌人的中间冲杀了不知道多少个来回，无数的敌人倒在了他的长枪之下。
<br/>　　视线开始模糊，体力也严重的透支，杀戮中一个个敌人似乎都变成了从前死在他枪下的亡魂，脸色狰狞的向他扑来，好像在向他索命。
<br/>　　不能倒下，绝对不能倒下，强弩之末的秦琼用腰间的短刀在自己的大腿上狠狠刺了一刀，剧痛让他的精神重新振作起来。短时间内连挑大食十余员战将，斩杀敌兵无数的秦琼隐约意识到这可能是自己的最后一战。
<br/>　　红颜易老，英雄迟暮，六十岁有余的秦琼已经不是当年那个锏打三州六府的‘小孟尝’，不管是精力还是体力都不足以支撑他进行如此高强度的战斗。
<br/>　　不过，秦琼此时却没有任何畏惧，面对数不清的敌人，体力透支严重的他突然间生起一股冲天豪气，在战马满是汗水的脖颈处轻轻一拍，豪迈大喝：“呼雷豹，与吾杀敌！”
<br/>　　“希律律……，昂……”一声如雷鸣般的战马嘶鸣，秦琼坐下战马似乎感觉到了主人决死的意志，陡然间人立而起，待前蹄落下，一人一马已如闪电般向着阿里·伊本的方向冲了过去。
<br/>　　说时迟，那时快，发了狠的呼雷豹像一道闪电般穿过大食人的重重阻拦，眨眼间已经载着秦琼接近到了阿里·伊本的面前。
<br/>　　面对突如其来的攻击，阿里·伊本顿时有些慌了手脚，其身边的护卫见机不对，一齐冲了上来，想要将秦琼拦住。
<br/>　　但就在这个时候，秦琼做出了一个出人意料的举动，只见他手腕一翻，长枪已经被他倒提手中，接着便是振臂一挥。
<br/>　　众人眼前恍如一道电光略过，还没等他们反应过来，一声惨叫已经传来，等那些阿里·伊本的护卫再回头时，他们所要保护的人已经被那长枪死死的钉在了身后的旗杆之上。
<br/>　　阿里·伊本的惨叫声引起了很多人的注意，很多大食人都到了他被钉死的一幕，瞬间，军心大散：“将，将军死了，将军被敌将杀死了！”
