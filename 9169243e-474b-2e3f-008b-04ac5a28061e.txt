　　罗昊、林晓阳、郑钰还有几名战备一队队员携带武器走进写字楼，脚踩在地面的玻璃墙碎片，玻璃碎片跟大理石地面摩擦发出令人牙齿发酸的“嘎吱嘎吱”声响。
<br/>　　写字楼内的电梯在刚才爆炸中遭到损坏，二十五层的电梯门被炸得凹陷下去，有一扇门甚至被炸得掉落下电梯间，砸落在电梯上，令得电梯彻底罢工了。
<br/>　　“咱不是要爬二十五层楼吧？”
<br/>　　郑钰看着自己面前的楼梯，努力咽了一口口水。
<br/>　　“怎么，不行啊？”
<br/>　　罗昊斜眼睨了郑钰一眼。
<br/>　　“不行！？”罗昊的话，瞬间让郑钰跳脚，说道：“你说谁不行？男人怎么可以说不行！？不就二十五层楼么，又不是没爬过！”
<br/>　　对于罗昊、郑钰这种一言不合背起负重背囊就来一个武装越野的人来说，还真不是事儿。
<br/>　　几个人沿着楼梯一路往上冲。
<br/>　　写字楼里的人虽然都离开了，但是写字楼里的监控系统却还在工作着。
<br/>　　智天使和吴云骑着摩托车沿着小路东钻西蹿，逃出一段距离后，两个人骑着摩托车进入一家废弃工厂，在工厂仓库中又把摩托车给扔了，换了一辆七成新的黑色宝马轿车。
<br/>　　智天使伸手在车窗上一按，车门解锁，拉开车门坐进驾驶座，刚钻进汽车，就听到放在中控台上的手机传来警报声。
<br/>　　这部手机连接着写字楼中的监控系统，通过手机可以清楚的看到发生在写字楼内的情况。
<br/>　　“果然还是来了。”
<br/>　　智天使看着手手机中的影像，冷笑道：“既然来了，我自然是要好好招待你们。”
<br/>　　退出监控影像，打开一个程序，在上面按了几下，把手机扔进茶水杯格子中，发动汽车从工厂中迅速离开。
<br/>　　写字楼内，罗昊、林晓阳、郑钰等人还在快速往写字楼的二十五层跑去，就在转过楼梯转角的时候，林晓阳突然停下了脚步。
<br/>　　“怎么了？”
<br/>　　罗昊不解林晓阳的举动。
<br/>　　林晓阳伸出手示意罗昊不要说话，用耳朵仔细的分辨着什么。周围除了喘气声，还有“嘀、嘀、嘀”的声响，这种声响很微弱可还是没有逃过林晓阳的耳朵。
<br/>　　“这是什么声音？”罗昊对林晓阳问道，显然他也是听到了这种轻微的声响。
<br/>　　这“嘀、嘀、嘀”的声响很杂乱，从不同的方向传来。林晓阳耳朵动了动，迅速朝着一个方向跑去，声音是从走廊中一个垃圾桶里传出来的。
<br/>　　林晓阳一把掀开垃圾桶的盖子，看到垃圾桶里东西的时候，眼睛瞬间一凝，垃圾桶中没有垃圾，塞在垃圾桶里的塑胶炸药，整整一个垃圾桶里塞满了塑胶炸药！
<br/>　　塞在垃圾桶里的塑胶炸药布线复杂，表面上安装了一个计时器，上面还有三分钟的倒计时，不，准确的说，还剩下两分五十九秒！
<br/>　　“水银平衡装置，布线杂乱但是却不失条理，做出这种布线的是个高手。”林晓阳很快就对垃圾桶中的炸药做出了判断。
<br/>　　林晓阳的脸色有些凝重，因为垃圾桶中塑胶炸药的布线方式，让他有种很熟悉的感觉，没错，就是熟悉！林晓阳几乎可以确定，垃圾桶中的塑胶炸药跟之前接触过的两次炸弹是出于同一个人之手！
<br/>　　也就是说……
<br/>　　林晓阳很快摇了摇头，打消自己这个推测，对身边其他几个人说道：“你们去看看其他垃圾桶，是不是也有这样的炸药。注意，这些炸药使用了水银平衡装置一定要小心，只要被触动，就有可能发生爆炸！”
<br/>　　罗昊、郑钰还有那几个战备一队的队员全都不是菜鸟，知道要怎么做，很快就把消息告诉了林晓阳，走廊其他垃圾桶里全都有塑胶炸药！
<br/>　　“离开这里！”林晓阳十分果断的说道：“应该不仅是这层楼，这栋楼其他楼层的垃圾桶里应该也有塑胶炸药，一旦爆炸，我们就算是有九条命都不够死的！”
<br/>　　几个人现在位于十二楼，在写字楼的正中间，两分半的时间跑下十二层楼，时间应该是足够的。
<br/>　　罗昊在自己手表上设置了一个两分二十秒的倒计时，然后迅速往楼下跑去。
<br/>　　两分二十秒的时间冲下十二层楼，在时间上算不上充裕，所以大家没有人敢耽搁。
<br/>　　如果这栋写字楼内所有楼层走廊中的垃圾桶中，全都是塞满了塑胶炸药，这样的爆炸当量是非常恐怖的。就如林晓阳所说的那样，如果这栋写字楼里所有的塑胶炸药一起爆炸，他们就算是有九条命都不够死的。
<br/>　　“所有人立刻离开马路！重复，所有人立刻离开马路！”
<br/>　　罗昊按住耳麦大声吼道：“写字楼里有炸药，立刻离开马路！”
<br/>　　写字楼中的炸药一旦爆炸，这栋写字楼的玻璃墙会被冲击波全部震碎，在冲击波的动能作用下，这些被震碎的玻璃墙碎片就会变成比手雷破片还要恐怖的存在，打在人身上绝对是有死无生。
<br/>　　马路上的K国快速反应部队和剩下的战备一队队员立刻撤离，一辆辆汽车迅速从马路上离开，撤入到安全区域内，很快马路上就没有人影了，只剩下散落在地面上的玻璃墙碎片反射出道道刺眼的光芒。
<br/>　　一分钟！
<br/>　　罗昊、郑钰、林晓阳这些人已经是来到了写字楼三楼。
<br/>　　五十五秒！
<br/>　　写字楼二楼！
<br/>　　五十秒！
<br/>　　写字楼底楼大厅！
<br/>　　“跑！跑！跑！快跑！”
<br/>　　虽然他们已经到了写字楼一楼，并且转眼就冲出了写字楼，但是他们的危机却还没有解除，他们还没有跑出玻璃墙碎片的攻击范围。
<br/>　　一辆MPV突然朝着罗昊他们冲来，一个急刹在掉头的同时，停在罗昊、林晓阳、郑钰这些人面前，驾驶座上一名战备一队队员对着他们吼道：“上车！”
<br/>　　罗昊一把拉开车门，直接跳上了汽车。
<br/>　　吱——！
<br/>　　车轮跟地面发出一阵尖锐刺耳的摩擦声，朝着前面蹿出去。
<br/>　　汽车跑出去两三百米后，那栋被用来作为影天使和子午对决场地的写字楼再次发生了爆炸……
