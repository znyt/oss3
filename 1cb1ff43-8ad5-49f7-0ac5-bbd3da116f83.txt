        &lt;h3&gt;第三卷第八十三章  甜蜜蜜&lt;/h3&gt;
<br/>
<br/>        江俞轩接到寒伯安发的图片时候，正在陪张倩楠吃晚餐，他们最近都在忙，张倩楠陪着张函在西城调查一件案子，牵扯的面积比较广。
<br/>
<br/>        而江俞轩也是因为月底要招标，所有的一切都进入紧张的操作流程，一丝也马虎不得。俩个人同时在西城，除了晚上回家外，基本上见不到面。
<br/>
<br/>        俩个人约着一起吃饭，西城古塔旁边的广场有个秀火锅，这个火锅比较出名，都是排队等餐，江俞轩在三天前就定了个座。
<br/>
<br/>        张倩楠说她想吃火锅，虽然张倩楠说要吃火锅，但是她却点的是清汤火锅。
<br/>
<br/>        “你几号投标？”张倩楠问江俞轩，着一段时间见江轩都是很晚了。
<br/>
<br/>        “23号。”江俞轩细心的将菜挑到张倩楠的锅里。
<br/>
<br/>        “都准备好了吗？”
<br/>
<br/>        “明天请人去勘查一下地方，在最后确认一下。”
<br/>
<br/>        “找的谁？”
<br/>
<br/>        “陈子昂的二哥陈子建，西城基本都是找的他。倩楠，如果西城项目成功了，我估计大部分时间我要在西城呆。”
<br/>
<br/>        “那你还回陈子昂的公司吗？”
<br/>
<br/>        “有可能我就要退出他们的管理层了，毕竟不能身兼数职，况且我和寒伯安郝景文他们也有合股。”江俞轩的语气里多少有些不舍。
<br/>
<br/>        “也挺可惜的呢，你在他们公司干了这么久，他们公司现在也不错呢。”张倩楠虽然工作也很忙，但是她还是很关注江俞轩的工作的。
<br/>
<br/>        “算是蒸蒸日上吧，陈子昂这个女人不会消停的。”江俞轩将涮好的菜挑到张倩楠的盘子里。
<br/>
<br/>        “我有时间也很羡慕陈子昂呢，叱咤风云，一个女人竟然如此的优秀。”
<br/>
<br/>        “优秀是优秀，但是她付出的辛苦不是一般人能达到的，你呀就等着安安宁宁的做我江俞轩的老婆吧，我才不要我老婆那么辛苦的卖命呢。”自己的老婆自己心疼，江俞轩可不想张倩楠和陈子昂那样辛苦。
<br/>
<br/>        “我才不要当金丝雀呢。”张倩楠心里甜滋滋的。
<br/>
<br/>        “你在家里只当我的金丝雀就好，让我捧在掌心。外面你就是一个顶天立地的女汉子，我就是你定着的天。”
<br/>
<br/>        “我发现你这张嘴是越了越甜了。”
<br/>
<br/>        “当然，来，你尝尝。”铺面而来的狗粮撒了一火锅店，旁边桌的美女看过来，江俞轩还不忘放一下电，惹得隔桌的美女掩着嘴笑个不停。
<br/>
<br/>        “吃顿饭你都不消停。”张倩楠塞了一块肉在江俞轩的嘴里。
<br/>
<br/>        “嗯呢。”江俞轩满意的看了一眼张倩楠，孺子可教也。
<br/>
<br/>        突然手机有提示音，张倩楠看了一眼江俞轩。江俞轩并没有想去看手机。
<br/>
<br/>        “看看是谁发的，说不定有重要的事情呢。”张倩楠催着江俞轩看手机，她知道江俞轩事情多。
<br/>
<br/>        “下班了还能有什么重要的事情。”江俞轩嘴上这样说着但是还是拿起电话，看了一下。
<br/>
<br/>        “哈哈哈，大哥发的，真有意思，大哥让我帮他找一个相同的手镯，让我发给那些考古的或者是收藏家，看看有没有认识的。”
<br/>
<br/>        “什么样的手镯？我看看。”张倩楠听说手镯心中一动。
<br/>
<br/>        “你看，这样的，这个手镯还真漂亮呢，怕值不少钱吧？”江俞轩递给张倩楠，张倩楠接过手机仔细的看了一下，递给江俞轩。
<br/>
<br/>        “这个手镯真的很漂亮，寒伯安为什么要找另外的一只？”
<br/>
<br/>        “我也不知道，等我改天问问他，大哥让找我就帮他找一下呗，反正也不碍什么事。”
<br/>
<br/>        俩个人吃完饭又去看了一场电影，张倩楠心中有事情，全程看的什么她都不记得了。
<br/>
<br/>        倒是江俞轩拉着张倩楠的手看的兴致勃勃，他和张倩楠其实也是很少在一起看电影的，希望以后这样的机会多一些，俩个人在一起的日子真好。
<br/>
<br/>        等忙过这一段时间他们就结婚，拖了一年又一年了，张倩楠不着急，他都着急了。眼看着水蜜桃挂在自家的树上就是吃不到嘴，太煎熬了。
<br/>
<br/>        “俞轩，我明天得回北京了。”
<br/>
<br/>        “就要走了？”江俞轩不舍。
<br/>
<br/>        “这边的事情怕是要告一段落，我们回去说不定过一段时间还会来。”
<br/>
<br/>        “好，那我明天送你？”
<br/>
<br/>        “不用，我和函哥一起走。”
<br/>
<br/>        “陈子昂最近也在西城呢。”江俞轩想着最近这一段时间没有给张函反馈情况，竟然觉得心里有点对不起张函。
<br/>
<br/>        “俞轩，西城的事情并不像想象中那样平静，你在这里搞开发，肯定会碰到各种各样的人，俞轩，你切记要洁身自好。”张倩楠紧紧抓着江俞轩的手。
<br/>
<br/>        “明白，老婆，遵纪守法，安分做事。不能给我老婆抹黑。”江俞轩知道张倩楠的忧心，这充分说明张倩楠了解他做事情的不易。
<br/>
<br/>        “我说的是真的。”张倩楠板起小脸严肃的说。
<br/>
<br/>        “我也是认真的。”江俞轩捏了捏张倩楠的小手，收起嬉皮笑脸的神情接着说：
<br/>
<br/>        “这个世上上坑坑洼洼的事情很多，一不留神就不知道哪个坑会拌你一跤，有可能是你身边的人，也有可能是你的对手，还有可能是压根不认识你，但是就是看你不爽的人。倩楠，人越往高处走，越会感觉到自己是有责任的。你放心，我会很小心的走好每一步的。
<br/>
<br/>        倩楠，你知道吗，在陈子昂的公司里这几年真的让我成长了很多，也让我懂得了很多，我从一个江家大少沦落成一个打工的，当然这个也是我自己选择的，有时候心里还感觉有落差。可是在工作中锻炼了我，也让我看到了很多之前没有看到的东西。陈子昂的经历不一样，算是一个好老板，但是她的奋斗和努力，她所受到我委屈我是实实在在地看在眼中的，所以倩楠，我宁愿你是我手中的金丝雀。”江俞轩拉着张倩楠的手。
<br/>
<br/>        “俞轩，你肯定也受了不少委屈吧？”听着江俞轩的诉说，张倩楠也能想象到他们工作的不易。
<br/>
<br/>        “不受苦中苦，那能人上人呢？每一个大佬都是从小白鼠努力起来的，谁天生下去就是强者？只有坚持不懈的努力才有美好的未来。”经得起风浪的人才能走得更远。
<br/>
<br/>        “嘿，俞轩，你这鸡汤灌得我信心十足，我一定努力成为业界大佬。”
<br/>
<br/>        “你们业界大佬都被张函霸去了。”
<br/>
<br/>        “那又怎么样？我也可以的。”张倩楠霸气十足。
<br/>
<br/>        “我当然知道你可以，你是谁啊，某知名大学的法系的高材生啊。”江俞轩揉了揉张倩楠那柔软的秀发。
<br/>
<br/>        当初身边粉黛三千，哈哈，虽然有点夸张，但是也是很多了，怎么感觉都比不上张倩楠，好马配好鞍，嗯，就是这么个理。
<br/>
<br/>        张倩楠并不知道江俞轩心里想什么，但是，她从江俞轩的动作中感受到了浓浓的爱意。
<br/>
<br/>        “俞轩，世间繁杂，绚丽也好，低迷也好，只要有你在身边，此生我都是幸福满足的。”
<br/>
<br/>        夜，是温柔的，多情的，在张倩楠和江俞轩的一生中，他们携手与共，过着看似平凡却又不平凡的生活。
