他想让出租车停下来，他不想让她走，他有很多话想要告诉她，告诉她他从来都没有不要她，告诉她在夏家他和夏妍妍什么都没有发生过…
<br/>
<br/>        不知道什么时候开始，她已经嵌入了他的骨血生命里，他原以为自己可以忍受她的离开，毕竟这么多年他忍受了痛苦，习惯了痛苦，黑暗和孤单就像是他生活的一部分，如影随形。
<br/>
<br/>        但是，她是他人生里的一个意外，她给予的温暖，甜蜜，呵护，守望…就像是一束阳光突然照耀到了他的头上，他曾一度以为，他并没有被上帝抛弃，他也可以拥有快乐和幸福。
<br/>
<br/>        可是一切就像是阳光下最梦幻的泡沫，手指轻轻一戳，都破了。
<br/>
<br/>        他的梦醒了。
<br/>
<br/>        陆寒霆觉得喉头腥甜，刚才在沐浴间里生生吞下的血气又从胸膛那里往上翻涌，很快噗一声，他一口鲜血直接喷了出来。
<br/>
<br/>        脚步缓缓停了下来，他高大挺拔的身躯摇晃了两下，“轰”一声直接栽倒在了地上。
<br/>
<br/>        很多路人围了过来，
<br/>
<br/>        先生，你怎么了，快打电话叫救护车！
<br/>
<br/>        天哪，好多血！
<br/>
<br/>        大家惊恐的目光都落在了陆寒霆的黑色西裤上，那里有大片大片的血液往外涌了出来，很快就汇成了一条小血河，看着触目惊心。
<br/>
<br/>        他那里…好像受伤了。
<br/>
<br/>        陆寒霆倒在地上，他猩红而偏执的狭眸一直紧锁着那个方向，但是，那辆出租车缓缓消失在了他的视线里，什么都看不见了。
<br/>
<br/>        她真的走了。
<br/>
<br/>        “绾绾…”
<br/>
<br/>        他嗓音沙哑的呢喃着她的名字，闭上了眼的瞬间才发觉眼眶滚烫，很快就有一滴泪从里面滑落了下来，然后没入他的发里消失不见。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆再睁开眼的时候已经在医院里了，病房里充斥着一股消毒水的味道，入眼都是冰冷的白色。
<br/>
<br/>        “寒霆，你醒了？”陆老夫人的声音传来。
<br/>
<br/>        陆寒霆扭头就看到了守在床边的老夫人，还有管家福伯，他的秘书严毅，大家脸色凝重，老夫人哭过了，眼眶红红的。
<br/>
<br/>        “奶奶…”陆寒霆动了动，可是身体下面的伤口瞬间就被扯痛，他的额头上出了一层冷汗。
<br/>
<br/>        老夫人迅速阻止，“寒霆，你现在不要动，医生说你下面的伤口太深了，虽然进行了手术，但是以后会影响到你的…性生活，以后你很有可能…不举…”
<br/>
<br/>        老夫人说着眼泪又下来了，这可是她的宝贝孙子，她连重孙都没有抱上，他的身体竟然就受了这么重的伤。
<br/>
<br/>        这可是一辈子的事。
<br/>
<br/>        陆寒霆不觉得意外，夏妍妍给他下的药太烈了，还带着迷幻的成分，他克制不住的将夏妍妍当成她，所以只能自己进了沐浴间拿刀划了自己一刀。
<br/>
<br/>        这一路他都在竭力的忍耐剧痛，大街上追她车的时候再也坚持不住，倒了下去。
<br/>
<br/>        至于床单上那抹血梅，是夏妍妍自己弄伤了自己，她为达目的，不择手段，不惜毁了自己的膜。
<br/>
<br/>        “寒霆，这是发生了什么事情，你怎么能对自己下这么狠的手，这事关你一辈子的性福还有以后的子嗣，你该如何向绾绾交代啊？”
<br/>
<br/>        陆寒霆握住了老夫人的手，“奶奶，对不起，我跟绾绾…离婚了。”
<br/>
<br/>        这件事瞒不住的，陆寒霆选择此刻坦白。
<br/>
<br/>        陆老夫人一僵，“什么？”
<br/>
<br/>        “奶奶，她留在我的身边真的很辛苦，我不会后悔划了自己一刀，至少我依然干净只属于她一个人，从未背叛，所以求你不要去找她，让她走吧。”
