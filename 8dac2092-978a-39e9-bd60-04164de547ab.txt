　　　　“父皇，您还别不信，就这一小坛酒，低于五万贯我都不卖。这可是最顶级的好酒，每年只有十坛的产量，再多一滴都没有了。”李承乾指着一边桌上人头大小的酒坛，张嘴就开始吹牛逼。
<br/>　　当着老头子的面，吹牛逼的机会不多，李承乾自然不会放过这个机会，一时间牛逼吹的山响，听的李二目瞪口呆，长孙瞠目结舌。
<br/>　　喝过两杯酒已经满面通红的方老太监更是一个劲的舔嘴唇，看样子恨不能把刚刚的酒杯拿来过再舔上两口。
<br/>　　酿酒的粮食，要求每一穗稻米的数量都是一样的，而且要求稻米的颗粒也同样要均匀无比，大小一至。选出稻米之后，由十四岁以下的纯洁处子，以樱唇贝齿将稻壳去掉……等等。
<br/>　　按李承乾所说，一坛酒，从选料，到加工，再到封存，工艺不下数百道。千年人灵芝、百年人参、十年鹿茸……各类名贵中草药数不胜数。
<br/>　　不过配方嘛……不足为外人道也。
<br/>　　这一番演讲，李承乾算是彻底过了瘾，整整吹了大概一个时辰，说的嘴角冒白沫，才堪堪住了口，拿过已经凉了的茶水，大大的灌了一口。
<br/>　　“怎，怎么会如此糜费？这酒……”长孙皇后已经不知说什么才好啦，一惯简朴的她完全想不到为了一坛酒竟然会如此繁琐，而且糜费甚多。
<br/>　　李二则轻轻拍了拍老婆的手，若有深意的看着李承乾：“这酒真就只有十坛？”
<br/>　　“父皇，这最上等的，只有十坛，每坛价格五万贯；还有次一等的，年产百坛，每坛价格一万贯；至于最下等的，年产千坛，每坛价格一千贯。”
<br/>　　“两百五十万贯！”李二概至的算了一下全部卖出去的价格，随后淡淡问道：“你真的以为我大唐百姓能消费的得起这么贵的酒？”
<br/>　　“啊？大唐百姓？这酒根本就不是在大唐卖的。”李承乾想了想，刚刚似乎吹的太嗨，忘记了一些事情，于是补充说道：“父皇，大秦、大食、吐蕃、吐谷浑还有高句丽、新罗、百济等等，这些国家有钱人太多了，五万贯买一坛限量版的大唐美酒对他们来说并不算什么。”
<br/>　　“那些异族需要的是身份、是体面，庆典的时候，能拿出大唐的限量版美酒那是何等的荣耀，区区几万贯他们还是出得起的。”
<br/>　　“这五万贯，卖的不单单是酒，还有权利和地位。在打开这坛酒的时候，就会想起，普天之下，最多只有九个人会与自己一样，那种独一无二的感觉，决不是五万贯能买回来的。”
<br/>　　又是盏茶时间过去，李二长长的叹了口气，一时间竟然不知应该说些什么。
<br/>　　因为在李承乾结束这段话的同时，他甚至想把这些酒全都留下，一坛都不卖。
<br/>　　可是，这可能么？只要是混小子亲自经手的事情，用屁股想都知道其中一定会有问题，而且他说的越好听，说的越认真，这里面的问题就越大，谁要是把他的话当成真的来听，那就是天下最大的傻子。
<br/>　　于是，李二问了一个很他最关心的问题：“太子，你就直接跟朕说，这些酒的成本一共是多少。”
<br/>　　所有人都竖起了耳朵，打算听到一个惊天的价格，必竟全部卖出去就可以收获二百五十万贯，这成本必定不会太低。
<br/>　　甚至长孙也把目光投向李承乾，深深的吸了一口气，准备听一个惊天的数字。
<br/>　　“两万贯？”李二看着儿子伸出来的两根手指，轻轻点了点头。
<br/>　　两万贯的投资，换取两百五十万，百倍收益，挺不错了。
<br/>　　结果，李承乾摇了摇头，表现老头子猜的不对。
<br/>　　“二十万贯？”长孙皇后追问一句。
<br/>　　“不是，高了。”李承乾继续比着两根手指，犹豫了一下，和老李和长孙对视一眼：“两，两贯！”
<br/>　　李二感觉李承乾已经疯了，又或者是自己已经疯了，两贯钱的东西，这个混小子竟然敢卖出两百五万贯的价格。
<br/>　　这个世界到底是怎么了，为什么会如此疯狂？以两贯钱的投资，换回一万两千五百倍的收益，真不知道这混小子脑子是怎么长的，为什么会有这样奇葩的想法？这特么可能实现么？
<br/>　　长孙皇后已经没什么想法了，只感觉这个世界已经没什么东西可以阻止儿子继续折腾下去了。
<br/>　　先是折腾‘兰若寺’，然后折腾东宫，接着是山东，再然后是整个大唐，到后来的倭国、突厥、吐蕃。
<br/>　　现在，已经没什么地方是他没折腾到的了，混小子开始把目光投向更加广袤的世界，大秦、大食那些远在万里之外的番邦也没有逃出他的手心。
<br/>　　想当年，李二带着她纵横中原大地，横刀立马，誓要斩尽天下豪杰，要让她成为中原大地上最尊贵的女人。
<br/>　　现在看看，和儿子一比，老头子当初的志向好像小了些，眼光也狭隘了些。
<br/>　　难道这就是青出于蓝？或者长江后浪推前浪？
<br/>　　大唐有没有粮食酿酒的问题已经没人考虑了，两贯钱才能买多少粮？现在已经是贞观四年，粮价已经掉到四～五文钱一斗，按斤算的话，两贯钱能买六千多斤。

<br/>　　而且就凭李承乾那张破嘴，死的都能说成活的，根本就不存在卖不出去这回事儿。
<br/>　　两百五十万贯，就算是一半，也有一百二十五万贯，这是多少钱啊？攒上几年就可以在龙首原另建一座大明宫了吧？
<br/>　　还有，洛阳的九成宫也要修一修，钱也要从这里面出。
<br/>　　一年一百二十五万贯，十年一千二百五十万，一百年……，不得不说，李二现在想的有些多。
<br/>　　但是很快李二就想到了一个很现实的问题，为什么要分给李孝恭一半呢？那家伙只是投了二十五万贯而以，十分之一的钱，就想拿走一半的利润？天下怎么可能会有这么好的事情……
